## 详细

**项目介绍：**
UniConverter 是万兴科技旗下面向 Windows 平台用户与内容创作者的综合多媒体处理软件，涵盖视频格式转换、压缩、下载、录制、编辑以及与视频、图片、音频相关的 AI 智能工具（水印去除、视频增强、图像增强、背景替换、图像生成、智能总结、字幕编辑、语音文字转换、人声噪声移除等）全套能力。项目使用 C# + WinForms 开发，架构上采用分层设计（View + Controller + Model + 公共服务层 + 原生引擎层），UI 层采用公司自研的 WUL（Wondershare UI Library）控件框架，底层通过 DllImport/COM 对接 AgileTrans 转码引擎、ffmpeg 等 C/C++ 原生库，AI 功能支持本地 SDK 与云端 API 双路策略。项目规模较大，约 80 多个子工程，具备较高的模块化与复杂度。

**项目内容：**
- 维护升级：负责视频转换、压缩、下载核心模块和 AI 智能工具的升级迭代与日常维护；日常收集 BugSplat 崩溃日志定位解决线上问题，并及时响应用户反馈客诉问题处理，持续优化产品质量。 
- 功能开发：主导以下模块的功能开发与迭代：
    - 云端 AI 任务处理框架： 设计并实现了贯穿多个 AI 功能模块的云端任务处理公共链路，封装「上传阿里云 OSS → 创建 AI 任务 → 轮询状态（排队/处理中/完成/失败）→ 下载结果」完整流程；以 AIProcess 抽象基类 + IAPITaskParams 参数 接口为扩展点，各 AI 业务模块通过继承覆写 API 端点、请求参数、响应解析实现差异化接入，无需重复实现通用链路；OSS 上传由 AliOssHelper 统一管理凭证获取与重试策略，轮询状态码语义（1=排队/2=处理中/3=成功/4~6=失败/取消/超时）在各模块间保持一致，便于统一的错误处理与任务上报。
    - 智能总结功能：基于视频下载和 STT 基础上结合 DeepSeek 扩展 AI 总结能力，构建「音视频转录 → 字幕编辑 → AI 总结 → 导出」完整闭环；转录阶段音频分离后复用云端 AI 任务处理链路提交 STT 服务，字幕结果由总结模块进行分段、说话人标注编辑并支持多格式导出，覆盖学习与轻办公场景。将碎片化音视频内容转化为可检索、可编辑的结构化摘要，有效降低用户信息提取成本。
    - 视频增强模块：基于 MVC 分层架构，支持 18 种 AI 增强模型，涵盖去频闪、视频稳像、智能调色、超分辨率、人像/动漫增强、降噪、帧插值等能力。通过检测 GPU SM 架构与驱动状态，在运行时动态分流：有 GPU 时优先走本地 SDK 加速，去频闪、超分辨率、智能调色等云端专属模型及无 GPU 时的人脸/动漫增强则切换至云端路径。云端处理由控制器编排三阶段流水线：视频按时长切片；复用云端 AI 任务处理链路完成增强处理；视频合并，三阶段进度按 10%/80%/10% 加权呈现；多模型串联时通过传递上一步返回的 OSS 路径直接作为下一模型输入，省去重复上传开销。显著提升了低质素材的可修复能力，覆盖降噪、修复、风格化等多类画质优化场景，拓展了内容创作者与专业用户的使用深度。
    - 图片增强模块：基于 MVC 分层架构，支持自动增强、AI 超分（2x/4x/8x）、旧照片修复/上色、人脸增强、色彩校正、NanaBanana 系列去模糊/光线修复等 12 种 AI 模型，全程采用云端处理。复用云端 AI 任务处理链路完成图片处理；多模型串联时通过上一模型返回的 OSS 路径直接作为下一模型输入，省去中间结果的下载再上传开销；支持批量并发处理，由统一的进度管理器协调多任务状态。引入免费预览缓存机制，降低重复请求开销，提升用户预览体验。覆盖修复、上色、降噪、超分等多类图像优化场景，有效满足内容创作者对老旧照片翻新与图像质量提升的核心诉求。 
    - 图片生成功能：基于 MVC 分层架构，对接 NanaBanana 生成式模型云端 API，实现营销主题图片生成；通过模板配置机制预设场景参数，支持灵活扩展新模板，降低运营接入成本。
    - 水印去除功能：支持图片与视频两种类型，提供手动与自动两种去除模式。手动模式由用户画笔框选生成 mask 文件，自动模式升级接入 Seedream 4.5，细分为文字水印、Logo 标记、人物三类场景，两种模式均复用云端 AI 任务处理链路完成水印去除，AI prompt 策略支持热更新，通过代码默认值、本地配置文件、远端下发三级优先级动态覆盖，无需发版即可调整各场景的去除效果。引入预览缓存，防止重复请求开销。本地处理在首次使用时由 SDK 管理器按需自动下载安装，供手动模式离线处理使用。覆盖文字、Logo、人物等多类水印去除场景，结合本地与云端双路策略，满足不同网络环境与隐私需求下的使用诉求。
    - 录制功能：基于 MVC 分层架构对录制模块屏幕录制、摄像头录制、音频录制模式进行全面重构；通过 COM 工厂对象对接底层采集引擎接口封装以及生命周期管理，新增游戏应用录制模式、摄像头镜像、鼠标样式与点击动效、麦克风实时降噪等能力。优化了用户体验，提升录制场景覆盖广度与专业度，提高了用户的使用意愿及转化率。活跃用户数显著增长 16.28%，转化率大幅提升 37.48%。  


## 精简

**项目介绍：**
UniConverter 是万兴科技旗下的一款面向 Windows 平台内容创作者的综合多媒体处理平台，覆盖视频转码、压缩、下载、录制、编辑及多种 AI 智能处理完整流程。
项目基于 C# + WinForms 构建，采用分层架构（View / Controller / Model / 服务层 / 原生引擎层），UI 基于自研 WUL 框架实现；底层通过 DllImport / COM 集成 AgileTrans、ffmpeg 等原生引擎。
AI 能力支持本地 SDK 与云端 API 双路执行，项目规模约 80+ 子工程。

**项目内容：**
1. 核心架构：云端 AI 任务处理框架

    设计并实现统一的云端 AI 任务处理链路，抽象「上传 → 任务创建 → 状态轮询 → 结果回收」完整流程：

    - 基于 AIProcess 抽象 + 参数接口解耦业务差异，各 AI 模块仅需实现模型接入逻辑即可复用通用流程
    - 统一 OSS 上传、任务状态管理与错误处理机制，支持多模型任务一致化调度
    - 支持多模型串联时复用中间结果（OSS 路径透传），减少重复上传与计算开销

    👉 效果：显著降低 AI 功能接入成本，提升多模块一致性与可扩展性

2. AI 能力与调度优化
    - 构建 本地 SDK + 云端 API 双执行路径，基于 GPU 能力与任务特征动态调度
    - 视频增强模块支持多模型组合处理，采用「分片处理 + 并发执行 + 合并输出」流水线方案
    - 图片增强模块支持批量并发任务与多模型串联，结合预览缓存机制降低重复请求开销

    👉 提升整体处理性能与资源利用率，优化用户等待体验

3. 多媒体处理模块开发
    - 视频增强模块： 支持 18 种 AI 模型（超分、稳像、调色等），提升低质量视频修复能力
    - 图片增强 / 生成 / 去水印模块： 支持多场景图像处理与生成能力，统一接入 AI 处理链路
    - 智能总结模块： 构建「转录 → 编辑 → AI 总结 → 导出」闭环，降低用户信息处理成本

4. 录制模块重构
    - 基于 MVC 架构重构录制系统，统一采集能力抽象
    - 对接底层采集引擎，实现游戏录制、摄像头镜像、鼠标动效、实时降噪等能力

    👉 成果：活跃用户数提升 16.28%，转化率提升 37.48%

5. 稳定性与质量保障
    - 基于 BugSplat 崩溃日志定位线上问题，排查多线程竞争、内存问题等复杂缺陷
    - 结合用户客诉反馈持续优化系统稳定性与用户体验
