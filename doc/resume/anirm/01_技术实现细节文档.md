# AniRemover Mobile - 技术实现细节文档

## 一、项目概述

AniRemover Mobile 是一款基于 Flutter 开发的跨平台（iOS/Android）AI 图片/视频智能消除工具应用。主要功能包括：图片智能去水印/去涂鸦、视频智能去水印、视频画质增强等。

**技术栈**：
- 开发框架：Flutter 3.35.4 + Dart 3.9.2
- 状态管理：GetX
- 网络层：Dio + Retrofit
- 异步任务：Stream + Future + Timer
- 支付系统：in_app_purchase（StoreKit1/2）
- 视频处理：FFmpeg + 原生插件
- 图像处理：image 库 + 自定义 Canvas 绘制
- 持久化：SharedPreferences + SQLite（ObjectBox）
- 云存储：阿里云 OSS

## 二、核心模块技术实现

### 2.1 任务处理框架（Task Processing Framework）

#### 2.1.1 架构设计

采用**策略模式 + 模板方法模式**构建的统一任务处理引擎，实现了业务逻辑与调度逻辑的完全解耦。

**核心组件**：

1. **TaskProcessor<TInput, TResult>**（策略接口）
   - 定义任务处理的标准生命周期：`preprocess → checkStatus → postprocess`
   - 各业务模块（图片擦除、视频增强等）通过实现该接口提供具体处理逻辑
   - 支持泛型输入/输出，保证类型安全

2. **TaskProcessingService**（单例调度器）
   - 管理任务的完整生命周期（提交、执行、取消、重试）
   - 维护活跃任务池：`Map<String, _ManagedTask> _activeTasks`
   - 实现并发控制：通过 `TaskCenterRuntimeManager` 的 slot 机制限制并发任务数
   - 提供 Stream 事件流：`TaskHandle.events` 供 UI 层订阅

3. **TaskCenterRepository**（持久化层）
   - 基于 ObjectBox 实现任务数据的增删改查
   - 支持原子批量操作（`persistGroupWithTasksBatch`）
   - 维护任务中心（TaskCenter）的历史记录

#### 2.1.2 核心技术实现

**1. 任务提交流程**

```dart
TaskHandle submit<TInput, TResult>(
  TaskProcessor<TInput, TResult> processor,
  TInput input,
) {
  // 1. 生成本地 taskId（UUID）
  final localTaskId = Uuid().v4();
  
  // 2. 尝试获取 runtime slot（并发控制）
  if (!runtimeManager.tryAcquireProcessingSlot(localTaskId)) {
    throw TaskProcessingSlotFullException();
  }
  
  // 3. 创建 TaskMeta 并持久化
  final taskMeta = processor.createTaskMetadata(input);
  await persistence.prepareInitialTask(taskMeta);
  
  // 4. 创建事件流（broadcast）
  final eventController = StreamController<TaskEvent>.broadcast();
  
  // 5. 异步执行任务（unawaited）
  unawaited(_executeTask(localTaskId, processor, input, eventController));
  
  // 6. 返回 TaskHandle（包含 taskId + events）
  return TaskHandle(taskId: localTaskId, events: eventController.stream);
}
```

**2. 任务执行引擎（模板方法）**

```dart
Future<void> _executeTask<TInput, TResult>(
  String taskId,
  TaskProcessor<TInput, TResult> processor,
  TInput input,
  StreamController<TaskEvent> eventController,
) async {
  final cancellationToken = CancellationToken();
  
  try {
    // ═══ Phase 1: Preprocessing ═══
    emitEvent(TaskPhase.preprocessing, progress: 0.0);
    final serverTaskId = await processor.preprocess(
      input, 
      cancellationToken, 
      emitEvent,
    );
    
    // ═══ Phase 2: Polling ═══
    await _pollUntilComplete(serverTaskId, processor, cancellationToken, emitEvent);
    
    // ═══ Phase 3: Postprocessing ═══
    emitEvent(TaskPhase.postprocessing, progress: 0.95);
    final result = await processor.postprocess(
      serverTaskId, 
      completedStatus, 
      cancellationToken,
      emitEvent,
    );
    
    // ═══ Phase 4: Completed ═══
    await _finalizeTask(taskId, result);
    emitEvent(TaskPhase.completed, progress: 1.0, result: result);
    
  } on TaskCancelledException {
    await processor.cancelServerTask(serverTaskId);
    emitEvent(TaskPhase.cancelled);
  } catch (e) {
    await processor.onTaskFailed(); // 安全网：释放资源（如预扣积分）
    emitEvent(TaskPhase.failed, error: e.toString());
  } finally {
    runtimeManager.releaseProcessingSlot(taskId);
    _activeTasks.remove(taskId);
  }
}
```

**3. 轮询机制 + 虚拟进度**

```dart
Future<void> _pollUntilComplete(...) async {
  double virtualProgress = 0.0;
  final config = processor.virtualProgressConfig; // 虚拟进度配置
  
  // 启动真实轮询 Timer
  final pollingTimer = Timer.periodic(processor.pollingInterval, (timer) async {
    final snapshot = await processor.checkStatus(serverTaskId, cancellationToken);
    
    if (snapshot.isCompleted || snapshot.isFailed) {
      timer.cancel();
      return snapshot;
    }
    
    // 合并真实进度与虚拟进度（取 max，避免倒退）
    final realProgress = snapshot.progress ?? 0.0;
    final mergedProgress = max(realProgress, virtualProgress);
    emitEvent(TaskPhase.polling, progress: mergedProgress);
  });
  
  // 启动虚拟进度 Timer（平滑动画）
  Timer.periodic(config.tickInterval, (timer) {
    virtualProgress = min(virtualProgress + config.increment, config.ceiling);
    // 虚拟进度上限 88%，真实进度可突破
  });
}
```

**4. 批量任务与组任务（Group）**

```dart
Future<List<TaskHandle>> submitBatch<TInput, TResult>(
  TaskProcessor<TInput, TResult> processor,
  List<TInput> inputs,
  {required String groupId}
) async {
  // 1. 预占 runtime slots（原子性保证）
  final tokens = <String>[];
  for (final input in inputs) {
    final token = Uuid().v4();
    if (!runtimeManager.tryAcquireProcessingSlot(token)) {
      // 回滚已预占的 slots
      for (final t in tokens) {
        runtimeManager.releaseProcessingSlot(t);
      }
      throw TaskProcessingSlotFullException();
    }
    tokens.add(token);
  }
  
  // 2. 批量持久化（原子事务）
  await persistence.persistGroupWithTasksBatch(
    group: TaskCenterTaskGroup(groupId: groupId, ...),
    childTasks: childTaskMetas,
  );
  
  // 3. 并行启动子任务
  final handles = <TaskHandle>[];
  for (int i = 0; i < inputs.length; i++) {
    final handle = _startChildTask(tokens[i], processor, inputs[i], groupId);
    handles.add(handle);
  }
  
  // 4. 监听所有子任务完成 → 触发 finalize
  _listenForGroupCompletion(groupId, handles);
  
  return handles;
}
```

**5. 取消机制**

```dart
class CancellationToken {
  bool _isCancelled = false;
  final List<VoidCallback> _callbacks = [];
  
  void onCancel(VoidCallback callback) {
    _callbacks.add(callback);
  }
  
  void cancel() {
    _isCancelled = true;
    for (final cb in _callbacks) {
      cb(); // 触发回调（如 AliyunTool.cancelUpload）
    }
  }
  
  void throwIfCancelled() {
    if (_isCancelled) throw TaskCancelledException();
  }
  
  CancelToken createDioCancelToken() {
    final dioCancelToken = CancelToken();
    onCancel(() => dioCancelToken.cancel());
    return dioCancelToken;
  }
}
```

### 2.2 图片智能消除模块（Image Mask Remove）

#### 2.2.1 业务流程

**手动模式**：用户涂抹 → 上传原图+Mask → 服务端处理 → 下载结果 → 延迟扣费（导出时）

**自动模式（Seedream）**：用户选择模板 → 上传原图+Prompt → AI 识别+预扣积分 → 服务端处理 → 下载结果 → 提交扣费

#### 2.2.2 核心实现

**1. ImageRemoveProcessor（策略实现）**

```dart
class ImageRemoveProcessor extends TaskProcessor<ImageRemoveInput, ImageRemoveResult> {
  final ImageVideoMaskRemoveTool _removeTool = ImageVideoMaskRemoveTool();
  String? _activeAICreditBizId; // 预扣积分 bizId
  
  @override
  Duration get pollingInterval => const Duration(seconds: 1);
  
  @override
  VirtualProgressConfig get virtualProgressConfig => const VirtualProgressConfig(
    tickInterval: Duration(seconds: 3),
    increment: 0.02, // 每 3 秒增加 2%，约 90 秒达到 88% 上限
  );
  
  @override
  Future<String> preprocess(
    ImageRemoveInput input,
    CancellationToken cancellationToken,
    void Function(TaskEvent) emitEvent,
  ) async {
    // ── Step 1: 上传原图到 OSS ──
    emitEvent(TaskEvent(phase: TaskPhase.preprocessing, progress: 0.0));
    cancellationToken.onCancel(AliyunTool.cancelUpload); // 注册取消回调
    
    final imageOssUri = await AliyunTool.getOSSTokenAndUploadWithFileOrThrow(
      bytes: input.imageBytes,
      cancelToken: cancellationToken.createDioCancelToken(),
    );
    cancellationToken.throwIfCancelled();
    
    // ── Step 2: 上传 Mask（手动模式）或 Prompt（自动模式） ──
    String? maskOssUri;
    if (!input.useSeedream) {
      // 手动模式：resize mask to original image size
      final resizedMask = await _resizeMask(input.maskBytes!, input.originImageSize!);
      maskOssUri = await AliyunTool.getOSSTokenAndUploadWithFileOrThrow(
        bytes: resizedMask,
        cancelToken: cancellationToken.createDioCancelToken(),
      );
    }
    
    // ── Step 3: 创建服务端任务 ──
    emitEvent(TaskEvent(phase: TaskPhase.preprocessing, progress: 0.5));
    
    final response = input.useSeedream
      ? await _removeTool.createSeedreamImageRemoveTask(
          SeedreamImageRemoveParams(
            imageUri: imageOssUri,
            prompt: input.seedreamPrompt!,
          ),
          cancellationToken.createDioCancelToken(),
        )
      : await _removeTool.createImageRemoveTask(
          ImageMaskRemoveParams(
            imageUri: imageOssUri,
            maskUri: maskOssUri!,
          ),
          cancellationToken.createDioCancelToken(),
        );
    
    final serverTaskId = response.data.taskId;
    
    // ── Step 4: AI 积分预扣（自动模式） ──
    if (input.useSeedream) {
      _activeAICreditBizId = await AniEraserAuthorization.instance.predeductAICredit(
        serverTaskId: serverTaskId,
      );
    }
    
    return serverTaskId; // 返回服务端 taskId 供轮询使用
  }
  
  @override
  Future<TaskStatusSnapshot> checkStatus(
    String serverTaskId,
    CancellationToken cancellationToken,
  ) async {
    final response = await _removeTool.createQueryResult(
      serverTaskId,
      cancellationToken.createDioCancelToken(),
    );
    
    return TaskStatusSnapshot(
      isCompleted: response.data.status == 'completed',
      isFailed: response.data.status == 'failed',
      progress: null, // 服务端不返回细粒度进度，使用虚拟进度
      resultData: response.data.resultUrl,
    );
  }
  
  @override
  Future<ImageRemoveResult> postprocess(
    String serverTaskId,
    TaskStatusSnapshot completedStatus,
    CancellationToken cancellationToken,
    void Function(TaskEvent) emitEvent,
  ) async {
    emitEvent(TaskEvent(phase: TaskPhase.postprocessing, progress: 0.95));
    
    // 下载结果图片
    final resultUrl = completedStatus.resultData as String;
    final localPath = await ImageTool.downloadImageFile(resultUrl);
    
    // 提交积分（自动模式）
    if (_activeAICreditBizId != null) {
      await AniEraserAuthorization.instance.commitDeduct(_activeAICreditBizId!);
      _activeAICreditBizId = null;
    }
    
    return ImageRemoveResult(
      localPath: localPath,
      remoteUrl: resultUrl,
      pendingCreditBizId: _useSeedream ? null : serverTaskId, // 手动模式：延迟扣费
    );
  }
  
  @override
  Future<void> onTaskFailed() async {
    // 安全网：任务失败时释放预扣积分（幂等操作）
    if (_activeAICreditBizId != null) {
      await AniEraserAuthorization.instance.releasePrededuct(_activeAICreditBizId!);
      _activeAICreditBizId = null;
    }
  }
}
```

```dart
        AlertManager.instance.dismissAlert(_progressAlertKey);
        break;

    // 前台取消：Controller 销毁时取消任务

      TaskProcessingService.instance.cancel(_activeTaskId!);
    }

  }
```

### 2.3 视频增强模块（Video Enhancer）

#### 2.3.1 业务流程

**短视频（<30秒）**：视频分段 → 并发上传 → 创建单任务 → 轮询 → 下载 → 提交扣费

**长视频（≥30秒）**：视频分片（每片≤30秒）→ 并发上传 → 创建批量任务（Group）→ 轮询所有子任务 → Finalize（下载+合并）→ 提交扣费

#### 2.3.2 核心实现

**1. VideoEnhanceProcessor（核心处理器）**

```dart
class VideoEnhanceProcessor extends TaskProcessor<VideoEnhanceInput, VideoEnhanceResult> {
  final VideoEnhanceRemoteDataSource _remoteDataSource;
  final VideoEnhanceBillingGateway _billingGateway;
  String? _activeCreditBizId;
  
  @override
  Duration get pollingInterval => const Duration(seconds: 2);
  
  @override
  VirtualProgressConfig get virtualProgressConfig => const VirtualProgressConfig(
    tickInterval: Duration(seconds: 5),
    increment: 0.01, // 更缓慢的虚拟进度（视频处理耗时更长）
    ceiling: 0.85,
  );
  
  @override
  Future<String> preprocess(
    VideoEnhanceInput input,
    CancellationToken cancellationToken,
    void Function(TaskEvent) emitEvent,
  ) async {
    // ── Step 1: 视频分段（segment）──
    emitEvent(TaskEvent(phase: TaskPhase.preprocessing, progress: 0.0));
    
    final segments = await _segmentVideo(input.videoPath, input.trimRange);
    cancellationToken.throwIfCancelled();
    
    // ── Step 2: 并发上传所有分段到 OSS ──
    emitEvent(TaskEvent(phase: TaskPhase.preprocessing, progress: 0.3));
    
    final uploadTasks = segments.map((segment) async {
      return await AliyunTool.getOSSTokenAndUploadWithFileOrThrow(
        filePath: segment.path,
        cancelToken: cancellationToken.createDioCancelToken(),
        onProgress: (progress) {
          // 聚合上传进度
        },
      );
    }).toList();
    
    final segmentUris = await Future.wait(uploadTasks);
    cancellationToken.throwIfCancelled();
    
    // ── Step 3: 创建服务端任务 ──
    emitEvent(TaskEvent(phase: TaskPhase.preprocessing, progress: 0.7));
    
    final response = await _remoteDataSource.createEnhanceTask(
      VideoEnhanceParams(
        segmentUris: segmentUris,
        resolution: input.targetResolution,
        frameRate: input.targetFps,
      ),
      cancellationToken.createDioCancelToken(),
    );
    
    final serverTaskId = response.data.taskId;
    
    // ── Step 4: 预扣积分 ──
    _activeCreditBizId = await _billingGateway.reserveDeduct(
      serverTaskId: serverTaskId,
      videoDurationSeconds: input.videoDuration,
    );
    
    return serverTaskId;
  }
  
  @override
  Future<VideoEnhanceResult> postprocess(
    String serverTaskId,
    TaskStatusSnapshot completedStatus,
    CancellationToken cancellationToken,
    void Function(TaskEvent) emitEvent,
  ) async {
    emitEvent(TaskEvent(phase: TaskPhase.postprocessing, progress: 0.9));
    
    // 下载增强后的视频
    final resultUrl = completedStatus.resultData as String;
    final localPath = await DioDownloader.downloadFile(
      url: resultUrl,
      savePath: await _generateSavePath(),
      onProgress: (received, total) {
        emitEvent(TaskEvent(
          phase: TaskPhase.postprocessing,
          progress: 0.9 + 0.1 * (received / total),
        ));
      },
    );
    
    // 提交扣费
    if (_activeCreditBizId != null) {
      await _billingGateway.commitDeduct(_activeCreditBizId!);
      _activeCreditBizId = null;
    }
    
    return VideoEnhanceResult(localPath: localPath, remoteUrl: resultUrl);
  }
  
  // 内部 retry helper（指数退避）
  Future<T> _withRetry<T>(Future<T> Function() operation, {int maxAttempts = 3}) async {
    int attempt = 0;
    while (true) {
      try {
        return await operation();
      } catch (e) {
        attempt++;
        if (attempt >= maxAttempts) rethrow;
        await Future.delayed(Duration(seconds: math.pow(2, attempt).toInt()));
      }
    }
  }
}
```

**2. 长视频分片处理（Group Workflow）**

```dart
class VideoBackgroundWorkflowFactory {
  static Future<WorkflowHandle> planAILong(VideoWorkflowRequest request) async {
    // 1. 切片视频（每片 ≤ 30 秒）
    final chunks = await _splitVideoIntoChunks(
      request.videoPath,
      maxChunkDuration: Duration(seconds: 30),
    );
    
    // 2. 构造每个子任务的输入
    final inputs = chunks.map((chunk) => VideoEnhanceInput(
      videoPath: chunk.path,
      trimRange: null, // 已切片，无需二次裁剪
      targetResolution: request.resolution,
    )).toList();
    
    // 3. 提交批量任务（Group）
    final groupId = Uuid().v4();
    final handles = await TaskProcessingService.instance.submitBatch(
      processor: VideoEnhanceProcessor(),
      inputs: inputs,
      groupId: groupId,
    );
    
    // 4. 注册 finalize action（所有子任务完成后执行）
    TaskProcessingService.instance.registerGroupFinalizeAction(
      bizType: BizType.videoEnhance,
      action: (groupId, childResults, cancellationToken, emitEvent) async {
        emitEvent(TaskEvent(phase: TaskPhase.finalizing, progress: 0.0));
        
        // 下载所有分片
        final downloadedChunks = <String>[];
        for (int i = 0; i < childResults.length; i++) {
          final result = childResults[i] as VideoEnhanceResult;
          downloadedChunks.add(result.localPath);
          emitEvent(TaskEvent(
            phase: TaskPhase.finalizing,
            progress: 0.5 * (i + 1) / childResults.length,
          ));
        }
        
        // 合并视频（FFmpeg concat）
        emitEvent(TaskEvent(phase: TaskPhase.finalizing, progress: 0.5));
        final mergedPath = await _mergeVideoChunks(downloadedChunks);
        
        // 清理临时分片
        for (final chunk in downloadedChunks) {
          File(chunk).deleteSync();
        }
        
        emitEvent(TaskEvent(phase: TaskPhase.finalizing, progress: 1.0));
        return VideoEnhanceResult(localPath: mergedPath);
      },
    );
    
    // 5. 构造 WorkflowHandle（聚合多个子任务的进度）
    return WorkflowHandle(
      groupId: groupId,
      events: _buildGroupStream(handles),
    );
  }
  
  static Stream<TaskEvent> _buildGroupStream(List<TaskHandle> childHandles) {
    // 聚合子任务进度（加权平均）
    final controller = StreamController<TaskEvent>.broadcast();
    
    final progressMap = <String, double>{};
    for (final handle in childHandles) {
      handle.events.listen((event) {
        progressMap[handle.taskId] = event.progress;
        
        final avgProgress = progressMap.values.reduce((a, b) => a + b) / 
                            childHandles.length;
        
        controller.add(TaskEvent(
          taskId: '', // groupId
          phase: event.phase,
          progress: avgProgress,
        ));
      });
    }
    
    return controller.stream;
  }
}
```

### 2.4 应用内购买模块（IAP）

#### 2.4.1 技术方案

**挑战**：
1. iOS：同时兼容 StoreKit 1（iOS 12+）和 StoreKit 2（iOS 15+）
2. Android：Google Play Billing 的收据验证与订阅管理
3. 跨平台：统一的购买流程接口

**解决方案**：
- 使用 `in_app_purchase` 插件的统一接口（iOS + Android）
- **平台工厂模式**：`StorePurchaseFactory` 根据平台返回不同实现
  - iOS → `IAPManager`（StoreKit 1/2 双栈兼容）
  - Android → `BillManager`（Google Play Billing）
- 实现完整的收据验证链：
  - iOS：本地验证 → Apple 服务器验证 → 自有后端验证
  - Android：Google Play token → 自有后端验证（服务端调用 Google Play Developer API）

```dart
      case PurchaseStatus.restored:

        break;

        _notifyPurchaseFailed(purchaseDetails.error!);
        break;

        _notifyPurchaseCanceled();
        break;
    }
  }

    T param,
    T Function(Map<String, dynamic>) fromJson,
  ) async {
    final params = await readPendingVerifyParams<T>(fromJson);
    params.add(param);
    await writePendingVerifyParams(params);
  }

  Future<List<AuthorizeInfo>> pollingCheckAuthorization() async {
    final deadline = DateTime.now().add(const Duration(seconds: 120));
    
    while (DateTime.now().isBefore(deadline)) {
      final info = await WSIDClient().fetchAllAuthorizeInfo();
      
      if (info.isNotEmpty && !_checkExpired(info)) {
        return info; // 验证成功且未过期
      }
      
      await Future.delayed(Duration(seconds: 3)); // 轮询间隔
    }
    
    throw TimeoutException('Authorization check timeout');
  }
  
  // 抽象方法：由子类实现平台特定逻辑
  Future<void> PurchaseSuccessStatusDone(PurchaseDetails purchaseDetails);
  Future<void> uploadLocalUnverifiedPurchases();
```

```dart
class IAPManager extends BaseStorePurchaseManager {
  static final IAPManager instance = IAPManager._();
  IAPManager._();
  
  @override
  Future<bool> buy(String productId) async {
    try {
      // ── Step 1: 清理未完成的交易（SK2 要求） ──
      if (Platform.isIOS) {
        await clearUnfinishedTransactions();
      }
      
      // ── Step 2: 查询商品详情 ──
      final productDetails = await _queryProductDetails(productId);
      if (productDetails == null) {
        throw Exception('Product not found');
      }
      
      // ── Step 3: 发起购买 ──
      final purchaseParam = PurchaseParam(productDetails: productDetails);
      final success = await InAppPurchase.instance.buyNonConsumable(
        purchaseParam: purchaseParam,
      );
      
      if (!success) return false;
      
      // ── Step 4: 监听购买结果（通过 Stream） ──
      // 实际处理在 _purchaseUpdatedStream.listen 中完成
      return true;
      
    } catch (e) {
      WsLog.e('IAPManager.buy failed: $e');
      return false;
    }
  }
  
  void _listenToPurchaseUpdated() {
    _purchaseSubscription = InAppPurchase.instance.purchaseStream.listen(
      (purchases) async {
        for (final purchase in purchases) {
          await _handlePurchaseUpdate(purchase);
        }
      },
      onError: (error) {
        WsLog.e('Purchase stream error: $error');
      },
    );
  }
  
  Future<void> _handlePurchaseUpdate(PurchaseDetails purchase) async {
    switch (purchase.status) {
      case PurchaseStatus.purchased:
        await _verifyAndCompletePurchase(purchase);
        break;
        
      case PurchaseStatus.error:
        _notifyPurchaseFailed(purchase.error!);
        break;
        
      case PurchaseStatus.pending:
        // 等待用户确认（如家庭共享审批）
        break;
        
      case PurchaseStatus.restored:
        // 恢复购买流程
        await _verifyAndCompletePurchase(purchase);
        break;
        
      case PurchaseStatus.canceled:
        _notifyPurchaseCanceled();
        break;
    }
  }
  
  Future<void> _verifyAndCompletePurchase(PurchaseDetails purchase) async {
    try {
      // ── Step 1: 区分 SK1/SK2 ──
      if (purchase is AppStorePurchaseDetails) {
        // StoreKit 1: 验证 base64 收据
        await _verifySK1Receipt(purchase);
      } else if (purchase is SK2PurchaseDetails) {
        // StoreKit 2: 验证 JWS token
        await _verifySK2Transaction(purchase);
      }
      
      // ── Step 2: 上报 Adjust 归因（仅用户主动购买） ──
      if (!_isRestoringPurchase) {
        AdjustTool.trackRevenue(purchase.productID, purchase.price);
      }
      
      // ── Step 3: 完成交易 ──
      await InAppPurchase.instance.completePurchase(purchase);
      
      // ── Step 4: 刷新本地权益 ──
      await AniEraserAuthorization.instance.refreshSubscriptionStatus();
      
      _notifyPurchaseSuccess();
      
    } catch (e) {
      WsLog.e('Verify purchase failed: $e');
      _notifyPurchaseFailed(e);
    }
  }
  
  // StoreKit 1 验证流程
  Future<void> _verifySK1Receipt(AppStorePurchaseDetails purchase) async {
    // 1. 读取本地 App Receipt（位于 Bundle.main.appStoreReceiptURL）
    final receiptData = await _loadLocalReceipt();
    if (receiptData == null) {
      throw Exception('Local receipt not found');
    }
    
    // 2. 调用自有后端验证接口
    final verifyResult = await WSIDClient().verifyReceipt(
      receipt: receiptData,
      productId: purchase.productID,
      transactionId: purchase.transactionDate,
    );
    
    if (!verifyResult.isValid) {
      // 3. Fallback: 直接调用苹果服务器（处理 21007/21008 沙盒/生产环境错误）
      await _verifyWithAppleServer(receiptData, purchase);
    }
  }
  
  // StoreKit 2 验证流程
  Future<void> _verifySK2Transaction(SK2PurchaseDetails purchase) async {
    // SK2 提供的 JWS token 已包含签名，直接发送给后端
    final verifyResult = await WSIDClient().verifySK2Transaction(
      jwsRepresentation: purchase.jwsRepresentation,
      productId: purchase.productID,
    );
    
    if (!verifyResult.isValid) {
      throw Exception('SK2 verification failed: ${verifyResult.reason}');
    }
  }
  
  // 恢复购买
  @override
  Future<void> restorePurchases() async {
    _isRestoringPurchase = true;
    
    try {
      // 触发 StoreKit 的恢复流程（会触发 purchaseStream 回调）
      await InAppPurchase.instance.restorePurchases();
      
      // iOS: 等待 purchaseStream 事件
      // Android: Google Play 推荐直接查询购买历史（queryPastPurchases）
      
    } finally {
      _isRestoringPurchase = false;
    }
  }
}
```

**2. 自动登录与重试机制**

```dart
Future<void> _verifyWithAutoLoginRetry(PurchaseDetails purchase) async {
  int attempt = 0;
  const maxAttempts = 2;
  
  while (attempt < maxAttempts) {
    try {
      // 检查登录状态
      if (!WSIDClient().isLoggedIn()) {
        WsLog.i('User not logged in, attempting auto-login...');
        final autoLoginSuccess = await WSIDClient().autoLogin();
        
        if (!autoLoginSuccess) {
          throw Exception('Auto-login failed');
        }
      }
      
      // 执行验证
      await _verifySK1Receipt(purchase);
      return; // 成功
      
    } on NetworkException catch (e) {
      // 403 Forbidden: token 过期，尝试刷新
      if (e.statusCode == 403 && attempt < maxAttempts - 1) {
        WsLog.i('Token expired, refreshing...');
        await WSIDClient().refreshToken();
        attempt++;
        continue;
      }
      rethrow;
      
    } catch (e) {
      WsLog.e('Verify attempt $attempt failed: $e');
      attempt++;
      if (attempt >= maxAttempts) rethrow;
      await Future.delayed(Duration(seconds: 2));
    }
  }
}
```

```dart
      return false;
    }
    
    printGooglePlayPurchaseDetails(purchaseDetails); // 调试日志

      user_id: WSIDClient().getUid().toString(),
      product_code: purchaseDetails.productID,
      package_name: "com.watermark.eraser.remover.photo.video",

      product_type: 1, // 0=非续费 1=续费 3=消耗型
      extra: extra,
    );

        WsLog.i("Google Play purchase completed");

        _trackAndroidSubscribeSuccess(purchaseDetails);
      }

      await saveUnverifiedVerifyParam(param, GoogleVerifyParam.fromJson);

    }

  @override

    final params = await readPendingVerifyParams<GoogleVerifyParam>(
      GoogleVerifyParam.fromJson,
    );
    
    if (params.isEmpty) return;
    
    WsLog.i("Found ${params.length} pending Google Play purchases, retrying...");
    
    final successList = <GoogleVerifyParam>[];
    
    for (final param in params) {
      try {
        final success = await WSIDClient().verifyGoogleReceipt(param);
        
        if (success) {
          successList.add(param);
          WsLog.i("Retry verification succeeded for ${param.product_code}");
        }
      } catch (e) {
        WsLog.e("Retry verification failed: $e");
      }
    }
    
    // 移除成功验证的记录
    if (successList.isNotEmpty) {
      params.removeWhere((p) => successList.contains(p));
      await writePendingVerifyParams(params);
    }
  }

        final offers = product.productDetails.subscriptionOfferDetails;
        
        if (offers != null && offers.isNotEmpty) {
          for (final offer in offers) {
            // 解析 pricing phases（定价阶段）
            final pricingPhases = offer.pricingPhases;
            
            if (pricingPhases.isNotEmpty) {
              final firstPhase = pricingPhases.first;
              
              // 判断是否为优惠价格
              if (firstPhase.priceAmountMicros < product.rawPrice * 1000000) {
                WsLog.i("Found introductory offer for ${product.id}");

                _saveIntroductoryOfferInfo(product.id, firstPhase);
              }
            }
          }
        }
      }
    }
  }
```

| **客户端验证** | 支持（但本项目未使用） | 支持（通过公钥，但本项目未使用） |

## 三、关键技术特性

### 3.1 并发控制

**Runtime Slot 机制**：
- 全局维护可用处理槽位（默认 3 个）
- 提交任务前必须获取 slot（`tryAcquireProcessingSlot`）
- 任务完成/取消/失败后释放 slot（`releaseProcessingSlot`）
- 批量任务原子预占：要么全部成功，要么全部回滚

**实现**：

```dart
class TaskCenterRuntimeManager {
  static const int _maxConcurrentTasks = 3;
  final Set<String> _activeSlots = {};
  
  bool tryAcquireProcessingSlot(String runtimeToken) {
    if (_activeSlots.length >= _maxConcurrentTasks) {
      return false;
    }
    _activeSlots.add(runtimeToken);
    return true;
  }
  
  void releaseProcessingSlot(String runtimeToken) {
    _activeSlots.remove(runtimeToken);
  }
}
```

### 3.2 错误处理与重试

**分层错误处理**：

1. **网络层**：Dio 拦截器统一处理 HTTP 错误
2. **Processor 层**：业务逻辑错误（如上传失败）触发内部 retry
3. **TPS 层**：捕获所有异常，调用 `processor.onTaskFailed()` 清理资源
4. **UI 层**：监听 `TaskPhase.failed` 事件，展示重试弹窗

**指数退避重试**：

```dart
Future<T> _withRetry<T>(
  Future<T> Function() operation, {
  int maxAttempts = 3,
  Duration initialDelay = const Duration(seconds: 1),
}) async {
  int attempt = 0;
  while (true) {
    try {
      return await operation();
    } catch (e) {
      attempt++;
      if (attempt >= maxAttempts) rethrow;
      
      final delay = initialDelay * math.pow(2, attempt - 1);
      WsLog.w('Retry attempt $attempt after $delay: $e');
      await Future.delayed(delay);
    }
  }
}
```

### 3.3 资源管理

**生命周期管理**：
- 临时文件：任务完成后自动清理（`TaskResourceManager`）
- 缩略图：持久化到应用沙盒（`ensureThumbnail`）
- 结果文件：用户主动删除或应用卸载时清理

**内存优化**：
- 图片加载：使用 `image_size_getter` 先获取尺寸，避免全量加载
- 大图处理：分块读取（chunk streaming）
- Mask 绘制：离屏渲染 + 局部刷新

### 3.4 取消机制

**多层级取消支持**：

1. **UI 层**：用户点击取消按钮 → `TaskProcessingService.cancel(taskId)`
2. **TPS 层**：调用 `CancellationToken.cancel()` → 触发所有注册的回调
3. **Processor 层**：
   - 检查点：`cancellationToken.throwIfCancelled()`
   - 网络请求：`Dio CancelToken` 自动关联
   - 上传任务：`AliyunTool.cancelUpload()` 回调
4. **服务端**：`processor.cancelServerTask(serverTaskId)` 通知后端取消
