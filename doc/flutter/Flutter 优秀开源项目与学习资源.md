## 🧱 二、实用库推荐
| 功能   | 库名                                    | 简介               |
| ---- | ------------------------------------- | ---------------- |
| 状态管理 | provider / riverpod / bloc            | 官方推荐和主流架构。       |
| 路由导航 | go_router / auto_route                | 强大的路由控制与深度链接支持。  |
| 网络请求 | dio                                   | 强大且易用的 HTTP 客户端。 |
| 本地存储 | hive / shared_preferences             | 快速轻量的键值数据库。      |
| 动画   | flutter_staggered_animations / lottie | 丰富的动画展示。         |
| 图片处理 | cached_network_image / image          | 图片缓存、加载、压缩。      |
| 国际化  | easy_localization                     | 简洁的多语言支持。        |

## 📚 三、系统学习路线与资料

### 🧭 学习路线（推荐顺序）
1. **基础入门**
   - 学习 Dart 语法基础（控制结构、函数、类）
   - 熟悉 Flutter Widget 树、布局、Stateful/Stateless 区别
2. **进阶开发**
   - 学习导航（Navigator 2.0 / go_router）
   - 掌握常见状态管理（Provider、Bloc、Riverpod）
   - 学习动画、主题切换、自定义控件
3. **项目实战**
   - 小项目练手（Todo、新闻、天气）
   - 大项目（电商、聊天、视频类 App）
4. **部署与优化**
   - 打包 Android / iOS
   - 性能优化、异步任务、依赖注入
   - 学习 Flutter Web / Desktop 开发
