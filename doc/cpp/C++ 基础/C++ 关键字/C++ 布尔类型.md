C++ 中的布尔类型（Boolean Type）是一个专门用于表示**真（true）或假（false）**的基本数据类型。

## ✅ 基本介绍

- 关键字：`bool`
- 取值：`true` 或 `false`
- 占用空间：通常为 **1 字节**（实现相关）

### 示例：

```cpp
bool isOK = true;
bool isEmpty = false;

if (isOK) {
    std::cout << "OK!" << std::endl;
}
```

## ✅ C++ 中布尔类型的特点

| 特性     | 描述                                                        |
| -------- | ----------------------------------------------------------- |
| 真/假值  | `true` 表示真（1），`false` 表示假（0）                     |
| 自动转换 | 可以与整数自动互相转换                                      |
| 输入输出 | `std::cout << boolalpha` 可以打印 `true/false` 而不是 `1/0` |



### 自动类型转换示例：

```cpp
bool b1 = 5;       // 非 0 转为 true
bool b2 = 0;       // 0 转为 false

int i1 = true;     // 转换为 1
int i2 = false;    // 转换为 0
```

## ✅ 与 C 中的差异

| 特性                | C语言                               | C++语言                |
| ------------------- | ----------------------------------- | ---------------------- |
| 是否有 `bool` 类型  | ❌ 没有（使用 `int`）                | ✅ 有内建的 `bool` 类型 |
| `true/false` 关键字 | ❌ 没有，通常用 0 和 非0 表示        | ✅ 有 `true` / `false`  |
| 标准头文件          | C99 才引入 `_Bool` 和 `<stdbool.h>` | `bool` 是内建关键字    |



> 🔹 说明：在 C 中，如果你写 `bool flag = true;`，没有包含 `<stdbool.h>` 就会报错，而在 C++ 中 `bool`、`true`、`false` 是语言内建的，**不需要任何头文件。**
