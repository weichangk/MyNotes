## 5.1 Qt 的事件处理机制是什么？

### 答案

Qt 的事件系统基于**事件循环（Event Loop）**和**事件分发机制**。

### 事件处理流程

```
事件产生 → 加入事件队列 → 事件循环 → 事件分发 → 事件处理
```

### 五级事件处理

1. **重新实现特定事件处理器**（如 `mousePressEvent()`）
2. **重新实现 event() 函数**
3. **安装事件过滤器到对象**（`installEventFilter()`）
4. **安装事件过滤器到应用程序**（`QApplication::installEventFilter()`）
5. **重新实现 notify() 函数**

### 示例

```cpp
class MyWidget : public QWidget {
protected:
    // 方式1：重写特定事件处理器
    void mousePressEvent(QMouseEvent* event) override {
        if (event->button() == Qt::LeftButton) {
            qDebug() << "Left button pressed at" << event->pos();
        }
    }
    
    // 方式2：重写 event() 函数
    bool event(QEvent* e) override {
        if (e->type() == QEvent::KeyPress) {
            QKeyEvent* ke = static_cast<QKeyEvent*>(e);
            qDebug() << "Key:" << ke->key();
            return true;  // 事件已处理
        }
        return QWidget::event(e);
    }
};
```

## 5.2 什么是事件过滤器？如何使用？

### 答案

事件过滤器允许一个对象拦截并处理其他对象的事件。

### 使用方法

```cpp
class EventFilter : public QObject {
protected:
    bool eventFilter(QObject* obj, QEvent* event) override {
        if (event->type() == QEvent::MouseButtonPress) {
            QMouseEvent* me = static_cast<QMouseEvent*>(event);
            qDebug() << "Mouse pressed on" << obj->objectName();
            return true;  // 拦截事件，不继续传递
        }
        return QObject::eventFilter(obj, event);
    }
};

// 安装过滤器
EventFilter* filter = new EventFilter();
button->installEventFilter(filter);
```

### 应用场景

1. 全局快捷键处理
2. 拖拽功能实现
3. 输入验证
4. 调试事件流

## 5.3 自定义事件如何实现？

### 答案

```cpp
// 1. 定义自定义事件类型
const QEvent::Type MyEventType = static_cast<QEvent::Type>(QEvent::User + 1);

// 2. 定义自定义事件类
class MyEvent : public QEvent {
public:
    MyEvent(const QString& data) 
        : QEvent(MyEventType), m_data(data) {}
    
    QString data() const { return m_data; }
    
private:
    QString m_data;
};

// 3. 发送自定义事件
QCoreApplication::postEvent(receiver, new MyEvent("Hello"));

// 4. 处理自定义事件
class MyWidget : public QWidget {
protected:
    bool event(QEvent* e) override {
        if (e->type() == MyEventType) {
            MyEvent* me = static_cast<MyEvent*>(e);
            qDebug() << "Received:" << me->data();
            return true;
        }
        return QWidget::event(e);
    }
};
```
