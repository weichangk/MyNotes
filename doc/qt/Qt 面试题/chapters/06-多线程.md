## 6.1 Qt 中如何创建和使用线程？

### 答案

### 方式1：继承 QThread（不推荐）

```cpp
class WorkerThread : public QThread {
    void run() override {
        qDebug() << "Thread ID:" << currentThreadId();
    }
};

WorkerThread* thread = new WorkerThread();
thread->start();
```

### 方式2：moveToThread（推荐）

```cpp
class Worker : public QObject {
    Q_OBJECT
public slots:
    void doWork() {
        qDebug() << "Working in thread:" << QThread::currentThread();
        emit resultReady("Done");
    }
signals:
    void resultReady(const QString& result);
};

QThread* thread = new QThread();
Worker* worker = new Worker();
worker->moveToThread(thread);

connect(thread, &QThread::started, worker, &Worker::doWork);
connect(worker, &Worker::resultReady, this, &MainWindow::handleResult);
connect(thread, &QThread::finished, worker, &QObject::deleteLater);

thread->start();
```

### 方式3：QtConcurrent（高层 API）

```cpp
QFuture<int> future = QtConcurrent::run([]() {
    return 42;
});

int result = future.result();  // 阻塞等待结果
```

## 6.2 QMutex、QReadWriteLock、QSemaphore 有什么区别？

### 答案

| 同步原语 | 用途 | 特点 |
|---------|------|------|
| QMutex | 互斥锁 | 保护临界区，同时只有一个线程访问 |
| QReadWriteLock | 读写锁 | 多读单写，读操作可并发 |
| QSemaphore | 信号量 | 控制资源访问数量 |

### QMutex 示例

```cpp
QMutex mutex;
QList<int> sharedData;

void thread1() {
    QMutexLocker locker(&mutex);  // RAII 式加锁
    sharedData.append(42);
}
```

### QReadWriteLock 示例

```cpp
QReadWriteLock rwLock;

void reader() {
    QReadLocker locker(&rwLock);
    qDebug() << data;  // 多个读者可并发
}

void writer() {
    QWriteLocker locker(&rwLock);
    data = newValue;  // 写者独占
}
```

## 6.3 Qt 线程间如何通信？

### 答案

### 1. 信号槽（推荐）

```cpp
// 跨线程信号自动使用 Qt::QueuedConnection
connect(workerThread, &WorkerThread::dataReady, 
        mainWindow, &MainWindow::handleData);
```

### 2. QMetaObject::invokeMethod

```cpp
QMetaObject::invokeMethod(receiver, "slot", 
                          Qt::QueuedConnection,
                          Q_ARG(int, value));
```

### 3. 共享内存 + 互斥锁

```cpp
QMutex mutex;
QList<int> sharedData;

// 线程1
{
    QMutexLocker locker(&mutex);
    sharedData.append(42);
}

// 线程2
{
    QMutexLocker locker(&mutex);
    int value = sharedData.takeFirst();
}
```
