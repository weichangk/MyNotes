## 4.1 Qt 有哪些布局管理器？各有什么特点？

### 答案

Qt 提供了 **5 种主要布局管理器**，用于自动管理控件的位置和大小。

### 布局管理器对比

| 布局管理器 | 用途 | 适用场景 |
|-----------|------|---------|
| QVBoxLayout | 垂直排列 | 表单、菜单、列表 |
| QHBoxLayout | 水平排列 | 工具栏、按钮组 |
| QGridLayout | 网格排列 | 复杂表单、计算器 |
| QFormLayout | 表单排列 | 标签-输入对 |
| QStackedLayout | 堆叠排列 | 多页面切换 |

---

### 1. QVBoxLayout（垂直布局）

**特点**：控件从上到下垂直排列

```cpp
QWidget* window = new QWidget();
QVBoxLayout* layout = new QVBoxLayout(window);

layout->addWidget(new QLabel("姓名:"));
layout->addWidget(new QLineEdit());
layout->addWidget(new QLabel("密码:"));
layout->addWidget(new QLineEdit());
layout->addWidget(new QPushButton("登录"));

window->show();
```

---

### 2. QHBoxLayout（水平布局）

**特点**：控件从左到右水平排列

```cpp
QWidget* window = new QWidget();
QHBoxLayout* layout = new QHBoxLayout(window);

layout->addWidget(new QPushButton("新建"));
layout->addWidget(new QPushButton("打开"));
layout->addWidget(new QPushButton("保存"));
layout->addStretch();  // 添加弹簧，将按钮推到左侧

window->show();
```

---

### 3. QGridLayout（网格布局）

**特点**：控件按网格排列（行列坐标）

```cpp
QWidget* window = new QWidget();
QGridLayout* layout = new QGridLayout(window);

// addWidget(widget, row, col, rowSpan, colSpan)
layout->addWidget(new QLabel("姓名:"), 0, 0);
layout->addWidget(new QLineEdit(), 0, 1);

layout->addWidget(new QLabel("年龄:"), 1, 0);
layout->addWidget(new QSpinBox(), 1, 1);

layout->addWidget(new QLabel("简介:"), 2, 0);
layout->addWidget(new QTextEdit(), 2, 1, 2, 1);  // 跨2行

layout->addWidget(new QPushButton("确定"), 4, 0, 1, 2);  // 跨2列

window->show();
```

---

### 4. QFormLayout（表单布局）

**特点**：自动管理标签-输入对，跨平台自适应

```cpp
QWidget* window = new QWidget();
QFormLayout* layout = new QFormLayout(window);

layout->addRow("姓名:", new QLineEdit());
layout->addRow("年龄:", new QSpinBox());
layout->addRow("邮箱:", new QLineEdit());
layout->addRow("地址:", new QTextEdit());

// macOS 上会右对齐标签，Windows 上会左对齐
window->show();
```

---

### 5. QStackedLayout（堆叠布局）

**特点**：多个控件堆叠，一次只显示一个（类似选项卡）

```cpp
QWidget* window = new QWidget();
QVBoxLayout* mainLayout = new QVBoxLayout(window);

// 创建切换按钮
QHBoxLayout* btnLayout = new QHBoxLayout();
QPushButton* btn1 = new QPushButton("页面1");
QPushButton* btn2 = new QPushButton("页面2");
QPushButton* btn3 = new QPushButton("页面3");
btnLayout->addWidget(btn1);
btnLayout->addWidget(btn2);
btnLayout->addWidget(btn3);

// 创建堆叠布局
QStackedLayout* stackedLayout = new QStackedLayout();
stackedLayout->addWidget(new QLabel("这是页面1"));
stackedLayout->addWidget(new QLabel("这是页面2"));
stackedLayout->addWidget(new QLabel("这是页面3"));

// 连接信号
QObject::connect(btn1, &QPushButton::clicked, [stackedLayout]() {
    stackedLayout->setCurrentIndex(0);
});
QObject::connect(btn2, &QPushButton::clicked, [stackedLayout]() {
    stackedLayout->setCurrentIndex(1);
});
QObject::connect(btn3, &QPushButton::clicked, [stackedLayout]() {
    stackedLayout->setCurrentIndex(2);
});

mainLayout->addLayout(btnLayout);
mainLayout->addLayout(stackedLayout);

window->show();
```

### 布局嵌套示例

```cpp
QWidget* window = new QWidget();
QVBoxLayout* mainLayout = new QVBoxLayout(window);

// 顶部水平布局
QHBoxLayout* topLayout = new QHBoxLayout();

// 底部水平布局

bottomLayout->addStretch();
bottomLayout->addWidget(new QPushButton("确定"));
bottomLayout->addWidget(new QPushButton("取消"));
mainLayout->addLayout(bottomLayout);

window->show();
```

### 布局管理器常用方法

```cpp
// 设置间距
layout->setSpacing(10);              // 控件间距
layout->setContentsMargins(5, 5, 5, 5);  // 边距

// 添加弹簧（占位符）
layout->addStretch(1);               // 弹性伸缩

// 设置拉伸因子
layout->setStretchFactor(widget, 2); // 控件权重

// 设置对齐方式
layout->setAlignment(Qt::AlignCenter);

// 删除控件
layout->removeWidget(widget);
widget->deleteLater();
```

## 4.2 如何实现自定义控件？

### 答案

Qt 提供了 **4 种自定义控件的方式**。

---

### 方式1：继承现有控件（扩展功能）

**适用场景**：在现有控件基础上添加功能

```cpp
// 示例：限制只能输入数字的 QLineEdit
class NumberLineEdit : public QLineEdit {
    Q_OBJECT
public:
    explicit NumberLineEdit(QWidget* parent = nullptr)
        : QLineEdit(parent) {
        // 设置输入验证器
        setValidator(new QIntValidator(0, 999999, this));
    }
    
protected:
    void keyPressEvent(QKeyEvent* event) override {
        // 拦截 Enter 键
        if (event->key() == Qt::Key_Return) {
            emit enterPressed();
            return;
        }
        QLineEdit::keyPressEvent(event);
    }
    
signals:
    void enterPressed();
};
```

---

### 方式2：组合现有控件（封装复杂控件）

**适用场景**：将多个控件组合成新控件

```cpp
// 示例：带清除按钮的搜索框
class SearchBox : public QWidget {
    Q_OBJECT
public:
    explicit SearchBox(QWidget* parent = nullptr)
        : QWidget(parent) {
        // 创建子控件
        m_lineEdit = new QLineEdit(this);
        m_searchBtn = new QPushButton("搜索", this);
        m_clearBtn = new QPushButton("清除", this);
        
        // 布局
        QHBoxLayout* layout = new QHBoxLayout(this);
        layout->addWidget(m_lineEdit);
        layout->addWidget(m_searchBtn);
        layout->addWidget(m_clearBtn);
        layout->setContentsMargins(0, 0, 0, 0);
        
        // 连接信号
        connect(m_clearBtn, &QPushButton::clicked, m_lineEdit, &QLineEdit::clear);
        connect(m_searchBtn, &QPushButton::clicked, this, &SearchBox::onSearch);
    }
    
    QString text() const { return m_lineEdit->text(); }
    void setText(const QString& text) { m_lineEdit->setText(text); }
    
signals:
    void searchRequested(const QString& text);
    
private slots:
    void onSearch() {
        emit searchRequested(m_lineEdit->text());
    }
    
private:
    QLineEdit* m_lineEdit;
    QPushButton* m_searchBtn;
    QPushButton* m_clearBtn;
};
```

---

### 方式3：完全自绘控件（QWidget + paintEvent）

**适用场景**：完全自定义外观的控件

```cpp
// 示例：自定义圆形按钮
class CircleButton : public QWidget {
    Q_OBJECT
public:
    explicit CircleButton(const QString& text, QWidget* parent = nullptr)
        : QWidget(parent), m_text(text), m_pressed(false) {
        setFixedSize(100, 100);
    }
    
    QSize sizeHint() const override {
        return QSize(100, 100);
    }
    
protected:
    void paintEvent(QPaintEvent*) override {
        QPainter painter(this);
        painter.setRenderHint(QPainter::Antialiasing);
        
        // 绘制圆形背景
        QColor color = m_pressed ? QColor(100, 150, 200) : QColor(150, 200, 250);
        painter.setBrush(color);
        painter.setPen(Qt::NoPen);
        painter.drawEllipse(rect());
        
        // 绘制文本
        painter.setPen(Qt::white);
        QFont font = painter.font();
        font.setPointSize(14);
        painter.setFont(font);
        painter.drawText(rect(), Qt::AlignCenter, m_text);
    }
    
    void mousePressEvent(QMouseEvent* event) override {
        if (event->button() == Qt::LeftButton) {
            m_pressed = true;
            update();
        }
    }
    
    void mouseReleaseEvent(QMouseEvent* event) override {
        if (event->button() == Qt::LeftButton) {
            m_pressed = false;
            update();
            if (rect().contains(event->pos())) {
                emit clicked();
            }
        }
    }
    
signals:
    void clicked();
    
private:
    QString m_text;
    bool m_pressed;
};
```

### 方式4：实现交互控件（QAbstractSlider 等抽象类）

**适用场景**：实现滑块、滚动条等交互控件

```cpp
// 示例：自定义音量滑块
class VolumeSlider : public QAbstractSlider {
    Q_OBJECT
public:
    explicit VolumeSlider(QWidget* parent = nullptr)
        : QAbstractSlider(parent) {
        setOrientation(Qt::Horizontal);
        setRange(0, 100);
        setValue(50);
    }
    
protected:
    void paintEvent(QPaintEvent*) override {
        QPainter painter(this);
        painter.setRenderHint(QPainter::Antialiasing);
        
        int w = width();
        int h = height();
        int barHeight = 6;
        int barY = (h - barHeight) / 2;
        
        // 绘制背景槽
        painter.setPen(Qt::NoPen);
        painter.setBrush(QColor(200, 200, 200));
        painter.drawRoundedRect(0, barY, w, barHeight, 3, 3);
        
        // 绘制进度条
        int percent = (value() - minimum()) * 100 / (maximum() - minimum());
        int progressWidth = w * percent / 100;
        painter.setBrush(QColor(66, 133, 244));
        painter.drawRoundedRect(0, barY, progressWidth, barHeight, 3, 3);
        
        // 绘制滑块
        int sliderX = progressWidth - 8;
        painter.setBrush(Qt::white);
        painter.setPen(QPen(QColor(66, 133, 244), 2));
        painter.drawEllipse(sliderX, barY - 6, 16, 16);
    }
    
    void mousePressEvent(QMouseEvent* event) override {
        if (event->button() == Qt::LeftButton) {
            updateValueFromPos(event->pos());
        }
    }
    
    void mouseMoveEvent(QMouseEvent* event) override {
        if (event->buttons() & Qt::LeftButton) {
            updateValueFromPos(event->pos());
        }
    }
    
private:
    void updateValueFromPos(const QPoint& pos) {
        int newValue = minimum() + (maximum() - minimum()) * pos.x() / width();
        newValue = qBound(minimum(), newValue, maximum());
        setValue(newValue);
    }
};
```

---

### 自定义控件最佳实践

```cpp
// 1. 提供 sizeHint() 和 minimumSizeHint()
QSize sizeHint() const override {
    return QSize(200, 50);
}

// 2. 使用 Q_PROPERTY 暴露属性
Q_PROPERTY(QString text READ text WRITE setText NOTIFY textChanged)

// 3. 提供信号通知变化
signals:
    void textChanged(const QString& text);

// 4. 支持样式表（可选）
void paintEvent(QPaintEvent*) override {
    QStyleOption opt;
    opt.initFrom(this);
    QPainter p(this);
    style()->drawPrimitive(QStyle::PE_Widget, &opt, &p, this);
    // ... 自定义绘制
}

// 5. 处理焦点和键盘事件
void focusInEvent(QFocusEvent* event) override {
    update();
    QWidget::focusInEvent(event);
}
```

## 4.3 QWidget、QDialog、QMainWindow 有什么区别？

### 答案

三者都是窗口类，但**用途和功能不同**。

### 对比表

| 特性 | QWidget | QDialog | QMainWindow |
|------|---------|---------|-------------|
| 用途 | 通用控件/窗口 | 对话框 | 主窗口 |
| 父类 | QObject + QPaintDevice | QWidget | QWidget |
| 特有功能 | 无 | exec()、accept()、reject() | 菜单栏、工具栏、状态栏、停靠窗口 |
| 典型场景 | 自定义控件、简单窗口 | 设置窗口、提示框 | 应用程序主界面 |
| 模态支持 | ❌ | ✅ | ❌ |
| 关闭行为 | 隐藏 | 隐藏 + 返回值 | 退出应用 |

---

### 1. QWidget（通用窗口/控件）

**特点**：最基础的可视化类，既可作为窗口，也可作为控件

```cpp
// 作为顶层窗口
QWidget* window = new QWidget();
window->setWindowTitle("简单窗口");
window->resize(400, 300);
window->show();

// 作为容器控件
QWidget* container = new QWidget(parent);
QVBoxLayout* layout = new QVBoxLayout(container);
layout->addWidget(new QLabel("子控件"));

// 作为自定义控件的基类
class MyWidget : public QWidget {
    // ...
};
```

---

### 2. QDialog（对话框）

**特点**：支持模态/非模态显示，有返回值（accept/reject）

#### 模态对话框（阻塞父窗口）

```cpp
QDialog dialog(parent);
dialog.setWindowTitle("设置");

QVBoxLayout* layout = new QVBoxLayout(&dialog);
layout->addWidget(new QLabel("请输入名称:"));
QLineEdit* lineEdit = new QLineEdit();
layout->addWidget(lineEdit);

QDialogButtonBox* btnBox = new QDialogButtonBox(
    QDialogButtonBox::Ok | QDialogButtonBox::Cancel);
layout->addWidget(btnBox);

QObject::connect(btnBox, &QDialogButtonBox::accepted, &dialog, &QDialog::accept);
QObject::connect(btnBox, &QDialogButtonBox::rejected, &dialog, &QDialog::reject);

if (dialog.exec() == QDialog::Accepted) {
    // 用户点击了"确定"
    qDebug() << "输入:" << lineEdit->text();
} else {
    // 用户点击了"取消"
    qDebug() << "取消";
}
```

#### 非模态对话框（不阻塞）

```cpp
QDialog* dialog = new QDialog(parent);
dialog->setWindowTitle("非模态对话框");
dialog->show();  // 使用 show() 而不是 exec()

// 处理返回值
QObject::connect(dialog, &QDialog::accepted, [dialog]() {
    qDebug() << "对话框接受";
    dialog->deleteLater();
});
```

#### 标准对话框示例

```cpp
// 消息框
QMessageBox::information(this, "提示", "操作成功");

// 文件对话框
QString fileName = QFileDialog::getOpenFileName(
    this, "打开文件", "", "文本文件 (*.txt);;所有文件 (*.*)");

// 颜色对话框
QColor color = QColorDialog::getColor(Qt::white, this, "选择颜色");

// 输入对话框
QString text = QInputDialog::getText(this, "输入", "请输入名称:");
```

### 3. QMainWindow（主窗口）

**特点**：应用程序主界面，支持菜单栏、工具栏、状态栏、停靠窗口

```cpp
class MainWindow : public QMainWindow {
    Q_OBJECT
public:
    MainWindow() {
        // 1. 设置中心控件
        QTextEdit* textEdit = new QTextEdit(this);
        setCentralWidget(textEdit);
        
        // 2. 创建菜单栏
        QMenuBar* menuBar = this->menuBar();
        QMenu* fileMenu = menuBar->addMenu("文件");
        QAction* newAction = fileMenu->addAction("新建");
        QAction* openAction = fileMenu->addAction("打开");
        fileMenu->addSeparator();
        QAction* exitAction = fileMenu->addAction("退出");
        
        QMenu* editMenu = menuBar->addMenu("编辑");
        editMenu->addAction("复制");
        editMenu->addAction("粘贴");
        
        // 3. 创建工具栏
        QToolBar* toolBar = addToolBar("主工具栏");
        toolBar->addAction(newAction);
        toolBar->addAction(openAction);
        toolBar->addSeparator();
        toolBar->addAction("保存");
        
        // 4. 创建状态栏
        QStatusBar* statusBar = this->statusBar();
        statusBar->showMessage("就绪", 2000);  // 显示2秒
        
        // 永久显示的状态标签
        QLabel* permanentLabel = new QLabel("行: 1  列: 1");
        statusBar->addPermanentWidget(permanentLabel);
        
        // 5. 创建停靠窗口
        QDockWidget* dockWidget = new QDockWidget("工具面板", this);
        dockWidget->setWidget(new QListWidget());
        addDockWidget(Qt::LeftDockWidgetArea, dockWidget);
        
        // 连接信号
        connect(exitAction, &QAction::triggered, qApp, &QApplication::quit);
        connect(newAction, &QAction::triggered, textEdit, &QTextEdit::clear);
    }
};

// 使用
int main(int argc, char* argv[]) {
    QApplication app(argc, argv);
    MainWindow window;
    window.resize(800, 600);
    window.show();
    return app.exec();
}
```

### 选择建议

| 需求 | 推荐 |
|------|------|
| 自定义控件 | QWidget |
| 简单窗口（无菜单工具栏） | QWidget |
| 需要返回值的窗口 | QDialog |
| 设置窗口、输入窗口 | QDialog |
| 应用程序主界面 | QMainWindow |
| 需要菜单栏/工具栏 | QMainWindow |

---

### 常见陷阱

#### ❌ 错误：用 QWidget 实现需要返回值的窗口

```cpp
// 不推荐：QWidget 没有 exec() 和 accept/reject
QWidget* widget = new QWidget();
// widget->exec();  // ❌ 编译错误
```

#### ✅ 正确：使用 QDialog

```cpp
QDialog dialog;
if (dialog.exec() == QDialog::Accepted) {
    // ...
}
```

---

#### ❌ 错误：在 QMainWindow 中直接添加控件

```cpp
QMainWindow window;
QPushButton* btn = new QPushButton("按钮", &window);  // ❌ 位置错误
```

#### ✅ 正确：设置中心控件

```cpp
QMainWindow window;
QWidget* central = new QWidget();
QVBoxLayout* layout = new QVBoxLayout(central);
layout->addWidget(new QPushButton("按钮"));
window.setCentralWidget(central);
```

## 本章小结

### 核心知识点

1. **5 种布局管理器**
   - QVBoxLayout/QHBoxLayout（单向排列）
   - QGridLayout（网格排列）
   - QFormLayout（表单排列）
   - QStackedLayout（堆叠切换）

2. **4 种自定义控件方式**
   - 继承现有控件（扩展功能）
   - 组合现有控件（封装复杂控件）
   - 完全自绘控件（paintEvent）
   - 实现交互控件（抽象类）

3. **3 种窗口类**
   - QWidget：通用控件/窗口
   - QDialog：对话框（模态/非模态）
   - QMainWindow：主窗口（菜单/工具栏/状态栏）

### 最佳实践

- ✅ 优先使用布局管理器，避免硬编码坐标
- ✅ 布局嵌套时注意逻辑清晰
- ✅ 自定义控件提供 sizeHint() 和信号
- ✅ 根据场景选择合适的窗口类
- ✅ QMainWindow 必须设置中心控件
