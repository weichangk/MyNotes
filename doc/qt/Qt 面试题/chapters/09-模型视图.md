## 9.1 什么是模型/视图架构？有什么优势?

### 答案

模型/视图架构（Model/View Architecture）是 Qt 提供的一种**数据与界面分离**的设计模式。

### 核心概念

```
模型（Model）  ←→  视图（View）
    ↓                 ↑
  数据存储          数据显示
    ↓                 ↑
    └────委托(Delegate)────┘
         (自定义绘制/编辑)
```

### 三个核心组件

| 组件 | 职责 | 常用类 |
|------|------|--------|
| Model | 管理数据 | QStandardItemModel、QSqlTableModel |
| View | 显示数据 | QTableView、QListView、QTreeView |
| Delegate | 自定义渲染/编辑 | QStyledItemDelegate |

### 优势

1. **数据与界面分离**：一个模型可以被多个视图共享
2. **易于维护**：修改数据显示方式不影响数据存储
3. **高效**：只渲染可见区域，支持大数据集
4. **灵活**：可自定义渲染和编辑器

---

### 简单示例

```cpp
#include <QApplication>
#include <QTableView>
#include <QStandardItemModel>

int main(int argc, char* argv[]) {
    QApplication app(argc, argv);
    
    // 1. 创建模型
    QStandardItemModel model(3, 2);  // 3行2列
    model.setHorizontalHeaderLabels({"姓名", "年龄"});
    
    // 2. 填充数据
    model.setItem(0, 0, new QStandardItem("张三"));
    model.setItem(0, 1, new QStandardItem("25"));
    model.setItem(1, 0, new QStandardItem("李四"));
    model.setItem(1, 1, new QStandardItem("30"));
    model.setItem(2, 0, new QStandardItem("王五"));
    model.setItem(2, 1, new QStandardItem("28"));
    
    // 3. 创建视图并绑定模型
    QTableView view;
    view.setModel(&model);
    view.resize(400, 300);
    view.show();
    
    return app.exec();
}
```

## 9.2 如何实现自定义模型？

### 答案

自定义模型需要**继承 QAbstractItemModel 或 QAbstractTableModel**。

### 必须实现的方法

| 方法 | 说明 |
|------|------|
| rowCount() | 返回行数 |
| columnCount() | 返回列数 |
| data() | 返回单元格数据 |
| index() | 创建模型索引（仅树形模型） |
| parent() | 返回父索引（仅树形模型） |

### 可选实现的方法

| 方法 | 说明 |
|------|------|
| setData() | 设置单元格数据（可编辑模型） |
| flags() | 返回单元格标志（可编辑/可选中） |
| headerData() | 返回表头数据 |
| insertRows() / removeRows() | 插入/删除行 |

---

### 示例：自定义表格模型

```cpp
#include <QAbstractTableModel>
#include <QVector>

struct Person {
    QString name;
    int age;
};

class PersonModel : public QAbstractTableModel {
    Q_OBJECT
public:
    explicit PersonModel(QObject* parent = nullptr)
        : QAbstractTableModel(parent) {
        // 初始化数据
        m_persons = {
            {"张三", 25},
            {"李四", 30},
            {"王五", 28}
        };
    }
    
    // 1. 必须实现：行数
    int rowCount(const QModelIndex& parent = QModelIndex()) const override {
        Q_UNUSED(parent);
        return m_persons.size();
    }
    
    // 2. 必须实现：列数
    int columnCount(const QModelIndex& parent = QModelIndex()) const override {
        Q_UNUSED(parent);
        return 2;  // 姓名、年龄
    }
    
    // 3. 必须实现：返回数据
    QVariant data(const QModelIndex& index, int role = Qt::DisplayRole) const override {
        if (!index.isValid() || index.row() >= m_persons.size()) {
            return QVariant();
        }
        
        const Person& person = m_persons[index.row()];
        
        if (role == Qt::DisplayRole || role == Qt::EditRole) {
            switch (index.column()) {
                case 0: return person.name;
                case 1: return person.age;
            }
        }
        
        return QVariant();
    }
    
    // 4. 可选：返回表头
    QVariant headerData(int section, Qt::Orientation orientation,
                       int role = Qt::DisplayRole) const override {
        if (role == Qt::DisplayRole && orientation == Qt::Horizontal) {
            return section == 0 ? "姓名" : "年龄";
        }
        return QVariant();
    }
    
    // 5. 可选：设置单元格标志（可编辑）
    Qt::ItemFlags flags(const QModelIndex& index) const override {
        if (!index.isValid()) {
            return Qt::NoItemFlags;
        }
        return Qt::ItemIsEnabled | Qt::ItemIsSelectable | Qt::ItemIsEditable;
    }
    
    // 6. 可选：设置数据（支持编辑）
    bool setData(const QModelIndex& index, const QVariant& value,
                 int role = Qt::EditRole) override {
        if (!index.isValid() || index.row() >= m_persons.size() || role != Qt::EditRole) {
            return false;
        }
        
        Person& person = m_persons[index.row()];
        
        switch (index.column()) {
            case 0:
                person.name = value.toString();
                break;
            case 1:
                person.age = value.toInt();
                break;
            default:
                return false;
        }
        
        emit dataChanged(index, index, {role});
        return true;
    }
    
    // 7. 自定义方法：添加数据
    void addPerson(const QString& name, int age) {
        int row = m_persons.size();
        beginInsertRows(QModelIndex(), row, row);
        m_persons.append({name, age});
        endInsertRows();
    }
    
    // 8. 自定义方法：删除数据
    void removePerson(int row) {
        if (row < 0 || row >= m_persons.size()) {
            return;
        }
        beginRemoveRows(QModelIndex(), row, row);
        m_persons.removeAt(row);
        endRemoveRows();
    }
    
private:
    QVector<Person> m_persons;
};
```

### 使用自定义模型

```cpp
PersonModel* model = new PersonModel();
QTableView* view = new QTableView();
view->setModel(model);
view->show();

// 添加数据
model->addPerson("赵六", 35);

// 删除数据
model->removePerson(0);
```

## 9.3 如何实现自定义委托（Delegate）？

### 答案

自定义委托用于**自定义单元格的显示和编辑器**，需要继承 `QStyledItemDelegate`。

### 可重写的方法

| 方法 | 说明 |
|------|------|
| paint() | 自定义绘制 |
| createEditor() | 创建自定义编辑器 |
| setEditorData() | 将模型数据设置到编辑器 |
| setModelData() | 将编辑器数据写回模型 |
| updateEditorGeometry() | 更新编辑器位置和大小 |
| sizeHint() | 返回单元格大小提示 |

---

### 示例1：自定义绘制（进度条）

```cpp
#include <QStyledItemDelegate>
#include <QPainter>
#include <QApplication>

class ProgressBarDelegate : public QStyledItemDelegate {
    Q_OBJECT
public:
    using QStyledItemDelegate::QStyledItemDelegate;
    
    void paint(QPainter* painter, const QStyleOptionViewItem& option,
               const QModelIndex& index) const override {
        if (index.column() == 1) {  // 仅在第2列绘制进度条
            int progress = index.data().toInt();
            
            // 绘制背景
            painter->save();
            if (option.state & QStyle::State_Selected) {
                painter->fillRect(option.rect, option.palette.highlight());
            }
            
            // 绘制进度条
            QRect progressRect = option.rect.adjusted(5, 5, -5, -5);
            painter->setPen(Qt::NoPen);
            
            // 背景
            painter->setBrush(QColor(200, 200, 200));
            painter->drawRoundedRect(progressRect, 3, 3);
            
            // 进度
            int width = progressRect.width() * progress / 100;
            QRect fillRect = progressRect;
            fillRect.setWidth(width);
            painter->setBrush(QColor(76, 175, 80));
            painter->drawRoundedRect(fillRect, 3, 3);
            
            // 文本
            painter->setPen(Qt::black);
            painter->drawText(progressRect, Qt::AlignCenter, QString("%1%").arg(progress));
            
            painter->restore();
        } else {
            QStyledItemDelegate::paint(painter, option, index);
        }
    }
};
```

### 示例2：自定义编辑器（下拉框）

```cpp
#include <QStyledItemDelegate>
#include <QComboBox>

class ComboBoxDelegate : public QStyledItemDelegate {
    Q_OBJECT
public:
    using QStyledItemDelegate::QStyledItemDelegate;
    
    // 1. 创建编辑器
    QWidget* createEditor(QWidget* parent, const QStyleOptionViewItem& option,
                         const QModelIndex& index) const override {
        Q_UNUSED(option);
        Q_UNUSED(index);
        
        QComboBox* comboBox = new QComboBox(parent);
        comboBox->addItems({"初级", "中级", "高级", "专家"});
        return comboBox;
    }
    
    // 2. 设置编辑器数据
    void setEditorData(QWidget* editor, const QModelIndex& index) const override {
        QString value = index.data(Qt::EditRole).toString();
        QComboBox* comboBox = static_cast<QComboBox*>(editor);
        comboBox->setCurrentText(value);
    }
    
    // 3. 将编辑器数据写回模型
    void setModelData(QWidget* editor, QAbstractItemModel* model,
                     const QModelIndex& index) const override {
        QComboBox* comboBox = static_cast<QComboBox*>(editor);
        model->setData(index, comboBox->currentText(), Qt::EditRole);
    }
    
    // 4. 更新编辑器几何
    void updateEditorGeometry(QWidget* editor, const QStyleOptionViewItem& option,
                             const QModelIndex& index) const override {
        Q_UNUSED(index);
        editor->setGeometry(option.rect);
    }
};
```

### 使用委托

```cpp
QTableView* view = new QTableView();
view->setModel(model);

// 为整个视图设置委托
view->setItemDelegate(new ProgressBarDelegate());

// 为特定列设置委托
view->setItemDelegateForColumn(2, new ComboBoxDelegate());
```

### 示例3：复杂自定义委托（图文混排）

```cpp
class ImageTextDelegate : public QStyledItemDelegate {
    Q_OBJECT
public:
    using QStyledItemDelegate::QStyledItemDelegate;
    
    void paint(QPainter* painter, const QStyleOptionViewItem& option,
               const QModelIndex& index) const override {
        painter->save();
        
        // 绘制选中背景
        if (option.state & QStyle::State_Selected) {
            painter->fillRect(option.rect, option.palette.highlight());
        }
        
        QRect rect = option.rect.adjusted(5, 5, -5, -5);
        
        // 绘制图标
        QIcon icon = qvariant_cast<QIcon>(index.data(Qt::DecorationRole));
        QRect iconRect(rect.left(), rect.top(), 32, 32);
        icon.paint(painter, iconRect);
        
        // 绘制文本
        QString text = index.data(Qt::DisplayRole).toString();
        QRect textRect = rect.adjusted(40, 0, 0, 0);
        painter->setPen(option.palette.text().color());
        painter->drawText(textRect, Qt::AlignLeft | Qt::AlignVCenter, text);
        
        painter->restore();
    }
    
    QSize sizeHint(const QStyleOptionViewItem& option,
                   const QModelIndex& index) const override {
        Q_UNUSED(option);
        Q_UNUSED(index);
        return QSize(200, 42);
    }
};
```

## 9.4 QTableView、QListView、QTreeView 如何使用？

### 答案

三种视图类分别用于显示**表格、列表、树形**数据。

### 视图对比

| 视图 | 数据结构 | 常用模型 |
|------|---------|---------|
| QTableView | 二维表格 | QStandardItemModel、QSqlTableModel |
| QListView | 一维列表 | QStringListModel、QStandardItemModel |
| QTreeView | 树形结构 | QStandardItemModel、QFileSystemModel |

---

### 1. QTableView（表格视图）

```cpp
#include <QTableView>
#include <QStandardItemModel>

QStandardItemModel* model = new QStandardItemModel(3, 3);
model->setHorizontalHeaderLabels({"列1", "列2", "列3"});

for (int row = 0; row < 3; ++row) {
    for (int col = 0; col < 3; ++col) {
        QStandardItem* item = new QStandardItem(
            QString("单元格(%1,%2)").arg(row).arg(col));
        model->setItem(row, col, item);
    }
}

QTableView* view = new QTableView();
view->setModel(model);

// 常用设置
view->setSelectionBehavior(QAbstractItemView::SelectRows);  // 选中整行
view->setSelectionMode(QAbstractItemView::SingleSelection);  // 单选
view->setEditTriggers(QAbstractItemView::DoubleClicked);     // 双击编辑
view->horizontalHeader()->setStretchLastSection(true);       // 最后一列拉伸
view->verticalHeader()->setVisible(false);                   // 隐藏行号

view->show();
```

---

### 2. QListView（列表视图）

```cpp
#include <QListView>
#include <QStringListModel>

// 方式1：使用 QStringListModel
QStringList list = {"项目1", "项目2", "项目3", "项目4"};
QStringListModel* model = new QStringListModel(list);

QListView* view = new QListView();
view->setModel(model);

// 常用设置
view->setViewMode(QListView::IconMode);      // 图标模式
view->setIconSize(QSize(64, 64));
view->setSpacing(10);
view->setMovement(QListView::Static);        // 禁止拖动

view->show();

// 方式2：使用 QStandardItemModel（支持图标）
QStandardItemModel* iconModel = new QStandardItemModel();
for (int i = 0; i < 5; ++i) {
    QStandardItem* item = new QStandardItem(
        QIcon(":/icons/file.png"), QString("文件%1").arg(i + 1));
    iconModel->appendRow(item);
}

QListView* iconView = new QListView();
iconView->setModel(iconModel);
iconView->setViewMode(QListView::IconMode);
iconView->show();
```

### 3. QTreeView（树形视图）

```cpp
#include <QTreeView>
#include <QStandardItemModel>

QStandardItemModel* model = new QStandardItemModel();
model->setHorizontalHeaderLabels({"名称", "类型", "大小"});

// 创建根节点
QStandardItem* rootItem = model->invisibleRootItem();

// 创建一级节点
QStandardItem* folderItem = new QStandardItem(QIcon(":/icons/folder.png"), "文件夹1");
QStandardItem* folderType = new QStandardItem("文件夹");
QStandardItem* folderSize = new QStandardItem("--");

rootItem->appendRow({folderItem, folderType, folderSize});

// 创建二级节点
QStandardItem* fileItem1 = new QStandardItem(QIcon(":/icons/file.png"), "文件1.txt");
QStandardItem* fileType1 = new QStandardItem("文本文件");
QStandardItem* fileSize1 = new QStandardItem("1.2 KB");

folderItem->appendRow({fileItem1, fileType1, fileSize1});

QStandardItem* fileItem2 = new QStandardItem(QIcon(":/icons/file.png"), "文件2.txt");
QStandardItem* fileType2 = new QStandardItem("文本文件");
QStandardItem* fileSize2 = new QStandardItem("2.5 KB");

folderItem->appendRow({fileItem2, fileType2, fileSize2});

QTreeView* view = new QTreeView();
view->setModel(model);

// 常用设置
view->setIndentation(20);                              // 缩进
view->setAnimated(true);                               // 展开/折叠动画
view->expandAll();                                     // 展开所有
view->header()->setSectionResizeMode(QHeaderView::ResizeToContents);

view->show();
```

```cpp
// 响应选中变化
connect(view->selectionModel(), &QItemSelectionModel::currentChanged,
        [](const QModelIndex& current, const QModelIndex& previous) {
    Q_UNUSED(previous);
    QString text = current.data().toString();
    qDebug() << "选中:" << text;

    qDebug() << "双击:" << index.row() << index.column();

view->setContextMenuPolicy(Qt::CustomContextMenu);
connect(view, &QTableView::customContextMenuRequested,
        [view](const QPoint& pos) {
    QModelIndex index = view->indexAt(pos);
    if (!index.isValid()) return;
    
    QMenu menu;
    menu.addAction("编辑");
    menu.addAction("删除");
    menu.exec(view->mapToGlobal(pos));
});
```

## 本章小结

### 核心知识点

1. **模型/视图架构**
   - Model（数据管理）
   - View（数据显示）
   - Delegate（自定义渲染/编辑）

2. **自定义模型**
   - 继承 QAbstractTableModel 或 QAbstractItemModel
   - 实现 rowCount()、columnCount()、data()
   - 实现 setData()、flags() 支持编辑

3. **自定义委托**
   - 继承 QStyledItemDelegate
   - 重写 paint() 自定义绘制
   - 重写 createEditor() 自定义编辑器

4. **三种视图**
   - QTableView：表格数据
   - QListView：列表数据
   - QTreeView：树形数据

### 最佳实践

- ✅ 数据量大时使用模型/视图（而非 QTableWidget）
- ✅ 一个模型可被多个视图共享
- ✅ 使用 beginInsertRows()/endInsertRows() 修改数据
- ✅ 委托中避免复杂计算，影响性能
- ✅ 使用 QFileSystemModel 显示文件系统
