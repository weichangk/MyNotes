## 1.1 什么是 Qt？Qt 有哪些特点？

### 答案

Qt 是一个**跨平台的 C++ 图形用户界面应用程序开发框架**，由 Qt Company 开发和维护。

### 主要特点

#### 1. 跨平台性
- 支持 Windows、macOS、Linux、Android、iOS、嵌入式等多个平台
- **"一次编写，到处编译"**（Write Once, Compile Everywhere）
- 通过 QPA（Qt Platform Abstraction）抽象平台差异

#### 2. 丰富的类库
- **GUI 控件**：QPushButton、QLabel、QTableView 等
- **网络编程**：QNetworkAccessManager、QTcpSocket 等
- **数据库**：QSqlDatabase、QSqlQuery 等
- **多媒体**：QMediaPlayer、QCamera 等
- **图形渲染**：QPainter、QGraphicsView 等

#### 3. 信号与槽机制
- 松耦合的对象间通信机制
- 类型安全、支持多对多连接
- 可跨线程传递信号

#### 4. 元对象系统
- 运行时类型信息（RTTI）
- 属性系统（Q_PROPERTY）
- 动态调用（QMetaObject::invokeMethod）

#### 5. 国际化支持
- 内置翻译系统（QTranslator、tr()）
- Unicode 支持

#### 6. 工具链完善
- Qt Creator IDE
- Qt Designer（UI 设计器）
- qmake/CMake 构建系统
- Qt Linguist（翻译工具）

## 1.2 Qt 的核心模块有哪些？

### 答案

Qt 分为多个模块，常用的核心模块包括：

| 模块 | 说明 |
|------|------|
| **Qt Core** | 核心非 GUI 功能，包括事件循环、容器类、元对象系统等 |
| **Qt GUI** | GUI 基础类，包括窗口系统、事件处理、字体、图像等 |
| **Qt Widgets** | 传统桌面风格控件，如 QPushButton、QTableWidget 等 |
| **Qt Network** | 网络编程，支持 TCP/UDP、HTTP、WebSocket 等 |
| **Qt SQL** | 数据库集成，支持 SQLite、MySQL、PostgreSQL 等 |
| **Qt Multimedia** | 多媒体功能，音视频播放、摄像头等 |
| **Qt QML** | QML 声明式 UI 语言及 Qt Quick 框架 |
| **Qt Concurrent** | 多线程编程，提供 QtConcurrent::run 等高层 API |
| **Qt Test** | 单元测试框架 |
| **Qt XML** | XML 解析（DOM、SAX）|

### 扩展模块
- Qt WebEngine（Chromium 浏览器引擎）
- Qt Charts（图表绘制）
- Qt 3D（3D 渲染）
- Qt Bluetooth/NFC（蓝牙/NFC 通信）
- Qt Positioning（GPS 定位）

## 1.3 Qt 对象树是什么？有什么作用？

### 答案

**Qt 对象树（Object Tree）** 是 Qt 内存管理的核心机制，基于父子关系构建。

### 核心概念

```cpp
QWidget* parent = new QWidget();
QLabel* label = new QLabel(parent);         // label 的父对象是 parent
QPushButton* btn = new QPushButton(parent); // btn 的父对象是 parent
```

**对象树结构**：
```
parent (QWidget)
├─ label (QLabel)
└─ btn (QPushButton)
```

### 作用

#### 1. 自动内存管理
- 父对象析构时，自动递归删除所有子对象
- 避免手动 delete 导致的内存泄漏

#### 2. 层次化组织
- 清晰的对象所有权关系
- 方便查找子对象（`findChild<T>()`、`findChildren<T>()`）

#### 3. 事件传播
- 子控件的事件可以向父控件传播
- 父控件可以过滤子控件的事件（`installEventFilter`）

### 示例

```cpp
void createUI() {
    QWidget* window = new QWidget();
    
    QVBoxLayout* layout = new QVBoxLayout(window);
    QLabel* label = new QLabel("Hello", window);
    QPushButton* btn = new QPushButton("Click", window);
    
    layout->addWidget(label);
    layout->addWidget(btn);
    
    window->show();
    
    // 只需 delete window，label 和 btn 会自动释放
}
```

### 注意事项

#### 1. 不在对象树中的对象需要手动管理
```cpp
QObject* obj = new QObject();  // 无父对象
// ...
delete obj;  // 必须手动删除
```

#### 2. 栈对象不受对象树管理
```cpp
void func() {
    QWidget window;
    QLabel label(&window);  // label 生命周期由栈管理
}  // 离开作用域，自动析构
```

#### 3. 避免重复删除
```cpp
QWidget* parent = new QWidget();
QLabel* label = new QLabel(parent);
delete label;   // 手动删除
delete parent;  // parent 析构时不会再删除 label（已被移除）
```

## 1.4 Qt 中的 QObject 有什么特殊之处？

### 答案

`QObject` 是 Qt 对象模型的基类，几乎所有 Qt 类都继承自 `QObject`。

### 核心特性

#### 1. 元对象系统（Meta-Object System）

```cpp
class MyClass : public QObject {
    Q_OBJECT  // 必须声明，启用元对象功能
    
public:
    MyClass(QObject* parent = nullptr) : QObject(parent) {}
    
signals:
    void mySignal();
    
public slots:
    void mySlot() {
        qDebug() << "Slot called";
    }
};
```

**元对象功能**：
- 信号与槽机制
- 运行时类型信息（`metaObject()->className()`）
- 属性系统（`Q_PROPERTY`）
- 动态调用（`QMetaObject::invokeMethod`）

#### 2. 父子关系管理

```cpp
QObject* parent = new QObject();
QObject* child1 = new QObject(parent);
QObject* child2 = new QObject(parent);

// parent 析构时自动删除 child1 和 child2
delete parent;
```

#### 3. 对象名称与查找

```cpp
QObject* obj = new QObject();
obj->setObjectName("myObject");

// 按名称查找
QObject* found = parent->findChild<QObject*>("myObject");
```

#### 4. 事件处理

```cpp
class MyObject : public QObject {
protected:
    bool event(QEvent* e) override {
        if (e->type() == QEvent::Timer) {
            // 处理定时器事件
            return true;
        }
        return QObject::event(e);
    }
};
```

#### 5. 定时器支持

```cpp
QObject* obj = new QObject();
int timerId = obj->startTimer(1000);  // 1 秒定时器

// 重写 timerEvent
void timerEvent(QTimerEvent* event) override {
    if (event->timerId() == m_timerId) {
        qDebug() << "Timer fired";
    }
}
```

#### 6. 线程亲和性（Thread Affinity）

```cpp
QObject* obj = new QObject();
qDebug() << obj->thread();  // 返回对象所属线程

// 移动对象到另一个线程
QThread* thread = new QThread();
obj->moveToThread(thread);
thread->start();
```

#### 7. 信号阻塞

```cpp
QObject* obj = new QObject();
obj->blockSignals(true);   // 阻塞信号发射
emit obj->someSignal();    // 信号不会发射
obj->blockSignals(false);  // 恢复信号
```

### 限制

#### 1. 不可拷贝
```cpp
QObject obj1;
QObject obj2 = obj1;  // ❌ 编译错误，拷贝构造被禁用
```

#### 2. 必须在堆上创建或作为成员
- 跨线程使用时必须在堆上创建
- 栈对象不能跨线程移动

#### 3. Q_OBJECT 宏的限制
- 必须在类声明的 private 区域之前
- 模板类不能使用 Q_OBJECT

## 1.5 Qt 的坐标系统是怎样的？

### 答案

Qt 使用 **二维笛卡尔坐标系**，原点在控件左上角，X 轴向右，Y 轴向下。

### 基本坐标系

```
(0,0) ─────────────────> X
  │
  │  Widget
  │
  │
  ▼
  Y
```

### 常用坐标相关类

| 类 | 说明 |
|----|------|
| `QPoint` | 整数坐标点 |
| `QPointF` | 浮点数坐标点 |
| `QRect` | 整数矩形（x, y, width, height）|
| `QRectF` | 浮点数矩形 |
| `QSize` | 尺寸（width, height）|

### 坐标转换

#### 1. 本地坐标 vs 全局坐标

```cpp
QWidget* widget = new QWidget();
QPoint localPos(10, 20);  // 相对于 widget 左上角

// 转换为全局坐标（屏幕坐标）
QPoint globalPos = widget->mapToGlobal(localPos);

// 转换回本地坐标
QPoint local = widget->mapFromGlobal(globalPos);
```

#### 2. 父子控件坐标转换

```cpp
QWidget* parent = new QWidget();
QWidget* child = new QWidget(parent);

QPoint childPos(10, 20);  // 相对于 child 的坐标

// 转换为父控件坐标
QPoint parentPos = child->mapToParent(childPos);

// 转换为指定控件的坐标
QPoint otherPos = child->mapTo(parent, childPos);
```

#### 3. Graphics View 坐标系统

Graphics View 框架有多层坐标系统：

```cpp
// 场景坐标（Scene）
QPointF scenePos(100, 200);

// 视图坐标（View，屏幕像素）
QPoint viewPos = view->mapFromScene(scenePos);

// Item 本地坐标
QPointF itemPos = item->mapFromScene(scenePos);
```

### Hi-DPI 支持

```cpp
// Qt 5.6+ 支持高 DPI 缩放
QWidget* widget = new QWidget();

// 获取设备像素比
qreal dpr = widget->devicePixelRatioF();  // 例如：2.0（Retina）

// 逻辑像素 vs 物理像素
int logicalWidth = 100;
int physicalWidth = logicalWidth * dpr;  // 200
```

### 常见坐标操作

```cpp
// 获取控件位置
QPoint pos = widget->pos();           // 相对于父控件
QPoint globalPos = widget->mapToGlobal(QPoint(0, 0));  // 屏幕坐标

// 获取控件尺寸
QSize size = widget->size();
QRect rect = widget->rect();          // (0, 0, width, height)
QRect geometry = widget->geometry();  // 相对于父控件的几何矩形

// 移动控件
widget->move(100, 200);               // 移动到父控件的 (100, 200)
widget->setGeometry(100, 200, 300, 400);  // 设置位置和大小
```
