# 属性动画

## 1. 你需要先建立的认知

Qt 动画框架（Qt Animation Framework）的核心思想是"对 `QObject` 的属性随时间做插值"。
整套 API 在模块 `QtCore` 中，无需额外依赖。

### 1.1 核心类继承树

```
QAbstractAnimation
├── QVariantAnimation          ← 插值引擎基类
│   └── QPropertyAnimation     ← 最常用：直接驱动 Q_PROPERTY
└── QAnimationGroup            ← 动画编排
    ├── QSequentialAnimationGroup  ← 顺序播放
    └── QParallelAnimationGroup    ← 并行播放
```

### 1.2 缓动曲线（Easing Curve）

动画的"节奏感"来自 `QEasingCurve`，它决定插值随时间的变化方式：

| 常用类型 | 含义 |
|---|---|
| `QEasingCurve::Linear` | 匀速 |
| `QEasingCurve::InQuad` / `OutQuad` / `InOutQuad` | 二次曲线加速/减速 |
| `QEasingCurve::InOutCubic` | 三次曲线平滑进出 |
| `QEasingCurve::OutBounce` | 弹跳收尾 |
| `QEasingCurve::InOutElastic` | 弹性效果 |
| `QEasingCurve::InOutBack` | 超出后回弹 |


## 3. QPropertyAnimation 全解

### 3.1 最小可运行示例

```cpp
#include <QPropertyAnimation>
#include <QPushButton>

// 按钮从 x=0 移到 x=300，持续 800ms
auto* anim = new QPropertyAnimation(button, "pos", this);
anim->setDuration(800);
anim->setStartValue(QPoint(0, 50));
anim->setEndValue(QPoint(300, 50));
anim->setEasingCurve(QEasingCurve::OutCubic);
anim->start(QAbstractAnimation::DeleteWhenStopped);
```

**注意**：`"pos"` 必须对应目标对象上一个 `READ`/`WRITE` 的 `Q_PROPERTY`。

### 3.2 生命周期管理

`QPropertyAnimation` 构造时传入父对象，不用手动 `delete`；或者用：

```cpp
anim->start(QAbstractAnimation::DeleteWhenStopped); // 停止后自动销毁
```

两种策略选一种，避免泄漏或 double-free。

### 3.3 关键帧：`setKeyValueAt`

不局限于起止两端，可以指定任意中间值：

```cpp
auto* anim = new QPropertyAnimation(label, "windowOpacity", this);
anim->setDuration(1200);
anim->setKeyValueAt(0.0, 1.0);   // 起始：完全不透明
anim->setKeyValueAt(0.4, 0.0);   // 40% 处：完全透明
anim->setKeyValueAt(0.6, 0.0);   // 60% 处：保持透明
anim->setKeyValueAt(1.0, 1.0);   // 结束：恢复不透明（闪烁效果）
anim->setLoopCount(3);           // 循环 3 次
anim->start(QAbstractAnimation::DeleteWhenStopped);
```

### 3.4 循环与往返

```cpp
anim->setLoopCount(-1);  // 无限循环
// 往返效果：手动接信号反转
connect(anim, &QPropertyAnimation::finished, this, [anim]() {
    if (anim->direction() == QAbstractAnimation::Forward)
        anim->setDirection(QAbstractAnimation::Backward);
    else
        anim->setDirection(QAbstractAnimation::Forward);
    anim->start();
});
```

### 3.5 可动画化的常见 Q_PROPERTY

| 属性名 | 类型 | 适用对象 |
|---|---|---|
| `pos` | `QPoint` | `QWidget` |
| `geometry` | `QRect` | `QWidget` |
| `size` | `QSize` | `QWidget` |
| `windowOpacity` | `qreal` | `QWidget` |
| `maximumWidth` / `maximumHeight` | `int` | `QWidget` |
| `value` | `int` | `QProgressBar` |
| `pos` | `QPointF` | `QGraphicsItem`（通过代理）|

## 4. 自定义可动画属性

如果目标对象没有现成属性，或者你需要动画驱动一个自定义逻辑（如滚动偏移、渐变色），可以在自己的类里声明属性：

```cpp
class AnimatedWidget : public QWidget
{
    Q_OBJECT
    Q_PROPERTY(qreal glowIntensity READ glowIntensity WRITE setGlowIntensity)

public:
    explicit AnimatedWidget(QWidget* parent = nullptr) : QWidget(parent) {}

    qreal glowIntensity() const { return m_glow; }
    void setGlowIntensity(qreal v)
    {
        m_glow = v;
        update(); // 触发重绘
    }

protected:
    void paintEvent(QPaintEvent*) override
    {
        QPainter p(this);
        QColor c(Qt::cyan);
        c.setAlphaF(m_glow);
        p.fillRect(rect(), c);
    }

private:
    qreal m_glow = 0.0;
};
```

然后直接用 `QPropertyAnimation` 驱动 `"glowIntensity"`。

## 5. QGraphicsItem 属性动画

`QGraphicsItem` 不继承自 `QObject`，无法直接使用 `QPropertyAnimation`。标准做法是通过 `QGraphicsObject` 或代理类 `QGraphicsProxyWidget`；更灵活的方法是继承 `QGraphicsObject`：

```cpp
class BallItem : public QGraphicsObject
{
    Q_OBJECT
    Q_PROPERTY(QPointF pos READ pos WRITE setPos)

public:
    QRectF boundingRect() const override { return QRectF(-15, -15, 30, 30); }
    void paint(QPainter* p, const QStyleOptionGraphicsItem*, QWidget*) override
    {
        p->setBrush(Qt::red);
        p->drawEllipse(boundingRect());
    }
};
```

然后正常使用 `QPropertyAnimation(ball, "pos", ...)` 驱动它。

## 6. 动画编排

### 6.1 顺序动画组

```cpp
auto* seq = new QSequentialAnimationGroup(this);

auto* a1 = new QPropertyAnimation(widget, "geometry");
a1->setDuration(500);
a1->setStartValue(QRect(0, 100, 100, 30));
a1->setEndValue(QRect(200, 100, 100, 30));

auto* a2 = new QPropertyAnimation(widget, "windowOpacity");
a2->setDuration(300);
a2->setStartValue(1.0);
a2->setEndValue(0.0);

seq->addAnimation(a1);  // 先移动
seq->addPause(200);     // 停顿 200ms
seq->addAnimation(a2);  // 再淡出
seq->start(QAbstractAnimation::DeleteWhenStopped);
```

### 6.2 并行动画组

```cpp
auto* par = new QParallelAnimationGroup(this);

auto* moveAnim = new QPropertyAnimation(widget, "pos");
moveAnim->setDuration(600);
moveAnim->setStartValue(QPoint(0, 0));
moveAnim->setEndValue(QPoint(300, 200));

auto* fadeAnim = new QPropertyAnimation(widget, "windowOpacity");
fadeAnim->setDuration(600);
fadeAnim->setStartValue(0.0);
fadeAnim->setEndValue(1.0);

par->addAnimation(moveAnim);
par->addAnimation(fadeAnim);
par->start(QAbstractAnimation::DeleteWhenStopped);
```

### 6.3 嵌套组

顺序组和并行组可以互相嵌套，实现"先并行，再顺序"或任意组合：

```cpp
auto* outer = new QSequentialAnimationGroup(this);
auto* inner = new QParallelAnimationGroup;

inner->addAnimation(moveAnim);
inner->addAnimation(scaleAnim);

outer->addAnimation(inner);    // 先并行完移动+缩放
outer->addAnimation(fadeAnim); // 再淡出
outer->start(QAbstractAnimation::DeleteWhenStopped);
```

## 7. 状态机驱动动画（QStateMachine）

`QStateMachine` + `QPropertyAnimation` 可以让动画跟随状态自动切换，适合按钮悬浮、选中等交互状态。

```cpp
#include <QStateMachine>
#include <QState>
#include <QPropertyAnimation>

QStateMachine* machine = new QStateMachine(this);

QState* normalState = new QState(machine);
QState* hoverState = new QState(machine);

normalState->assignProperty(button, "geometry", QRect(100, 100, 120, 40));
hoverState->assignProperty(button, "geometry", QRect(90, 95, 140, 50));

auto* toHover = normalState->addTransition(button, &QPushButton::entered, hoverState);
toHover->addAnimation(new QPropertyAnimation(button, "geometry"));

auto* toNormal = hoverState->addTransition(button, &QPushButton::leaveEvent, normalState);
toNormal->addAnimation(new QPropertyAnimation(button, "geometry"));

machine->setInitialState(normalState);
machine->start();
```

## 8. 实战案例：通知弹窗（滑入 + 停留 + 淡出）

这是实际产品级场景——页面角落出现一条通知，从底部滑入，停留 2 秒，再淡出消失。

### 8.1 NotificationBar 类

```cpp
// notificationbar.h
#pragma once
#include <QWidget>
#include <QLabel>
#include <QPropertyAnimation>
#include <QSequentialAnimationGroup>
#include <QPauseAnimation>

class NotificationBar : public QWidget
{
    Q_OBJECT
    Q_PROPERTY(qreal opacity READ windowOpacity WRITE setWindowOpacity)

public:
    explicit NotificationBar(QWidget* parent = nullptr);
    void show(const QString& message, int stayMs = 2000);

private:
    QLabel* m_label = nullptr;
};
```

```cpp
// notificationbar.cpp
#include "notificationbar.h"
#include <QVBoxLayout>

NotificationBar::NotificationBar(QWidget* parent)
    : QWidget(parent, Qt::ToolTip | Qt::FramelessWindowHint)
{
    setAttribute(Qt::WA_TranslucentBackground);
    setAttribute(Qt::WA_ShowWithoutActivating);

    m_label = new QLabel(this);
    m_label->setStyleSheet(
        "QLabel { background: rgba(40,40,40,220); color: white; "
        "padding: 8px 16px; border-radius: 6px; font-size: 14px; }");

    auto* layout = new QVBoxLayout(this);
    layout->setContentsMargins(0, 0, 0, 0);
    layout->addWidget(m_label);
}

void NotificationBar::show(const QString& message, int stayMs)
{
    m_label->setText(message);
    adjustSize();

    // 定位到父窗口右下角
    if (parentWidget()) {
        QPoint br = parentWidget()->rect().bottomRight();
        move(br.x() - width() - 16, br.y() - height() - 16);
    }

    QWidget::show();

    int startY = y() + 30;
    int endY = y();

    // 1. 滑入
    auto* slideIn = new QPropertyAnimation(this, "pos");
    slideIn->setDuration(300);
    slideIn->setStartValue(QPoint(x(), startY));
    slideIn->setEndValue(QPoint(x(), endY));
    slideIn->setEasingCurve(QEasingCurve::OutCubic);

    // 2. 停留
    auto* pause = new QPauseAnimation(stayMs);

    // 3. 淡出
    auto* fadeOut = new QPropertyAnimation(this, "windowOpacity");
    fadeOut->setDuration(400);
    fadeOut->setStartValue(1.0);
    fadeOut->setEndValue(0.0);
    fadeOut->setEasingCurve(QEasingCurve::InCubic);

    auto* seq = new QSequentialAnimationGroup(this);
    seq->addAnimation(slideIn);
    seq->addAnimation(pause);
    seq->addAnimation(fadeOut);

    connect(seq, &QSequentialAnimationGroup::finished, this, &QWidget::hide);
    seq->start(QAbstractAnimation::DeleteWhenStopped);

    setWindowOpacity(1.0);
}
```

### 8.2 调用方式

```cpp
auto* notif = new NotificationBar(mainWindow);
notif->show("文件保存成功", 2500);
```

效果：从右下角滑入，停留 2.5 秒，透明度渐变到 0，然后 `hide()`。

## 9. 实战案例：可伸缩侧边栏（展开 / 收起动画）

侧边栏最大宽度 220，折叠后宽度 0，点击按钮触发动画。

```cpp
class SideBar : public QWidget
{
    Q_OBJECT

public:
    explicit SideBar(QWidget* parent = nullptr) : QWidget(parent)
    {
        setFixedWidth(0);
    }

    void toggle()
    {
        const bool expanding = (width() == 0);
        auto* anim = new QPropertyAnimation(this, "maximumWidth", this);
        anim->setDuration(250);
        anim->setStartValue(width());
        anim->setEndValue(expanding ? k_maxWidth : 0);
        anim->setEasingCurve(expanding ? QEasingCurve::OutQuart : QEasingCurve::InQuart);
        // 同步 minimumWidth 防止布局抖动
        auto* animMin = new QPropertyAnimation(this, "minimumWidth", this);
        animMin->setDuration(250);
        animMin->setStartValue(width());
        animMin->setEndValue(expanding ? k_maxWidth : 0);
        animMin->setEasingCurve(anim->easingCurve());

        auto* par = new QParallelAnimationGroup(this);
        par->addAnimation(anim);
        par->addAnimation(animMin);
        par->start(QAbstractAnimation::DeleteWhenStopped);
    }

private:
    static constexpr int k_maxWidth = 220;
};
```

## 10. 实战案例：加载转圈动画（自定义属性插值）

```cpp
class SpinnerWidget : public QWidget
{
    Q_OBJECT
    Q_PROPERTY(int angle READ angle WRITE setAngle)

public:
    explicit SpinnerWidget(QWidget* parent = nullptr) : QWidget(parent)
    {
        setFixedSize(40, 40);
    }

    int angle() const { return m_angle; }
    void setAngle(int a) { m_angle = a; update(); }

    void startSpin()
    {
        auto* anim = new QPropertyAnimation(this, "angle", this);
        anim->setStartValue(0);
        anim->setEndValue(360);
        anim->setDuration(900);
        anim->setLoopCount(-1);
        anim->setEasingCurve(QEasingCurve::Linear);
        anim->start();
        m_anim = anim;
    }

    void stopSpin()
    {
        if (m_anim) m_anim->stop();
    }

protected:
    void paintEvent(QPaintEvent*) override
    {
        QPainter p(this);
        p.setRenderHint(QPainter::Antialiasing);
        p.translate(20, 20);
        p.rotate(m_angle);

        QPen pen(Qt::white, 3);
        pen.setCapStyle(Qt::RoundCap);
        p.setPen(pen);
        p.drawArc(QRectF(-12, -12, 24, 24), 0, 270 * 16); // 3/4 圆弧
    }

private:
    int m_angle = 0;
    QPropertyAnimation* m_anim = nullptr;
};
```

## 11. 性能与坑

### 11.1 常见性能问题

1. 动画驱动 `geometry` 会触发 `resizeEvent` + 布局重算，高频刷新代价高。
   建议：用 `pos` 替代 `geometry`，或把控件移出布局（绝对定位）。
2. 大量控件同时动画：改用 `QGraphicsView` 场景 + `QGraphicsObject`，GPU 加速绘制。
3. 刷新频率：默认 60 FPS，若控件重绘耗时 > 16ms 会掉帧，要优化 `paintEvent`。

### 11.2 常见坑

1. 属性没有 `WRITE` 方法 → `QPropertyAnimation` 不报错但什么也不发生，检查 `Q_PROPERTY` 声明。
2. 对象提前销毁 → 崩溃，确保动画的父对象生命周期足够长，或用 `DeleteWhenStopped`。
3. 动画属性类型不匹配 → 插值失败，`qRegisterAnimationInterpolator` 注册自定义类型插值。

### 11.3 注册自定义类型插值器

```cpp
#include <QPropertyAnimation>

// 自定义 MyColor 的线性插值
template<>
MyColor qvariant_cast(const QVariant& v);

QVariant myColorInterpolator(const MyColor& from, const MyColor& to, qreal progress)
{
    MyColor c;
    c.r = from.r + (to.r - from.r) * progress;
    c.g = from.g + (to.g - from.g) * progress;
    c.b = from.b + (to.b - from.b) * progress;
    return QVariant::fromValue(c);
}

// 注册（在 main 或某个初始化函数中调用）
qRegisterAnimationInterpolator<MyColor>(myColorInterpolator);
```

## 12. 工程化建议

1. 把动画工厂封装为静态帮助函数（如 `AnimHelper::fadeIn(widget, 300)`），避免到处 `new QPropertyAnimation`。
2. 动画的时长、缓动曲线定义为常量，方便全局统一风格。
3. 为每种 UI 交互模式（进入/退出/强调）定义标准动画，减少设计与开发之间的沟通成本。
4. 高频/高性能场景考虑 Qt Quick（QML）的动画系统，它天然走 GPU 管线。

## 13. 学习路径建议

1. 第 1 阶段：`QPropertyAnimation` 驱动 `pos` / `windowOpacity`，掌握基本用法。
2. 第 2 阶段：缓动曲线实验，关键帧 `setKeyValueAt`，掌握节奏控制。
3. 第 3 阶段：顺序组 + 并行组，掌握复合动画编排。
4. 第 4 阶段：自定义 `Q_PROPERTY` + 自定义插值器，解锁任意动画效果。
5. 第 5 阶段：状态机联动，动画回应交互状态。
6. 第 6 阶段：`QGraphicsObject` + `QGraphicsView` 场景动画，处理大规模动画对象。
