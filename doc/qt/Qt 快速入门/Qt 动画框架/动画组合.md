# 动画组合

## 1. 核心认知：动画组合解决了什么问题

`QPropertyAnimation` 只能控制单个属性的变化。实际 UI 中往往需要：

- 多个控件同时做不同动画（并行）
- 动画有先后顺序，上一段结束触发下一段（顺序）
- 两种方式任意嵌套，形成完整的场景过渡效果

Qt 提供了两个专门用于"编排"的类：

| 类 | 作用 |
|---|---|
| `QParallelAnimationGroup` | 并行执行所有子动画，全部结束才结束 |
| `QSequentialAnimationGroup` | 按加入顺序依次执行子动画 |

两者都继承自 `QAnimationGroup`，`QAnimationGroup` 又继承自 `QAbstractAnimation`，因此动画组本身也可以作为子动画加入另一个组，**支持无限嵌套**。

```
QAbstractAnimation
└── QAnimationGroup
    ├── QParallelAnimationGroup
    └── QSequentialAnimationGroup
        └── (还可以包含 QPauseAnimation)
```

## 2. QParallelAnimationGroup — 并行动画组

### 2.1 基本用法

```cpp
#include <QParallelAnimationGroup>
#include <QPropertyAnimation>

auto* par = new QParallelAnimationGroup(this);

// 子动画 1：移动
auto* move = new QPropertyAnimation(widget, "pos");
move->setDuration(600);
move->setStartValue(QPoint(0, 0));
move->setEndValue(QPoint(300, 0));
move->setEasingCurve(QEasingCurve::OutCubic);

// 子动画 2：淡入
auto* fade = new QPropertyAnimation(widget, "windowOpacity");
fade->setDuration(400);
fade->setStartValue(0.0);
fade->setEndValue(1.0);

par->addAnimation(move);
par->addAnimation(fade);
par->start(QAbstractAnimation::DeleteWhenStopped);
```

- 总时长由持续时间**最长**的子动画决定（这里为 600ms）。
- `fade` 在 400ms 时结束；`move` 继续跑到 600ms；两者都结束，组才发出 `finished`。

### 2.2 各子动画独立缓动

并行组中每个子动画可有完全不同的时长和缓动曲线，互不影响：

```cpp
auto* scaleW = new QPropertyAnimation(widget, "maximumWidth");
scaleW->setDuration(500);
scaleW->setStartValue(0);
scaleW->setEndValue(200);
scaleW->setEasingCurve(QEasingCurve::OutBack);   // 超出回弹

auto* scaleH = new QPropertyAnimation(widget, "maximumHeight");
scaleH->setDuration(500);
scaleH->setStartValue(0);
scaleH->setEndValue(120);
scaleH->setEasingCurve(QEasingCurve::OutBounce);  // 弹跳

auto* par = new QParallelAnimationGroup(this);
par->addAnimation(scaleW);
par->addAnimation(scaleH);
par->start(QAbstractAnimation::DeleteWhenStopped);
```

## 3. QSequentialAnimationGroup — 顺序动画组

### 3.1 基本用法

```cpp
#include <QSequentialAnimationGroup>
#include <QPauseAnimation>

auto* seq = new QSequentialAnimationGroup(this);

// 第 1 段：移入
auto* slideIn = new QPropertyAnimation(widget, "pos");
slideIn->setDuration(300);
slideIn->setStartValue(QPoint(-200, 100));
slideIn->setEndValue(QPoint(50, 100));
slideIn->setEasingCurve(QEasingCurve::OutQuart);

// 第 2 段：停留 1 秒
seq->addAnimation(slideIn);
seq->addPause(1000);  // QPauseAnimation 的快捷写法

// 第 3 段：淡出
auto* fadeOut = new QPropertyAnimation(widget, "windowOpacity");
fadeOut->setDuration(400);
fadeOut->setStartValue(1.0);
fadeOut->setEndValue(0.0);

seq->addAnimation(fadeOut);
seq->start(QAbstractAnimation::DeleteWhenStopped);
```

### 3.2 `currentAnimationChanged` 信号

顺序组每切换到下一段时会发出此信号，可用来同步 UI 状态（如更新步骤指示器）：

```cpp
connect(seq, &QSequentialAnimationGroup::currentAnimationChanged,
        this, [this](QAbstractAnimation* current) {
            qDebug() << "now playing:" << current;
        });
```

### 3.3 `QPauseAnimation` 显式使用

`addPause(ms)` 是以下写法的简写：

```cpp
auto* pause = new QPauseAnimation(1000, seq);
seq->addAnimation(pause);
```

`QPauseAnimation` 本身也是 `QAbstractAnimation` 子类，可以独立 `start()`，但通常只在顺序组中使用。

## 4. 嵌套组合：终极编排能力

### 4.1 "先并行，再顺序"

```cpp
// 场景：先同时淡入两个控件，完成后再顺序弹出两个提示框
auto* outer = new QSequentialAnimationGroup(this);

// 阶段一：并行淡入
auto* phase1 = new QParallelAnimationGroup;
phase1->addAnimation(makeFadeIn(labelA, 400));
phase1->addAnimation(makeFadeIn(labelB, 400));

// 阶段二：顺序弹出
auto* phase2 = new QSequentialAnimationGroup;
phase2->addAnimation(makePopIn(tipA, 300));
phase2->addPause(150);
phase2->addAnimation(makePopIn(tipB, 300));

outer->addAnimation(phase1);
outer->addAnimation(phase2);
outer->start(QAbstractAnimation::DeleteWhenStopped);
```

### 4.2 "先顺序，再并行"

```cpp
auto* outer = new QParallelAnimationGroup(this);

// 分支 1：先移动再变色
auto* branch1 = new QSequentialAnimationGroup;
branch1->addAnimation(makeMoveAnim(widgetA));
branch1->addAnimation(makeColorAnim(widgetA));

// 分支 2：直接淡入
auto* branch2 = makeFadeIn(widgetB, 800);

outer->addAnimation(branch1);
outer->addAnimation(branch2);
outer->start(QAbstractAnimation::DeleteWhenStopped);
```

## 5. 控制所有组与动画的通用 API

`QAnimationGroup` / `QAbstractAnimation` 上的控制方法对所有动画（包括组）一致：

| 方法 | 说明 |
|---|---|
| `start()` | 从当前位置开始（或从头） |
| `pause()` | 暂停（保持当前值） |
| `resume()` | 从暂停处继续 |
| `stop()` | 停止（属性停在当前值） |
| `setCurrentTime(ms)` | 跳转到指定时间位置 |
| `setDirection(Forward/Backward)` | 正向/反向播放 |
| `setLoopCount(n)` | 循环次数（-1 无限） |
| `duration()` | 总时长（ms） |
| `state()` | `Stopped` / `Paused` / `Running` |

信号：

| 信号 | 触发时机 |
|---|---|
| `finished()` | 动画（组）播放结束 |
| `stateChanged(new, old)` | 状态切换 |
| `currentLoopChanged(loop)` | 进入新的循环轮次 |

## 6. 实战案例一：页面切换过渡

场景：从"列表页"切换到"详情页"。列表页向左滑出，详情页从右滑入，同时各自有透明度变化。

```cpp
// pageTransition.cpp
void switchToDetail(QWidget* listPage, QWidget* detailPage, QWidget* container)
{
    const int W = container->width();

    // 将详情页排在屏幕右侧待命
    detailPage->setGeometry(W, 0, W, container->height());
    detailPage->show();
    detailPage->setWindowOpacity(0.8);

    auto* par = new QParallelAnimationGroup(container);

    // 列表页：向左滑出 + 淡出
    auto* listSlide = new QPropertyAnimation(listPage, "pos");
    listSlide->setDuration(400);
    listSlide->setStartValue(QPoint(0, 0));
    listSlide->setEndValue(QPoint(-W, 0));
    listSlide->setEasingCurve(QEasingCurve::InOutQuart);

    auto* listFade = new QPropertyAnimation(listPage, "windowOpacity");
    listFade->setDuration(400);
    listFade->setStartValue(1.0);
    listFade->setEndValue(0.0);

    // 详情页：从右侧滑入 + 完全不透明
    auto* detailSlide = new QPropertyAnimation(detailPage, "pos");
    detailSlide->setDuration(400);
    detailSlide->setStartValue(QPoint(W, 0));
    detailSlide->setEndValue(QPoint(0, 0));
    detailSlide->setEasingCurve(QEasingCurve::InOutQuart);

    auto* detailFade = new QPropertyAnimation(detailPage, "windowOpacity");
    detailFade->setDuration(400);
    detailFade->setStartValue(0.8);
    detailFade->setEndValue(1.0);

    par->addAnimation(listSlide);
    par->addAnimation(listFade);
    par->addAnimation(detailSlide);
    par->addAnimation(detailFade);

    QObject::connect(par, &QParallelAnimationGroup::finished, [listPage]() {
        listPage->hide();
    });

    par->start(QAbstractAnimation::DeleteWhenStopped);
}
```

## 7. 实战案例二：引导动画（步骤式演示）

场景：依次高亮三个功能按钮，每个按钮"放大闪烁"后缩回，再切换到下一个。

```cpp
// 生成单个"强调"动画：放大 → 缩回（用 geometry 实现缩放感）
QSequentialAnimationGroup* makeEmphasis(QWidget* w)
{
    QRect orig = w->geometry();
    QRect big = orig.adjusted(-8, -8, 8, 8);

    auto* grow = new QPropertyAnimation(w, "geometry");
    grow->setDuration(180);
    grow->setStartValue(orig);
    grow->setEndValue(big);
    grow->setEasingCurve(QEasingCurve::OutQuad);

    auto* shrink = new QPropertyAnimation(w, "geometry");
    shrink->setDuration(180);
    shrink->setStartValue(big);
    shrink->setEndValue(orig);
    shrink->setEasingCurve(QEasingCurve::InQuad);

    auto* seq = new QSequentialAnimationGroup;
    seq->addAnimation(grow);
    seq->addAnimation(shrink);
    return seq;
}

// 串联三个按钮的引导动画
void startGuide(QPushButton* btn1, QPushButton* btn2, QPushButton* btn3, QObject* parent)
{
    auto* guide = new QSequentialAnimationGroup(parent);

    guide->addAnimation(makeEmphasis(btn1));
    guide->addPause(300);
    guide->addAnimation(makeEmphasis(btn2));
    guide->addPause(300);
    guide->addAnimation(makeEmphasis(btn3));

    guide->start(QAbstractAnimation::DeleteWhenStopped);
}
```

## 8. 实战案例三：仪表盘进场动画（多阶段编排）

场景：数据看板打开时，
- 阶段一（0ms）：背景卡片淡入（并行 3 张）
- 阶段二（300ms 后）：数字从 0 滚动到目标值（并行 3 个）
- 阶段三（所有数字就位后）：底部操作栏从下方滑入

```cpp
void DashboardWidget::playEntranceAnimation()
{
    auto* master = new QSequentialAnimationGroup(this);

    // —— 阶段一：卡片淡入 ——
    auto* phaseCards = new QParallelAnimationGroup;
    for (QWidget* card : { cardA, cardB, cardC }) {
        card->setWindowOpacity(0.0);
        auto* f = new QPropertyAnimation(card, "windowOpacity");
        f->setDuration(350);
        f->setStartValue(0.0);
        f->setEndValue(1.0);
        f->setEasingCurve(QEasingCurve::OutQuad);
        phaseCards->addAnimation(f);
    }

    // —— 阶段二：数字滚动（用自定义 Q_PROPERTY "displayValue" 驱动 setText）——
    auto* phaseNums = new QParallelAnimationGroup;
    struct Target { CounterLabel* label; int target; };
    for (auto [lbl, val] : { Target{counterA, 1024}, Target{counterB, 87}, Target{counterC, 312} }) {
        auto* a = new QPropertyAnimation(lbl, "displayValue");
        a->setDuration(800);
        a->setStartValue(0);
        a->setEndValue(val);
        a->setEasingCurve(QEasingCurve::OutCubic);
        phaseNums->addAnimation(a);
    }

    // —— 阶段三：底栏滑入 ——
    bottomBar->setGeometry(0, height(), width(), 48);
    auto* phaseBar = new QPropertyAnimation(bottomBar, "pos");
    phaseBar->setDuration(300);
    phaseBar->setStartValue(QPoint(0, height()));
    phaseBar->setEndValue(QPoint(0, height() - 48));
    phaseBar->setEasingCurve(QEasingCurve::OutBack);

    master->addAnimation(phaseCards);
    master->addPause(100);
    master->addAnimation(phaseNums);
    master->addAnimation(phaseBar);

    master->start(QAbstractAnimation::DeleteWhenStopped);
}
```

## 9. 动态控制运行中的动画组

### 9.1 暂停与恢复

动画组的 `pause()` / `resume()` 会递归暂停/恢复所有子动画：

```cpp
par->pause();    // 暂停整组
par->resume();   // 从当前时间点继续
```

### 9.2 反向播放（回退动画）

```cpp
// 正向播完后反向回退，实现"展开/收起"来回切换
connect(anim, &QAbstractAnimation::finished, this, [this]() {
    m_expanded = !m_expanded;
    m_group->setDirection(m_expanded
        ? QAbstractAnimation::Forward
        : QAbstractAnimation::Backward);
    m_group->start();
});
```

### 9.3 跳转进度

```cpp
// 跳到动画 1/2 处
group->setCurrentTime(group->duration() / 2);
```

## 10. 性能与注意事项

1. **子动画不要设父对象**：子动画由组管理生命周期，不需要再指定父对象，否则 double-free。
2. **`DeleteWhenStopped` 只需在最外层组设置**：内层子动画由组销毁，不需要单独设。
3. **跨线程禁止**：所有动画组必须在 GUI 线程（主线程）操作，不能跨线程调用 `start()`。
4. **避免在动画进行中修改子动画**：会导致时长计算错乱，建议先 `stop()` 再重建。
5. **大量并行控件动画**：`QParallelAnimationGroup` 每帧会驱动每个子动画，控件多时建议改用 `QGraphicsView` + `QGraphicsObject`。

## 11. 工具函数封装建议

生产项目里推荐把常用模式封装为工厂函数，让业务层只关心"做什么动画"：

```cpp
namespace AnimHelper {

inline QPropertyAnimation* fadeIn(QWidget* w, int ms = 300,
                                  QEasingCurve::Type curve = QEasingCurve::OutQuad)
{
    w->setWindowOpacity(0.0);
    auto* a = new QPropertyAnimation(w, "windowOpacity");
    a->setDuration(ms);
    a->setStartValue(0.0);
    a->setEndValue(1.0);
    a->setEasingCurve(curve);
    return a;
}

inline QPropertyAnimation* fadeOut(QWidget* w, int ms = 300)
{
    auto* a = new QPropertyAnimation(w, "windowOpacity");
    a->setDuration(ms);
    a->setStartValue(1.0);
    a->setEndValue(0.0);
    return a;
}

inline QPropertyAnimation* slideInFromLeft(QWidget* w, int ms = 350)
{
    QPoint orig = w->pos();
    auto* a = new QPropertyAnimation(w, "pos");
    a->setDuration(ms);
    a->setStartValue(QPoint(orig.x() - 80, orig.y()));
    a->setEndValue(orig);
    a->setEasingCurve(QEasingCurve::OutCubic);
    return a;
}

} // namespace AnimHelper
```

## 12. 学习路径建议

1. **第 1 阶段**：实现两个控件的并行淡入，掌握 `QParallelAnimationGroup`。
2. **第 2 阶段**：实现"弹窗出现 → 停留 → 消失"，掌握 `QSequentialAnimationGroup` + `addPause`。
3. **第 3 阶段**：实现页面切换（并行 + 信号回收），掌握动画组与 UI 状态同步。
4. **第 4 阶段**：嵌套组实现仪表盘进场，掌握多阶段编排。
5. **第 5 阶段**：封装 AnimHelper，建立项目级动画规范。
