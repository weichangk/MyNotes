# 主题切换

Qt 应用的主题切换涉及 QStyle、QPalette、QSS 三大机制的协同工作。本文从基础方案到生产级架构，系统讲解如何实现灵活的主题系统。

---

## 1. 主题切换的三种核心方案

| 方案 | 原理 | 优点 | 缺点 |
|---|---|---|---|
| **QStyle 切换** | 更换整个绘制引擎 | 控件外形彻底改变 | 可选风格有限，无法精细控制 |
| **QPalette 切换** | 更换颜色方案 | 性能好，原生感强 | 仅能控制颜色，需 Fusion 风格配合 |
| **QSS 切换** | 更换样式表文件 | 最灵活，可控制任意视觉细节 | 有解析开销，部分属性不支持 |

实际项目中通常采用 **QPalette + QSS** 组合方案。

---

## 2. 方案一：QStyle 风格切换

### 2.1 内置风格

```cpp
#include <QStyleFactory>

// 查看当前平台可用的风格
qDebug() << QStyleFactory::keys();
// Windows: ("Windows", "WindowsVista", "Fusion")
// macOS:   ("macOS", "Fusion")
// Linux:   ("Fusion", "Windows")
```

### 2.2 切换风格

```cpp
// 方式 1：代码设置
QApplication::setStyle("Fusion");

// 方式 2：命令行参数
// ./myapp -style Fusion

// 方式 3：环境变量
// QT_STYLE_OVERRIDE=Fusion
```

### 2.3 运行时切换

```cpp
void MainWindow::onStyleChanged(const QString &styleName) {
    QApplication::setStyle(QStyleFactory::create(styleName));
}

// 构建风格选择下拉框
QComboBox *styleCombo = new QComboBox;
styleCombo->addItems(QStyleFactory::keys());
connect(styleCombo, &QComboBox::currentTextChanged, this, &MainWindow::onStyleChanged);
```

> **注意**：QStyle 切换改变的是控件的绘制方式（边框形状、按钮外观等），而非颜色。颜色由 QPalette 控制。

## 3. 方案二：QPalette 调色板切换

### 3.1 定义调色板

```cpp
QPalette createDarkPalette() {

    pal.setColor(QPalette::ButtonText,      QColor(212, 212, 212));

    pal.setColor(QPalette::HighlightedText, Qt::white);
    pal.setColor(QPalette::Link,            QColor(42, 130, 218));
    pal.setColor(QPalette::ToolTipBase,     QColor(45, 45, 45));
    pal.setColor(QPalette::ToolTipText,     QColor(212, 212, 212));

    pal.setColor(QPalette::Light,           QColor(80, 80, 80));
    pal.setColor(QPalette::Mid,             QColor(50, 50, 50));
```

### 3.2 应用调色板

```cpp
QApplication::setStyle("Fusion");  // QPalette 需配合 Fusion 使用

```

### 3.3 运行时切换

```cpp

    qApp->setPalette(createDarkPalette());
}

void switchToLight() {
    qApp->setPalette(createLightPalette());
}
```

最佳实践是用 QPalette 定义颜色变量，QSS 中通过 `palette()` 引用，切换时只需更换 QPalette。

### 5.2 切换只需更换 QPalette

```cpp
// 初始化：加载一次 QSS，之后不再变动
QApplication::setStyle("Fusion");
loadStyleSheet(":/styles/theme.qss");

// 切换主题：只更换调色板
void switchTheme(bool dark) {
    if (dark)
        qApp->setPalette(createDarkPalette());
    else
        qApp->setPalette(createLightPalette());
}
```

**好处**：
- QSS 只维护一份，减少重复代码
- 新增主题只需添加新的 QPalette，无需编写新 QSS
- `palette()` 函数实时解析，无需重新加载 QSS

## 6. QSS 变量替换方案

QSS 不支持 CSS 变量，但可以通过代码预处理实现变量机制。

### 6.1 简单字符串替换

```cpp
struct ThemeColors {
    QString bgPrimary;
    QString bgSecondary;
    QString textPrimary;
    QString textSecondary;
    QString accent;
    QString accentHover;
    QString border;
};

ThemeColors darkColors {
    "#1e1e1e", "#2d2d2d", "#d4d4d4", "#888888",
    "#0e639c", "#1177bb", "#3c3c3c"
};

ThemeColors lightColors {
    "#f5f5f5", "#ffffff", "#333333", "#666666",
    "#0078d7", "#006cc1", "#cccccc"
};

QString applyThemeVars(const QString &qss, const ThemeColors &colors) {
    QString result = qss;
    result.replace("@bgPrimary",   colors.bgPrimary);
    result.replace("@bgSecondary", colors.bgSecondary);
    result.replace("@textPrimary", colors.textPrimary);
    result.replace("@textSecondary", colors.textSecondary);
    result.replace("@accent",      colors.accent);
    result.replace("@accentHover", colors.accentHover);
    result.replace("@border",      colors.border);
    return result;
}
```

**带变量的 QSS 模板**：

```css
/* template.qss */
QWidget {
    background-color: @bgPrimary;
    color: @textPrimary;
}

QPushButton {
    background-color: @bgSecondary;
    border: 1px solid @border;
    color: @textPrimary;
    border-radius: 4px;
    padding: 6px 20px;
}

QPushButton:hover {
    border-color: @accent;
}

QPushButton[btnType="primary"] {
    background-color: @accent;
    border-color: @accent;
    color: white;
}

QPushButton[btnType="primary"]:hover {
    background-color: @accentHover;
}

QLineEdit {
    background-color: @bgSecondary;
    border: 1px solid @border;
    color: @textPrimary;
}

QLineEdit:focus {
    border-color: @accent;
}
```

**加载与切换**：

```cpp
void switchTheme(const ThemeColors &colors) {
    QFile file(":/styles/template.qss");
    if (file.open(QFile::ReadOnly | QFile::Text)) {
        QString qss = applyThemeVars(file.readAll(), colors);
        qApp->setStyleSheet(qss);
        file.close();
    }
}

// 使用
switchTheme(darkColors);
switchTheme(lightColors);
```

### 6.2 基于 QMap 的通用替换

```cpp
using ThemeVars = QMap<QString, QString>;

QString processQss(const QString &qss, const ThemeVars &vars) {
    QString result = qss;
    for (auto it = vars.cbegin(); it != vars.cend(); ++it) {
        result.replace(it.key(), it.value());
    }
    return result;
}

// 定义变量集

    {"@error",         "#f44747"},
```

## 7. 图标主题切换

主题切换不仅涉及颜色，图标也需要适配（暗色主题需要浅色图标，反之亦然）。

### 7.1 双套图标资源

```
resources/
├── icons/
│   ├── light/           # 浅色图标（用于亮色主题）
│   │   ├── close.png
│   │   ├── settings.png
│   │   └── search.png
│   └── dark/            # 深色图标（用于暗色主题，实为浅色图标）
│       ├── close.png
│       ├── settings.png
│       └── search.png
```

```cpp
class IconProvider {
public:
    static QString iconPath(const QString &name) {
        QString theme = isDarkTheme() ? "dark" : "light";
        return QString(":/icons/%1/%2").arg(theme, name);
    }

    static QIcon icon(const QString &name) {
        return QIcon(iconPath(name));
    }

private:
    static bool isDarkTheme();
};

// 使用
btn->setIcon(IconProvider::icon("settings.png"));
```

### 7.2 QSS 中动态图标路径

```css
/* 在 QSS 变量替换方案中 */
QToolButton#closeBtn {
    image: url(@iconPath/close.png);
}
```

```cpp
// 替换变量时动态设置图标路径前缀
darkVars["@iconPath"] = ":/icons/dark";
lightVars["@iconPath"] = ":/icons/light";
```

### 7.3 SVG + 颜色设置

使用 SVG 图标可以通过 CSS/代码改变颜色，避免维护两套图标：

```cpp
QIcon colorizedIcon(const QString &svgPath, const QColor &color) {
    QPixmap pix(svgPath);
    QPainter painter(&pix);
    painter.setCompositionMode(QPainter::CompositionMode_SourceIn);
    painter.fillRect(pix.rect(), color);
    painter.end();
    return QIcon(pix);
}

// 切换主题时重新着色
QColor iconColor = isDark ? QColor(212, 212, 212) : QColor(51, 51, 51);
btn->setIcon(colorizedIcon(":/icons/settings.svg", iconColor));
```

### 8.1 Windows 10/11 检测暗色模式


### 8.2 监听系统主题变化（Windows）

```cpp
#ifdef Q_OS_WIN
#include <Windows.h>

// 通过 QAbstractNativeEventFilter 监听注册表变化
class ThemeWatcher : public QAbstractNativeEventFilter {
public:
    ThemeWatcher() {
        // 注册注册表变化通知
        RegOpenKeyExW(HKEY_CURRENT_USER,
            L"Software\\Microsoft\\Windows\\CurrentVersion\\Themes\\Personalize",
            0, KEY_NOTIFY, &m_hKey);

        m_watchThread = std::thread([this]() {
            while (m_running) {
                DWORD result = RegNotifyChangeKeyValue(m_hKey, FALSE,
                    REG_NOTIFY_CHANGE_LAST_SET, nullptr, FALSE);
                if (result == ERROR_SUCCESS && m_running) {
                    QMetaObject::invokeMethod(qApp, [this]() {
                        emit systemThemeChanged(isWindowsDarkMode());
                    }, Qt::QueuedConnection);
                }
            }
        });
    }

    ~ThemeWatcher() {
        m_running = false;
        RegCloseKey(m_hKey);
        if (m_watchThread.joinable())
            m_watchThread.join();
    }

signals:
    void systemThemeChanged(bool isDark);

private:
    HKEY m_hKey = nullptr;
    std::thread m_watchThread;
    std::atomic<bool> m_running{true};
};
#endif
```

### 8.3 Qt 6.5+ 原生支持

```cpp
// Qt 6.5 提供了跨平台的 colorScheme API
auto *hints = QApplication::styleHints();

// 获取当前系统主题
Qt::ColorScheme scheme = hints->colorScheme();
bool isDark = (scheme == Qt::ColorScheme::Dark);

            switchTheme(true);
        else
            switchTheme(false);
    });
```

## 9. 完整的 ThemeManager 实现

```cpp
// thememanager.h
#pragma once
#include <QObject>
#include <QPalette>
#include <QMap>

class ThemeManager : public QObject {
    Q_OBJECT
    Q_PROPERTY(Theme currentTheme READ currentTheme NOTIFY themeChanged)

public:
    enum Theme { Light, Dark, System };
    Q_ENUM(Theme)

    static ThemeManager *instance();

    Theme currentTheme() const { return m_currentTheme; }
    bool isDark() const { return m_isDark; }

    // 主题切换
    void setTheme(Theme theme);

    // 注册自定义主题
    void registerTheme(const QString &name, const QPalette &palette,
                       const QString &qssPath = {});

signals:
    void themeChanged(Theme theme);
    void darkModeChanged(bool isDark);

private:
    explicit ThemeManager(QObject *parent = nullptr);
    void applyPalette(const QPalette &pal);
    void loadAndApplyQss(const QString &path);
    bool detectSystemDarkMode();
    void setupSystemThemeWatcher();

    QPalette createDarkPalette();
    QPalette createLightPalette();

    Theme m_currentTheme = System;
    bool m_isDark = false;
    QString m_baseQssPath;

    struct ThemeData {
        QPalette palette;
        QString qssPath;
    };
    QMap<QString, ThemeData> m_customThemes;
};
```

```cpp
// thememanager.cpp
#include "thememanager.h"

ThemeManager *ThemeManager::instance() {
    static ThemeManager mgr;
    return &mgr;
}

ThemeManager::ThemeManager(QObject *parent)
    : QObject(parent)
    , m_baseQssPath(":/styles/theme.qss")
{
    QApplication::setStyle("Fusion");
    setupSystemThemeWatcher();
}

void ThemeManager::setTheme(Theme theme) {
    m_currentTheme = theme;

    bool dark = false;
    switch (theme) {
    case Dark:   dark = true;  break;
    case Light:  dark = false; break;
    case System: dark = detectSystemDarkMode(); break;
    }

    if (dark != m_isDark || theme != m_currentTheme) {
        m_isDark = dark;
        applyPalette(dark ? createDarkPalette() : createLightPalette());
        loadAndApplyQss(m_baseQssPath);

        // 刷新所有控件
        for (QWidget *w : QApplication::allWidgets()) {
            w->style()->unpolish(w);
            w->style()->polish(w);
            w->update();
        }

        emit themeChanged(theme);
        emit darkModeChanged(dark);
    }
}

void ThemeManager::applyPalette(const QPalette &pal) {
    qApp->setPalette(pal);
}

void ThemeManager::loadAndApplyQss(const QString &path) {
    QFile file(path);
    if (file.open(QFile::ReadOnly | QFile::Text)) {
        qApp->setStyleSheet(file.readAll());
        file.close();
    }
}

bool ThemeManager::detectSystemDarkMode() {
#ifdef Q_OS_WIN
    QSettings settings(
        "HKEY_CURRENT_USER\\Software\\Microsoft\\Windows\\CurrentVersion\\Themes\\Personalize",
        QSettings::NativeFormat);
    return settings.value("AppsUseLightTheme", 1).toInt() == 0;
#elif defined(Q_OS_MACOS)
    // macOS 实现（需要 Objective-C++ 桥接）
    return false;
#else
    return false;
#endif
}

void ThemeManager::setupSystemThemeWatcher() {
#if QT_VERSION >= QT_VERSION_CHECK(6, 5, 0)
    connect(QApplication::styleHints(), &QStyleHints::colorSchemeChanged,
            this, [this](Qt::ColorScheme scheme) {
        if (m_currentTheme == System) {
            m_isDark = (scheme == Qt::ColorScheme::Dark);
            applyPalette(m_isDark ? createDarkPalette() : createLightPalette());
            loadAndApplyQss(m_baseQssPath);
            for (QWidget *w : QApplication::allWidgets())
                w->update();
            emit darkModeChanged(m_isDark);
        }
    });
#endif
}
```

## 11. 主题切换动画（平滑过渡）

### 11.1 截屏渐变过渡

切换瞬间截取当前画面，叠加在窗口上方后渐变消失，避免视觉闪烁：

```cpp
#include <QPropertyAnimation>

void MainWindow::switchThemeWithAnimation(bool dark) {
    // 1. 截取当前屏幕
    QPixmap snapshot = this->grab();

    // 2. 创建覆盖层
    QLabel *overlay = new QLabel(this);
    overlay->setPixmap(snapshot);
    overlay->setGeometry(this->rect());
    overlay->raise();
    overlay->show();

    // 3. 切换主题（在覆盖层下面）
    ThemeManager::instance()->setTheme(dark ? ThemeManager::Dark : ThemeManager::Light);

    // 4. 淡出覆盖层
    QGraphicsOpacityEffect *effect = new QGraphicsOpacityEffect(overlay);
    overlay->setGraphicsEffect(effect);

    QPropertyAnimation *anim = new QPropertyAnimation(effect, "opacity");
    anim->setStartValue(1.0);
    anim->setEndValue(0.0);
    anim->setDuration(300);
    anim->setEasingCurve(QEasingCurve::OutCubic);

    connect(anim, &QPropertyAnimation::finished, overlay, &QLabel::deleteLater);
    anim->start(QAbstractAnimation::DeleteWhenStopped);
}
```

### 11.2 圆形扩展过渡

从点击点向外扩展圆形，类似 Android 的 Ripple 切换效果：

```cpp
class CircleTransition : public QWidget {
    Q_OBJECT
    Q_PROPERTY(qreal radius READ radius WRITE setRadius)

public:
    CircleTransition(QWidget *parent, QPoint center)
        : QWidget(parent), m_center(center), m_radius(0) {
        setAttribute(Qt::WA_TransparentForMouseEvents);
        setGeometry(parent->rect());
        raise();
        show();
    }

    void start(const QPixmap &oldSnapshot, std::function<void()> switchFunc) {
        m_oldSnapshot = oldSnapshot;
        switchFunc();  // 实际切换主题
        m_newSnapshot = parentWidget()->grab();

        qreal maxR = QLineF(m_center, rect().topLeft()).length();
        maxR = qMax(maxR, QLineF(m_center, rect().topRight()).length());
        maxR = qMax(maxR, QLineF(m_center, rect().bottomLeft()).length());
        maxR = qMax(maxR, QLineF(m_center, rect().bottomRight()).length());

        QPropertyAnimation *anim = new QPropertyAnimation(this, "radius");
        anim->setStartValue(0.0);
        anim->setEndValue(maxR);
        anim->setDuration(500);
        anim->setEasingCurve(QEasingCurve::OutQuad);
        connect(anim, &QPropertyAnimation::finished, this, &QWidget::deleteLater);
        anim->start(QAbstractAnimation::DeleteWhenStopped);
    }

    qreal radius() const { return m_radius; }
    void setRadius(qreal r) { m_radius = r; update(); }

protected:
    void paintEvent(QPaintEvent *) override {
        QPainter p(this);
        // 先画旧截图
        p.drawPixmap(0, 0, m_oldSnapshot);
        // 用圆形区域裁剪，画新截图
        QPainterPath path;
        path.addEllipse(m_center, m_radius, m_radius);
        p.setClipPath(path);
        p.drawPixmap(0, 0, m_newSnapshot);
    }

private:
    QPoint m_center;
    qreal m_radius;
    QPixmap m_oldSnapshot;
    QPixmap m_newSnapshot;
};

// 使用
void MainWindow::onThemeToggle(QMouseEvent *event) {
    QPixmap oldSnap = this->grab();
    auto *transition = new CircleTransition(this, event->pos());
    transition->start(oldSnap, [this]() {
        bool dark = !ThemeManager::instance()->isDark();
        ThemeManager::instance()->setTheme(dark ? ThemeManager::Dark : ThemeManager::Light);
    });
}
```

## 12. 开发期热重载

开发阶段可以让 QSS 文件修改后自动重新加载，免去重启应用：

```cpp
#include <QFileSystemWatcher>

class StyleHotReloader : public QObject {
    Q_OBJECT
public:
    StyleHotReloader(const QString &qssPath, QObject *parent = nullptr)
        : QObject(parent), m_path(qssPath)
    {
        m_watcher = new QFileSystemWatcher(this);
        m_watcher->addPath(qssPath);

        connect(m_watcher, &QFileSystemWatcher::fileChanged, this, [this](const QString &path) {
            QFile file(path);
            if (file.open(QFile::ReadOnly | QFile::Text)) {
                qApp->setStyleSheet(file.readAll());
                file.close();
                qDebug() << "[HotReload] 样式已重新加载:" << path;
            }
            // fileChanged 后文件可能被移除再创建，需要重新添加监视
            if (!m_watcher->files().contains(path))
                m_watcher->addPath(path);
        });
    }

private:
    QFileSystemWatcher *m_watcher;
    QString m_path;
};

// 仅在 Debug 模式启用
#ifdef QT_DEBUG
    auto *reloader = new StyleHotReloader("E:/project/styles/theme.qss", &app);
#endif
```

## 14. 常见问题与注意事项

### Q1：切换主题后部分控件没更新？

```cpp
// 方法 1：强制刷新所有控件
for (QWidget *w : QApplication::allWidgets()) {
    w->style()->unpolish(w);
    w->style()->polish(w);
    w->update();
}

// 方法 2：对使用动态属性的控件逐个刷新
widget->style()->unpolish(widget);
widget->style()->polish(widget);
```

### Q2：自定义控件在主题切换后不变化？

自定义 QWidget 子类需重写 `paintEvent` 并使用 `palette()` 取色，而非硬编码颜色：

```cpp
// ❌ 硬编码 → 主题切换后不变
void MyWidget::paintEvent(QPaintEvent *) {
    QPainter p(this);
    p.fillRect(rect(), QColor("#1e1e1e"));
}

// ✅ 使用调色板 → 自动跟随主题
void MyWidget::paintEvent(QPaintEvent *) {
    QStyleOption opt;
    opt.initFrom(this);
    QPainter p(this);
    style()->drawPrimitive(QStyle::PE_Widget, &opt, &p, this);

    p.setPen(palette().color(QPalette::WindowText));
    p.drawText(rect(), Qt::AlignCenter, m_text);
}
```

### Q3：QPalette 方案切换后 WindowsVista 风格不变化？

WindowsVista、macOS 等原生风格大量使用系统 API 绘制，基本忽略 QPalette。**必须使用 Fusion 风格**：

```cpp
QApplication::setStyle("Fusion");  // 放在 setPalette 之前
```

### Q4：性能考虑

| 操作 | 开销 |
|---|---|
| `setPalette` | 低（仅修改颜色数据） |
| `setStyleSheet` 小文件 | 中（需要解析 QSS） |
| `setStyleSheet` 大文件（>100KB） | 高（解析 + 逐控件匹配） |
| 遍历 `allWidgets` 刷新 | 中（与控件数量成正比） |

**优化建议**：
- 用 `palette()` 引用代替硬编码颜色值，切换时只换 QPalette，不重新加载 QSS
- QSS 文件不宜过大，超过 50KB 考虑拆分
- 避免频繁切换（如实时预览场景，加防抖）

### Q5：主题切换后图片/图标没有更新？

QSS 中 `url()` 引用的图片在样式表重新应用时会自动更新。但通过代码 `setIcon()` 设置的图标不会自动更新，需要在 `themeChanged` 信号中手动处理：

```cpp
connect(ThemeManager::instance(), &ThemeManager::darkModeChanged,
        this, [this](bool isDark) {
    // 手动更新代码中设置的图标
    m_closeBtn->setIcon(QIcon(isDark ? ":/icons/dark/close.png"
                                     : ":/icons/light/close.png"));
});
```

## 15. 方案选型速查

| 项目规模 | 推荐方案 |
|---|---|
| 小型工具/Demo | 纯 QPalette + Fusion |
| 中型应用 | QPalette + 单份 QSS（用 `palette()` 引用） |
| 大型商业应用 | ThemeManager + QPalette + QSS 变量替换 + 图标管理 |
| 需要极致性能 | 纯 QPalette + 自定义 QStyle（不用 QSS） |
| 需要设计师参与 | QSS 变量替换方案（设计师可直接编辑 QSS） |
