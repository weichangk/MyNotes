# 视图控件

## 1. 视图控件概述

### 1.1 继承体系

Qt 的视图控件共享同一套继承体系，所有视图的根基是 `QAbstractItemView`：

```
QAbstractScrollArea
└── QAbstractItemView              ← 所有视图的抽象基类
    ├── QListView                  ← 一维列表视图
    │   └── QUndoView             ← 撤销历史视图
    ├── QTableView                 ← 二维表格视图
    │   └── QHeaderView           ← 表头视图
    ├── QTreeView                  ← 树形视图
    └── QColumnView                ← 多列级联视图
```

**便捷类（自带 Model）：**

```
QListView   → QListWidget         ← 内置 QListWidgetItem
QTableView  → QTableWidget        ← 内置 QTableWidgetItem
QTreeView   → QTreeWidget         ← 内置 QTreeWidgetItem
```

### 1.2 View vs Widget：何时用哪个？

| 特征 | Model/View 类（`QXxxView`） | 便捷类（`QXxxWidget`） |
|------|---------------------------|----------------------|
| 数据管理 | 外部 Model，数据与视图分离 | 内置 Model，直接操作 Item |
| 大数据量 | 高效（按需加载渲染） | 性能较差（所有 Item 常驻内存） |
| 多视图同步 | 多个 View 可共享同一 Model | 不支持 |
| 自定义灵活度 | 高（自定义 Model / Delegate） | 低（只能用预置 API） |
| 代码复杂度 | 较高 | 简单直接 |
| 适用场景 | 数据量大、结构复杂、需要多视图联动 | 简单列表、小规模数据、快速原型 |

**经验法则：** 如果数据超过几百项，或需要排序/过滤/多视图共享，优先选择 Model/View 方案。

## 2. QAbstractItemView 公共基类

所有视图控件的公共功能都在 `QAbstractItemView` 中定义。理解它是掌握所有视图的基础。

### 2.1 核心 API

```cpp
// 设置模型
void setModel(QAbstractItemModel *model);
QAbstractItemModel *model() const;

// 设置委托
void setItemDelegate(QAbstractItemDelegate *delegate);
void setItemDelegateForColumn(int column, QAbstractItemDelegate *delegate);
void setItemDelegateForRow(int row, QAbstractItemDelegate *delegate);

// 设置选择模型
void setSelectionModel(QItemSelectionModel *selectionModel);
QItemSelectionModel *selectionModel() const;

// 获取当前索引
QModelIndex currentIndex() const;
void setCurrentIndex(const QModelIndex &index);

// 获取指定位置的索引
QModelIndex indexAt(const QPoint &point) const;         // 视口坐标 → 索引
QRect visualRect(const QModelIndex &index) const;       // 索引 → 视口矩形

// 滚动定位
void scrollTo(const QModelIndex &index, ScrollHint hint = EnsureVisible);
void scrollToTop();
void scrollToBottom();
```

### 2.2 选择模式

```cpp
// 选择行为：选择单项 / 整行 / 整列
view->setSelectionBehavior(QAbstractItemView::SelectRows);
```

| 枚举值 | 说明 |
|--------|------|
| `SelectItems` | 选择单个单元格（默认） |
| `SelectRows` | 选择整行 |
| `SelectColumns` | 选择整列 |

```cpp
// 选择模式：单选 / 多选 / 范围选择等
view->setSelectionMode(QAbstractItemView::ExtendedSelection);
```

| 枚举值 | 说明 |
|--------|------|
| `NoSelection` | 禁止选择 |
| `SingleSelection` | 只能选一个（默认） |
| `MultiSelection` | 点击切换选中/取消，无需按 Ctrl |
| `ExtendedSelection` | 支持 Ctrl+点击、Shift+范围选择（最常用） |
| `ContiguousSelection` | 只能选择连续范围（Shift+点击） |

### 2.3 编辑触发方式

```cpp
view->setEditTriggers(QAbstractItemView::DoubleClicked | QAbstractItemView::EditKeyPressed);
```

| 枚举值 | 说明 |
|--------|------|
| `NoEditTriggers` | 禁止编辑 |
| `CurrentChanged` | 切换到新项时进入编辑 |
| `DoubleClicked` | 双击编辑（常用） |
| `SelectedClicked` | 单击已选中项编辑 |
| `EditKeyPressed` | 按 F2 或 Enter 编辑 |
| `AnyKeyPressed` | 任意按键开始编辑 |
| `AllEditTriggers` | 以上所有触发方式 |

### 2.4 拖放支持

```cpp
// 启用拖放
view->setDragEnabled(true);                                    // 允许拖出
view->setAcceptDrops(true);                                    // 允许拖入
view->setDropIndicatorShown(true);                             // 显示放置指示器
view->setDragDropMode(QAbstractItemView::InternalMove);        // 内部移动

// 拖放模式
// NoDragDrop       — 禁用
// DragOnly         — 只能拖出
// DropOnly         — 只能拖入
// DragDrop         — 拖入拖出都允许
// InternalMove     — 仅内部移动（不复制）
```

### 2.5 常用信号

```cpp
// 点击信号
void clicked(const QModelIndex &index);          // 左键单击
void doubleClicked(const QModelIndex &index);    // 双击
void pressed(const QModelIndex &index);          // 按下（鼠标按下瞬间）
void activated(const QModelIndex &index);        // 激活（回车或双击，跟平台有关）

// 视口进入
void entered(const QModelIndex &index);          // 鼠标进入某项（需开启 mouseTracking）
void viewportEntered();                          // 鼠标进入视口
```

### 2.6 右键上下文菜单

```cpp
// 方式一：设置上下文菜单策略
view->setContextMenuPolicy(Qt::CustomContextMenu);
connect(view, &QAbstractItemView::customContextMenuRequested,
        this, [this, view](const QPoint &pos) {
    QModelIndex index = view->indexAt(pos);
    if (!index.isValid()) return;

    QMenu menu;
    menu.addAction("编辑", [=]() { /* ... */ });
    menu.addAction("删除", [=]() { /* ... */ });
    menu.exec(view->viewport()->mapToGlobal(pos));
});
```

## 3. QListView — 列表视图

### 3.1 基本用法

`QListView` 以一维列表形式展示数据，支持列表模式和图标模式两种布局。

```cpp
#include <QListView>
#include <QStringListModel>

// 创建 Model
QStringListModel *model = new QStringListModel(this);
model->setStringList({"苹果", "香蕉", "橘子", "葡萄", "西瓜"});

// 创建 View 并绑定
QListView *listView = new QListView(this);
listView->setModel(model);
```

### 3.2 视图模式

`QListView` 最强大的特性之一是支持两种截然不同的展示方式：

```cpp
// 列表模式（默认）— 一项一行，类似文件列表详细视图
listView->setViewMode(QListView::ListMode);

// 图标模式 — 网格排列，类似桌面图标 / 缩略图
listView->setViewMode(QListView::IconMode);
listView->setIconSize(QSize(64, 64));          // 图标尺寸
listView->setGridSize(QSize(100, 100));        // 网格单元尺寸
listView->setSpacing(10);                      // 项间距
listView->setWrapping(true);                   // 自动换行
listView->setResizeMode(QListView::Adjust);    // 窗口大小变化时重新布局
listView->setMovement(QListView::Static);      // 禁止拖拽移动位置
```

### 3.3 流方向

```cpp
// 垂直排列（默认）：从上到下
listView->setFlow(QListView::TopToBottom);

// 水平排列：从左到右
listView->setFlow(QListView::LeftToRight);
listView->setWrapping(true);   // 配合换行使用
```

### 3.4 批量加载（性能优化）

面对大量数据时，可以启用批量延迟加载：

```cpp
listView->setLayoutMode(QListView::Batched);    // 批量布局
listView->setBatchSize(100);                     // 每批处理 100 项
// 默认值 QListView::SinglePass 为一次性全部布局
```

### 3.5 统一项尺寸（性能优化）

如果所有项高度相同，设置此属性可大幅提升性能（避免逐项计算尺寸）：

```cpp
listView->setUniformItemSizes(true);
```

### 3.6 实战示例 — 图片缩略图浏览器

```cpp
class ThumbnailBrowser : public QWidget {
    Q_OBJECT
public:
    ThumbnailBrowser(QWidget *parent = nullptr) : QWidget(parent) {
        auto *layout = new QVBoxLayout(this);

        // 创建标准模型
        m_model = new QStandardItemModel(this);

        // 创建列表视图（图标模式）
        m_listView = new QListView(this);
        m_listView->setModel(m_model);
        m_listView->setViewMode(QListView::IconMode);
        m_listView->setIconSize(QSize(128, 128));
        m_listView->setGridSize(QSize(150, 170));
        m_listView->setSpacing(8);
        m_listView->setResizeMode(QListView::Adjust);
        m_listView->setWrapping(true);
        m_listView->setMovement(QListView::Static);
        m_listView->setSelectionMode(QAbstractItemView::ExtendedSelection);
        m_listView->setUniformItemSizes(true);

        layout->addWidget(m_listView);

        // 加载图片
        loadImages("C:/Pictures");

        // 双击打开
        connect(m_listView, &QListView::doubleClicked,
                this, &ThumbnailBrowser::onItemDoubleClicked);
    }

private:
    void loadImages(const QString &dirPath) {
        QDir dir(dirPath);
        QStringList filters = {"*.png", "*.jpg", "*.jpeg", "*.bmp"};
        for (const QFileInfo &fi : dir.entryInfoList(filters, QDir::Files)) {
            QPixmap pix(fi.absoluteFilePath());
            if (pix.isNull()) continue;

            auto *item = new QStandardItem;
            item->setIcon(QIcon(pix.scaled(128, 128, Qt::KeepAspectRatio,
                                           Qt::SmoothTransformation)));
            item->setText(fi.fileName());
            item->setData(fi.absoluteFilePath(), Qt::UserRole);  // 存储完整路径
            item->setEditable(false);
            m_model->appendRow(item);
        }
    }

    void onItemDoubleClicked(const QModelIndex &index) {
        QString filePath = index.data(Qt::UserRole).toString();
        QDesktopServices::openUrl(QUrl::fromLocalFile(filePath));
    }

    QStandardItemModel *m_model;
    QListView *m_listView;
};
```

## 4. QTreeView — 树形视图

### 4.1 基本用法

`QTreeView` 适用于展示层级结构数据，如文件系统、XML/JSON 文档树、组织架构图等。

```cpp
#include <QTreeView>
#include <QFileSystemModel>

// 使用文件系统模型
QFileSystemModel *model = new QFileSystemModel(this);
model->setRootPath(QDir::homePath());

QTreeView *treeView = new QTreeView(this);
treeView->setModel(model);
treeView->setRootIndex(model->index(QDir::homePath()));

// 设置列宽
treeView->setColumnWidth(0, 250);   // 名称列
treeView->header()->setStretchLastSection(true);
```

### 4.2 展开/折叠控制

```cpp
// 展开/折叠操作
treeView->expand(index);              // 展开指定节点
treeView->collapse(index);            // 折叠指定节点
treeView->expandAll();                // 展开所有节点（大数据量慎用！）
treeView->collapseAll();              // 折叠所有
treeView->expandToDepth(2);           // 展开到第 2 层

// 展开/折叠信号
connect(treeView, &QTreeView::expanded,  this, &MyWidget::onExpanded);
connect(treeView, &QTreeView::collapsed, this, &MyWidget::onCollapsed);

// 初始展开所有第一层节点
for (int i = 0; i < model->rowCount(rootIndex); ++i) {
    treeView->expand(model->index(i, 0, rootIndex));
}
```

### 4.3 缩进与装饰

```cpp
treeView->setIndentation(20);           // 子级缩进像素（默认 20）
treeView->setRootIsDecorated(true);     // 顶层项是否显示展开/折叠箭头
treeView->setItemsExpandable(true);     // 是否允许展开/折叠
treeView->setAnimated(true);            // 展开/折叠动画
treeView->setExpandsOnDoubleClick(true);// 双击展开（默认 true）
```

### 4.4 表头控制

`QTreeView` 自带列表头，通过 `header()` 方法获取 `QHeaderView` 进行配置：

```cpp
QHeaderView *header = treeView->header();

// 隐藏表头
treeView->setHeaderHidden(true);

// 列宽策略
header->setSectionResizeMode(0, QHeaderView::Stretch);         // 第 0 列自动拉伸
header->setSectionResizeMode(1, QHeaderView::ResizeToContents); // 内容自适应
header->setSectionResizeMode(2, QHeaderView::Fixed);            // 固定宽度
header->resizeSection(2, 100);                                  // 设为 100px

// 排序
treeView->setSortingEnabled(true);       // 启用点击表头排序
treeView->sortByColumn(0, Qt::AscendingOrder);  // 默认按第 0 列升序
```

### 4.5 统一行高（性能优化）

```cpp
treeView->setUniformRowHeights(true);   // 所有行等高，大幅提升大数据量性能
```

> **重要：** 对于数据量大的树形视图，`setUniformRowHeights(true)` 能带来数量级的性能提升，因为 Qt 不再需要逐行查询尺寸提示。

### 4.6 实战示例 — JSON 文档浏览器

```cpp
class JsonTreeModel : public QStandardItemModel {
public:
    explicit JsonTreeModel(QObject *parent = nullptr)
        : QStandardItemModel(parent) {
        setHorizontalHeaderLabels({"键", "值", "类型"});
    }

    void loadJson(const QJsonDocument &doc) {
        clear();
        setHorizontalHeaderLabels({"键", "值", "类型"});

        if (doc.isObject()) {
            parseObject(doc.object(), invisibleRootItem());
        } else if (doc.isArray()) {
            parseArray(doc.array(), invisibleRootItem());
        }
    }

private:
    void parseObject(const QJsonObject &obj, QStandardItem *parent) {
        for (auto it = obj.begin(); it != obj.end(); ++it) {
            addJsonValue(it.key(), it.value(), parent);
        }
    }

    void parseArray(const QJsonArray &arr, QStandardItem *parent) {
        for (int i = 0; i < arr.size(); ++i) {
            addJsonValue(QString("[%1]").arg(i), arr[i], parent);
        }
    }

    void addJsonValue(const QString &key, const QJsonValue &value,
                      QStandardItem *parent) {
        auto *keyItem   = new QStandardItem(key);
        auto *valueItem = new QStandardItem;
        auto *typeItem  = new QStandardItem;

        keyItem->setEditable(false);
        valueItem->setEditable(false);
        typeItem->setEditable(false);

        switch (value.type()) {
        case QJsonValue::Object:
            typeItem->setText("Object");
            keyItem->setIcon(QIcon(":/icons/object.png"));
            parent->appendRow({keyItem, valueItem, typeItem});
            parseObject(value.toObject(), keyItem);
            break;
        case QJsonValue::Array:
            typeItem->setText(QString("Array[%1]").arg(value.toArray().size()));
            keyItem->setIcon(QIcon(":/icons/array.png"));
            parent->appendRow({keyItem, valueItem, typeItem});
            parseArray(value.toArray(), keyItem);
            break;
        case QJsonValue::String:
            valueItem->setText(value.toString());
            typeItem->setText("String");
            parent->appendRow({keyItem, valueItem, typeItem});
            break;
        case QJsonValue::Double:
            valueItem->setText(QString::number(value.toDouble()));
            typeItem->setText("Number");
            parent->appendRow({keyItem, valueItem, typeItem});
            break;
        case QJsonValue::Bool:
            valueItem->setText(value.toBool() ? "true" : "false");
            typeItem->setText("Bool");
            parent->appendRow({keyItem, valueItem, typeItem});
            break;
        case QJsonValue::Null:
            valueItem->setText("null");
            typeItem->setText("Null");
            parent->appendRow({keyItem, valueItem, typeItem});
            break;
        default:
            break;
        }
    }
};

// 使用
JsonTreeModel *model = new JsonTreeModel(this);
QJsonDocument doc = QJsonDocument::fromJson(jsonData);
model->loadJson(doc);

QTreeView *treeView = new QTreeView(this);
treeView->setModel(model);
treeView->setAnimated(true);
treeView->setUniformRowHeights(true);
treeView->expandToDepth(1);
treeView->setColumnWidth(0, 200);
treeView->setColumnWidth(1, 300);
treeView->setAlternatingRowColors(true);
```

## 5. QTableView — 表格视图

### 5.1 基本用法

`QTableView` 以二维行列网格展示数据，适用于类电子表格、数据库查询结果等场景。

```cpp
#include <QTableView>
#include <QStandardItemModel>

QStandardItemModel *model = new QStandardItemModel(5, 3, this);
model->setHorizontalHeaderLabels({"姓名", "年龄", "城市"});

// 填充数据
model->setItem(0, 0, new QStandardItem("张三"));
model->setItem(0, 1, new QStandardItem("28"));
model->setItem(0, 2, new QStandardItem("北京"));
model->setItem(1, 0, new QStandardItem("李四"));
model->setItem(1, 1, new QStandardItem("35"));
model->setItem(1, 2, new QStandardItem("上海"));

QTableView *tableView = new QTableView(this);
tableView->setModel(model);
```

### 5.2 网格与外观

```cpp
// 网格线
tableView->setShowGrid(true);                       // 显示网格（默认 true）
tableView->setGridStyle(Qt::DotLine);               // 网格线样式：虚线

// 交替行颜色
tableView->setAlternatingRowColors(true);

// 单词换行
tableView->setWordWrap(true);

// 文字省略（标题过长时显示省略号）
tableView->setTextElideMode(Qt::ElideRight);

// 隐藏垂直表头（行号）
tableView->verticalHeader()->setVisible(false);
```

### 5.3 行列操作

```cpp
// 隐藏/显示列
tableView->setColumnHidden(2, true);    // 隐藏第 3 列
tableView->setColumnHidden(2, false);   // 显示第 3 列

// 隐藏/显示行
tableView->setRowHidden(0, true);

// 列宽设置
tableView->setColumnWidth(0, 150);
tableView->setColumnWidth(1, 80);

// 行高设置
tableView->setRowHeight(0, 40);

// 自动调整列宽到内容
tableView->resizeColumnsToContents();
tableView->resizeRowsToContents();

// 表头自适应策略
tableView->horizontalHeader()->setSectionResizeMode(QHeaderView::Stretch);
tableView->horizontalHeader()->setSectionResizeMode(0, QHeaderView::ResizeToContents);

// 拉伸最后一列填满空间
tableView->horizontalHeader()->setStretchLastSection(true);
```

### 5.4 排序

```cpp
tableView->setSortingEnabled(true);                 // 启用排序
tableView->sortByColumn(1, Qt::AscendingOrder);     // 按年龄升序
// 用户可点击表头切换排序方向
```

### 5.5 冻结列（Frozen Column）

某些表格需要在水平滚动时固定左侧列（如序号列、名称列），这需要叠加一个辅助视图：

```cpp
class FrozenTableView : public QTableView {
    Q_OBJECT
public:
    FrozenTableView(QAbstractItemModel *model, QWidget *parent = nullptr)
        : QTableView(parent) {
        setModel(model);

        // 创建冻结列视图
        m_frozenView = new QTableView(this);
        m_frozenView->setModel(model);
        m_frozenView->setFocusPolicy(Qt::NoFocus);
        m_frozenView->verticalHeader()->hide();
        m_frozenView->horizontalHeader()->setSectionResizeMode(QHeaderView::Fixed);
        m_frozenView->setSelectionModel(selectionModel());
        m_frozenView->setHorizontalScrollBarPolicy(Qt::ScrollBarAlwaysOff);
        m_frozenView->setVerticalScrollBarPolicy(Qt::ScrollBarAlwaysOff);
        m_frozenView->setStyleSheet("QTableView { border: none; }");

        // 只显示第 0 列，隐藏其余列
        for (int col = 1; col < model->columnCount(); ++col) {
            m_frozenView->setColumnHidden(col, true);
        }
        m_frozenView->setColumnWidth(0, columnWidth(0));

        // 同步垂直滚动
        connect(verticalScrollBar(), &QScrollBar::valueChanged,
                m_frozenView->verticalScrollBar(), &QScrollBar::setValue);
        connect(m_frozenView->verticalScrollBar(), &QScrollBar::valueChanged,
                verticalScrollBar(), &QScrollBar::setValue);

        updateFrozenGeometry();

        // 设置视口边距为冻结列宽度
        setViewportMargins(columnWidth(0), 0, 0, 0);
    }

protected:
    void resizeEvent(QResizeEvent *event) override {
        QTableView::resizeEvent(event);
        updateFrozenGeometry();
    }

private:
    void updateFrozenGeometry() {
        m_frozenView->setGeometry(
            verticalHeader()->width() + frameWidth(),
            frameWidth(),
            columnWidth(0),
            viewport()->height() + horizontalHeader()->height());
    }

    QTableView *m_frozenView;
};
```

### 5.6 合并单元格

```cpp
// setSpan(行, 列, 跨行数, 跨列数)
```

### 5.7 实战示例 — 数据库表格浏览器

```cpp
#include <QSqlDatabase>
#include <QSqlTableModel>
#include <QTableView>
#include <QHeaderView>

class DatabaseBrowser : public QWidget {
    Q_OBJECT
public:
    DatabaseBrowser(QWidget *parent = nullptr) : QWidget(parent) {
        auto *layout = new QVBoxLayout(this);

        // 连接数据库
        QSqlDatabase db = QSqlDatabase::addDatabase("QSQLITE");
        db.setDatabaseName("employees.db");
        db.open();

        // 创建 SQL 表模型
        m_model = new QSqlTableModel(this, db);
        m_model->setTable("employees");
        m_model->setEditStrategy(QSqlTableModel::OnManualSubmit);  // 手动提交
        m_model->select();

        // 设置列标题
        m_model->setHeaderData(0, Qt::Horizontal, "工号");
        m_model->setHeaderData(1, Qt::Horizontal, "姓名");
        m_model->setHeaderData(2, Qt::Horizontal, "部门");
        m_model->setHeaderData(3, Qt::Horizontal, "薪资");

        // 创建表格视图
        m_tableView = new QTableView(this);
        m_tableView->setModel(m_model);
        m_tableView->setAlternatingRowColors(true);
        m_tableView->setSelectionBehavior(QAbstractItemView::SelectRows);
        m_tableView->setSelectionMode(QAbstractItemView::SingleSelection);
        m_tableView->setSortingEnabled(true);
        m_tableView->horizontalHeader()->setStretchLastSection(true);
        m_tableView->verticalHeader()->setVisible(false);
        m_tableView->resizeColumnsToContents();

        // 工具栏
        auto *toolbar = new QHBoxLayout;
```

## 7. QHeaderView — 表头视图

### 7.1 概述

`QHeaderView` 管理 `QTableView` 和 `QTreeView` 的水平/垂直表头。通常不需要单独创建，通过 `horizontalHeader()` 和 `verticalHeader()` 获取。

```cpp
QHeaderView *hHeader = tableView->horizontalHeader();  // 水平表头（列标题）
QHeaderView *vHeader = tableView->verticalHeader();    // 垂直表头（行号）
```

### 7.2 列宽调整策略

```cpp
// 全局策略
hHeader->setSectionResizeMode(QHeaderView::Interactive);  // 用户可拖拽（默认）

// 单列策略
hHeader->setSectionResizeMode(0, QHeaderView::Stretch);           // 拉伸填满
hHeader->setSectionResizeMode(1, QHeaderView::ResizeToContents);  // 自适应内容
hHeader->setSectionResizeMode(2, QHeaderView::Fixed);             // 固定宽度
```

| 策略枚举 | 说明 |
|----------|------|
| `Interactive` | 用户可手动拖拽调整（默认） |
| `Fixed` | 固定大小，不可拖拽（只能通过代码调整） |
| `Stretch` | 自动拉伸以填满可用空间 |
| `ResizeToContents` | 根据内容自动适应宽度（不可拖拽） |

### 7.3 列移动与隐藏

```cpp
// 允许用户拖拽重排列

int visualIdx   = hHeader->visualIndex(0);     // 逻辑列 0 的视觉位置
```

```cpp
class CustomHeader : public QHeaderView {

        : QHeaderView(orientation, parent) {}

protected:
    void paintSection(QPainter *painter, const QRect &rect,

        painter->restore();
    }

    QSize sectionSizeFromContents(int logicalIndex) const override {
        QSize baseSize = QHeaderView::sectionSizeFromContents(logicalIndex);

    }
```

## 8. 选择模型（QItemSelectionModel）

### 8.1 概述

`QItemSelectionModel` 跟踪视图中的选择状态，是独立于 View 的对象。多个 View 可以共享同一个选择模型，从而实现**选中联动**。

```cpp
// 获取选择模型
QItemSelectionModel *selModel = view->selectionModel();

// 多视图共享选择模型
QItemSelectionModel *shared = new QItemSelectionModel(model, this);
listView->setSelectionModel(shared);
tableView->setSelectionModel(shared);
// 现在两个视图的选中状态完全同步
```

### 8.2 查询选择状态

```cpp
// 当前项（焦点所在）
QModelIndex current = selModel->currentIndex();

// 是否选中某项
bool selected = selModel->isSelected(index);

// 获取所有已选项
QModelIndexList selectedIndexes = selModel->selectedIndexes();   // 所有选中的单元格
QModelIndexList selectedRows    = selModel->selectedRows();      // 所有选中的行（第 0 列）
QModelIndexList selectedColumns = selModel->selectedColumns();   // 所有选中的列

// 是否有选中项
bool hasSelection = selModel->hasSelection();
```

### 8.3 编程控制选择

```cpp
// 选中单个项
selModel->select(index, QItemSelectionModel::Select);

// 选中并设为当前项
selModel->setCurrentIndex(index, QItemSelectionModel::Select);

// 选中一个范围
QItemSelection selection(topLeft, bottomRight);
selModel->select(selection, QItemSelectionModel::Select);

// 清除所有选择后选中新项
selModel->select(index, QItemSelectionModel::ClearAndSelect);

// 切换选中状态
selModel->select(index, QItemSelectionModel::Toggle);

// 选中整行
selModel->select(index, QItemSelectionModel::Select | QItemSelectionModel::Rows);

// 清空选择
selModel->clearSelection();
selModel->clear();   // 同时清除当前项
```

**SelectionFlag 组合：**

| 标志 | 说明 |
|------|------|
| `NoUpdate` | 不做任何更改 |
| `Clear` | 清除所有已有选择 |
| `Select` | 选中指定项 |
| `Deselect` | 取消选中指定项 |
| `Toggle` | 切换选中状态 |
| `Current` | 更新当前项 |
| `Rows` | 扩展到整行 |
| `Columns` | 扩展到整列 |
| `ClearAndSelect` | `Clear | Select` 的便捷组合 |

### 8.4 选择变化信号

```cpp

connect(selModel, &QItemSelectionModel::currentChanged,
```

### 8.5 实战示例 — 多视图联动

```cpp
class LinkedViews : public QWidget {
    Q_OBJECT
public:
    LinkedViews(QWidget *parent = nullptr) : QWidget(parent) {
        auto *layout = new QHBoxLayout(this);

        // 共享数据模型
        auto *model = new QStandardItemModel(this);
        model->setHorizontalHeaderLabels({"名称", "类型", "大小"});
        for (int i = 0; i < 20; ++i) {
            QList<QStandardItem *> row;
            row << new QStandardItem(QString("文件_%1.txt").arg(i))
                << new QStandardItem("文本文件")
                << new QStandardItem(QString("%1 KB").arg(qrand() % 1000));
            model->appendRow(row);

        listView->setModel(model);
        listView->setModelColumn(0);   // 只显示第 0 列

        auto *tableView = new QTableView(this);
        tableView->setModel(model);
        tableView->setSelectionBehavior(QAbstractItemView::SelectRows);

        // 共享选择模型 → 选中联动
        auto *selectionModel = new QItemSelectionModel(model, this);
        listView->setSelectionModel(selectionModel);
        tableView->setSelectionModel(selectionModel);

        layout->addWidget(listView, 1);
        layout->addWidget(tableView, 2);
    }
};
```

## 11. 性能优化要点

### 11.1 优化清单

| 优化手段 | 适用视图 | 说明 |
|----------|---------|------|
| `setUniformItemSizes(true)` | QListView | 所有项同尺寸时跳过逐项尺寸计算 |
| `setUniformRowHeights(true)` | QTreeView | 所有行等高时大幅提升滚动性能 |
| `setLayoutMode(Batched)` | QListView | 大量数据批量延迟布局 |
| `resizeMode = Fixed` | QHeaderView 列 | 避免频繁计算列宽 |
| 使用 `QSortFilterProxyModel` | 所有 | 排序/过滤不修改源数据 |
| 避免 `expandAll()` | QTreeView | 大数据树全展开极耗性能 |
| `fetchMore()` / `canFetchMore()` | 自定义 Model | 按需增量加载数据 |
| 减少 `dataChanged` 发射频率 | 自定义 Model | 批量更新后一次性通知 |
| `beginResetModel/endResetModel` | 自定义 Model | 大规模数据替换使用重置而非逐行操作 |

### 11.2 大数据量检验

当数据量达到**万级以上**时，注意以下原则：

```cpp
// ❌ 错误：逐行插入并通知
for (int i = 0; i < 100000; ++i) {
    model->insertRow(model->rowCount());
    model->setData(model->index(model->rowCount()-1, 0), data[i]);
}

// ✅ 正确：批量插入
model->beginInsertRows(QModelIndex(), 0, 99999);
// ... 内部批量填充数据 ...
model->endInsertRows();

// ✅ 或使用 reset
model->beginResetModel();
// ... 替换全部内部数据 ...
model->endResetModel();
```

## 13. 总结对比

| 特性 | QListView | QTreeView | QTableView | QColumnView |
|------|-----------|-----------|------------|-------------|
| 数据维度 | 一维列表 | 树形层级 | 二维表格 | 树形层级（列展开） |
| 行列 | 只有行 | 行 + 多列 | 行 + 列 | 行（每级一列） |

| 图标模式 | ✅ | ❌ | ❌ | ❌ |
| 合并单元格 | — | — | ✅ | — |
| 展开/折叠 | — | ✅ | — | 自动级联 |
| 适用模型基类 | QAbstractListModel | QAbstractItemModel | QAbstractTableModel | QAbstractItemModel |
| 便捷类 | QListWidget | QTreeWidget | QTableWidget | 无 |
| 典型场景 | 消息列表、缩略图 | 文件浏览器、配置树 | 数据库表、报表 | Finder 浏览 |
