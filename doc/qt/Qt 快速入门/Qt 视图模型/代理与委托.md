# 代理与委托（Delegate）

## 1. 委托的角色与职责

在 Qt 的 Model/View 架构中，**Delegate（委托/代理）** 是连接 Model 和 View 的桥梁，承担两项核心职责：

1. **渲染（Paint）**：控制数据项在视图中如何绘制显示
2. **编辑（Edit）**：当用户双击或按 Enter 进入编辑模式时，提供合适的编辑器控件

```
┌─────────────────────────────────────────────────────────────┐
│                         View                                │
│  ┌───────────┐  ┌───────────┐  ┌───────────┐               │
│  │  Item A   │  │  Item B   │  │  Item C   │  ← 每个 Item  │
│  │ (Delegate │  │ (Delegate │  │ (Delegate │    都由委托    │
│  │  负责绘制) │  │  负责绘制) │  │  负责绘制) │    负责渲染    │
│  └───────────┘  └───────────┘  └───────────┘               │
│       ↕               ↕               ↕                     │
│   双击时弹出        双击时弹出       双击时弹出               │
│   编辑器控件        编辑器控件       编辑器控件               │
└─────────────────────────────────────────────────────────────┘
```

**没有委托时**：View 使用默认的文本显示和 `QLineEdit` 编辑，所有数据项样式统一。

**使用委托后**：可以为不同列/行/单元格定制完全不同的显示样式和编辑控件。

---

## 2. 委托类继承体系

```
QAbstractItemDelegate            ← 抽象基类，定义最底层接口
    ├── QItemDelegate            ← 旧式委托（Qt 4 风格，使用 QPainter 绘制）
    └── QStyledItemDelegate      ← 推荐使用（Qt 4.4+，支持 QStyle 样式表）
```

| 类 | 特点 | 使用场景 |
|---|---|---|
| `QAbstractItemDelegate` | 纯虚基类，需实现全部绘制/编辑逻辑 | 极少直接使用 |
| `QItemDelegate` | 自行处理绘制，不支持样式表 | 旧项目兼容 |
| `QStyledItemDelegate` | 利用当前 `QStyle` 绘制，支持 CSS 样式表 | **推荐用于所有新项目** |

> **最佳实践**：始终继承 `QStyledItemDelegate`，它支持样式表且与平台原生风格一致。

## 3. 委托的核心虚函数

`QStyledItemDelegate` 提供了 4 个核心虚函数，覆盖「创建 → 赋值 → 回收 → 绘制」的完整生命周期：

```cpp
class QStyledItemDelegate : public QAbstractItemDelegate
{
public:
    // 1. 创建编辑器：用户进入编辑模式时调用
    virtual QWidget *createEditor(QWidget *parent,
                                  const QStyleOptionViewItem &option,
                                  const QModelIndex &index) const;

    // 2. 设置编辑器数据：用 Model 中的数据初始化编辑器
    virtual void setEditorData(QWidget *editor,
                               const QModelIndex &index) const;

    // 3. 将编辑器数据写回 Model
    virtual void setModelData(QWidget *editor,
                              QAbstractItemModel *model,
                              const QModelIndex &index) const;

    // 4. 设置编辑器的几何位置和大小
    virtual void updateEditorGeometry(QWidget *editor,
                                      const QStyleOptionViewItem &option,
                                      const QModelIndex &index) const;

    // 5. 自定义绘制（可选重写）
    virtual void paint(QPainter *painter,
                       const QStyleOptionViewItem &option,
                       const QModelIndex &index) const;

    // 6. 自定义大小提示（可选重写）
    virtual QSize sizeHint(const QStyleOptionViewItem &option,
                           const QModelIndex &index) const;
};
```

### 3.1 编辑器生命周期流程

```
用户双击单元格
      │
      ▼
createEditor()          ← 创建编辑器控件（QSpinBox/QComboBox/...）
      │
      ▼
setEditorData()         ← 从 Model 读取数据填入编辑器
      │
      ▼
updateEditorGeometry()  ← 设置编辑器位置和大小
      │
      ▼
 [用户编辑数据...]
      │
      ▼
setModelData()          ← 用户确认后，将编辑器数据写回 Model
      │
      ▼
编辑器自动销毁
```

### 3.2 绘制流程

```
View 需要绘制某个 Item
      │
      ▼
paint(painter, option, index)
      │
      ├── option.state 包含状态信息（选中、悬停、获焦等）
      ├── index 可获取 Model 数据（DisplayRole, DecorationRole 等）
      └── painter 用于实际绘制操作
```

## 4. 实战示例

### 4.1 SpinBox 委托 — 整数编辑器

最经典的委托示例：为表格的某一列提供 `QSpinBox` 作为编辑器。

```cpp
#include <QStyledItemDelegate>
#include <QSpinBox>

class SpinBoxDelegate : public QStyledItemDelegate
{
    Q_OBJECT
public:
    explicit SpinBoxDelegate(QObject *parent = nullptr)
        : QStyledItemDelegate(parent) {}

    // 1. 创建 SpinBox 编辑器
    QWidget *createEditor(QWidget *parent,
                          const QStyleOptionViewItem & /*option*/,
                          const QModelIndex & /*index*/) const override
    {
        auto *editor = new QSpinBox(parent);
        editor->setFrame(false);       // 无边框，融入表格
        editor->setMinimum(0);
        editor->setMaximum(100);
        return editor;
    }

    // 2. 用 Model 数据初始化编辑器
    void setEditorData(QWidget *editor,
                       const QModelIndex &index) const override
    {
        int value = index.model()->data(index, Qt::EditRole).toInt();
        auto *spinBox = static_cast<QSpinBox *>(editor);
        spinBox->setValue(value);
    }

    // 3. 将编辑器数据写回 Model
    void setModelData(QWidget *editor,
                      QAbstractItemModel *model,
                      const QModelIndex &index) const override
    {
        auto *spinBox = static_cast<QSpinBox *>(editor);
        spinBox->interpretText();  // 确保手动输入的文本被解析
        int value = spinBox->value();
        model->setData(index, value, Qt::EditRole);
    }

    // 4. 设置编辑器尺寸
    void updateEditorGeometry(QWidget *editor,
                              const QStyleOptionViewItem &option,
                              const QModelIndex & /*index*/) const override
    {
        editor->setGeometry(option.rect);
    }
};
```

**使用方式：**

```cpp
auto *tableView = new QTableView;
auto *model = new QStandardItemModel(4, 3);  // 4行3列

// 填充数据
for (int row = 0; row < 4; ++row)
    for (int col = 0; col < 3; ++col)
        model->setItem(row, col, new QStandardItem(QString::number(row * 10 + col)));

tableView->setModel(model);

// 仅为第2列设置 SpinBox 委托
auto *delegate = new SpinBoxDelegate(tableView);
tableView->setItemDelegateForColumn(1, delegate);
```

### 4.2 ComboBox 委托 — 下拉选择器

当某列的值是有限枚举时，使用下拉框比手动输入更合适。

```cpp
class ComboBoxDelegate : public QStyledItemDelegate
{
    Q_OBJECT
public:
    explicit ComboBoxDelegate(const QStringList &items, QObject *parent = nullptr)
        : QStyledItemDelegate(parent), m_items(items) {}

    QWidget *createEditor(QWidget *parent,
                          const QStyleOptionViewItem &,
                          const QModelIndex &) const override
    {
        auto *combo = new QComboBox(parent);
        combo->addItems(m_items);
        return combo;
    }

    void setEditorData(QWidget *editor,
                       const QModelIndex &index) const override
    {
        QString value = index.model()->data(index, Qt::EditRole).toString();
        auto *combo = static_cast<QComboBox *>(editor);
        int idx = combo->findText(value);
        if (idx >= 0)
            combo->setCurrentIndex(idx);
    }

    void setModelData(QWidget *editor,
                      QAbstractItemModel *model,
                      const QModelIndex &index) const override
    {
        auto *combo = static_cast<QComboBox *>(editor);
        model->setData(index, combo->currentText(), Qt::EditRole);
    }

    void updateEditorGeometry(QWidget *editor,
                              const QStyleOptionViewItem &option,
                              const QModelIndex &) const override
    {
        editor->setGeometry(option.rect);
    }

private:
    QStringList m_items;
};
```

**使用方式：**

```cpp
QStringList levels = {"初级", "中级", "高级", "专家"};
tableView->setItemDelegateForColumn(2, new ComboBoxDelegate(levels, tableView));
```

### 4.3 进度条委托 — 自定义绘制（只读）

重写 `paint()` 可以实现完全自定义的显示效果。这里将整数值渲染为进度条：

```cpp
class ProgressBarDelegate : public QStyledItemDelegate
{
    Q_OBJECT
public:
    using QStyledItemDelegate::QStyledItemDelegate;

    void paint(QPainter *painter,
               const QStyleOptionViewItem &option,
               const QModelIndex &index) const override
    {
        int progress = index.model()->data(index, Qt::DisplayRole).toInt();

        // 使用 QStyleOptionProgressBar 绘制原生风格进度条
        QStyleOptionProgressBar progressBarOption;
        progressBarOption.rect = option.rect;
        progressBarOption.minimum = 0;
        progressBarOption.maximum = 100;
        progressBarOption.progress = progress;
        progressBarOption.text = QString("%1%").arg(progress);
        progressBarOption.textVisible = true;

        // 绘制选中背景
        if (option.state & QStyle::State_Selected) {
            painter->fillRect(option.rect, option.palette.highlight());
        }

        // 使用当前应用样式绘制进度条
        QApplication::style()->drawControl(QStyle::CE_ProgressBar,
                                           &progressBarOption,
                                           painter);
    }

    // 禁用编辑（只读委托）
    QWidget *createEditor(QWidget *, const QStyleOptionViewItem &,
                          const QModelIndex &) const override
    {
        return nullptr;  // 返回 nullptr 表示不可编辑
    }
};
```

### 4.4 星级评分委托 — 完整自定义绘制 + 编辑

这是一个综合示例：显示星形评分图标，点击可修改评分。

**第一步：定义星级数据类型**

```cpp
// starrating.h
class StarRating
{
public:
    enum EditMode { Editable, ReadOnly };

    explicit StarRating(int starCount = 1, int maxStarCount = 5)
        : m_starCount(starCount), m_maxStarCount(maxStarCount) {}

    void paint(QPainter *painter, const QRect &rect,
               const QPalette &palette, EditMode mode) const
    {
        painter->save();
        painter->setRenderHint(QPainter::Antialiasing, true);
        painter->setPen(Qt::NoPen);

        if (mode == Editable)
            painter->setBrush(palette.highlight());
        else
            painter->setBrush(palette.windowText());

        const int starSize = qMin(rect.width() / m_maxStarCount, rect.height());
        const QPointF starCenter(starSize / 2.0, rect.height() / 2.0);

        for (int i = 0; i < m_maxStarCount; ++i) {
            if (i < m_starCount)
                painter->setBrush(QColor(255, 165, 0));  // 金色填充
            else
                painter->setBrush(QColor(200, 200, 200));  // 灰色未选

            QRectF starRect(rect.x() + i * starSize, rect.y(),
                            starSize, rect.height());
            drawStar(painter, starRect);
        }

        painter->restore();
    }

    int starCount() const { return m_starCount; }
    int maxStarCount() const { return m_maxStarCount; }
    void setStarCount(int count) { m_starCount = qBound(0, count, m_maxStarCount); }

private:
    void drawStar(QPainter *painter, const QRectF &rect) const
    {
        // 绘制五角星路径
        const double cx = rect.center().x();
        const double cy = rect.center().y();
        const double r = qMin(rect.width(), rect.height()) / 2.0 * 0.8;

        QPainterPath path;
        for (int i = 0; i < 5; ++i) {
            double angle = -M_PI / 2 + i * 4 * M_PI / 5;
            QPointF pt(cx + r * cos(angle), cy + r * sin(angle));
            if (i == 0) path.moveTo(pt);
            else        path.lineTo(pt);
        }
        path.closeSubpath();
        painter->drawPath(path);
    }

    int m_starCount;
    int m_maxStarCount;
};

// 注册为 Qt 元类型，以便通过 QVariant 在 Model 中存储
Q_DECLARE_METATYPE(StarRating)
```

**第二步：自定义编辑器控件**

```cpp
// stareditor.h
class StarEditor : public QWidget
{
    Q_OBJECT
public:
    explicit StarEditor(QWidget *parent = nullptr)
        : QWidget(parent) { setMouseTracking(true); }

    QSize sizeHint() const override
    {
        return m_starRating.sizeHint();
    }

    void setStarRating(const StarRating &rating)
    {
        m_starRating = rating;
        update();
    }

    StarRating starRating() const { return m_starRating; }

signals:
    void editingFinished();

protected:
    void paintEvent(QPaintEvent *) override
    {
        QPainter painter(this);
        m_starRating.paint(&painter, rect(), palette(), StarRating::Editable);
    }

    void mouseMoveEvent(QMouseEvent *event) override
    {
        const int star = starAtPosition(event->x());
        if (star != m_starRating.starCount() && star > 0) {
            m_starRating.setStarCount(star);
            update();
        }
    }

    void mouseReleaseEvent(QMouseEvent *) override
    {
        emit editingFinished();  // 鼠标释放 → 结束编辑
    }

private:
    int starAtPosition(int x) const
    {
        int starWidth = width() / m_starRating.maxStarCount();
        return (starWidth > 0) ? (x / starWidth + 1) : 0;
    }

    StarRating m_starRating;
};
```

**第三步：实现星级委托**

```cpp
// starratingdelegate.h
class StarRatingDelegate : public QStyledItemDelegate
{
    Q_OBJECT
public:
    using QStyledItemDelegate::QStyledItemDelegate;

    void paint(QPainter *painter,
               const QStyleOptionViewItem &option,
               const QModelIndex &index) const override
    {
        if (index.data().canConvert<StarRating>()) {
            StarRating starRating = qvariant_cast<StarRating>(index.data());

            if (option.state & QStyle::State_Selected)
                painter->fillRect(option.rect, option.palette.highlight());

            starRating.paint(painter, option.rect, option.palette,
                             StarRating::ReadOnly);
        } else {
            QStyledItemDelegate::paint(painter, option, index);
        }
    }

    QSize sizeHint(const QStyleOptionViewItem &option,
                   const QModelIndex &index) const override
    {
        if (index.data().canConvert<StarRating>()) {
            StarRating starRating = qvariant_cast<StarRating>(index.data());
            return starRating.sizeHint();
        }
        return QStyledItemDelegate::sizeHint(option, index);
    }

    QWidget *createEditor(QWidget *parent,
                          const QStyleOptionViewItem &,
                          const QModelIndex &) const override
    {
        auto *editor = new StarEditor(parent);
        connect(editor, &StarEditor::editingFinished,
                this, &StarRatingDelegate::commitAndCloseEditor);
        return editor;
    }

    void setEditorData(QWidget *editor,
                       const QModelIndex &index) const override
    {
        if (index.data().canConvert<StarRating>()) {
            auto starRating = qvariant_cast<StarRating>(index.data());
            static_cast<StarEditor *>(editor)->setStarRating(starRating);
        }
    }

    void setModelData(QWidget *editor,
                      QAbstractItemModel *model,
                      const QModelIndex &index) const override
    {
        auto *starEditor = static_cast<StarEditor *>(editor);
        model->setData(index, QVariant::fromValue(starEditor->starRating()));
    }

private slots:
    void commitAndCloseEditor()
    {
        auto *editor = qobject_cast<StarEditor *>(sender());
        emit commitData(editor);      // 提交数据到 Model
        emit closeEditor(editor);     // 关闭编辑器
    }
};
```

### 4.5 日期选择委托

```cpp
class DateDelegate : public QStyledItemDelegate
{
    Q_OBJECT
public:
    using QStyledItemDelegate::QStyledItemDelegate;

        static_cast<QDateEdit *>(editor)->setDate(date);
    }

                      const QModelIndex &index) const override
    {
        auto *dateEdit = static_cast<QDateEdit *>(editor);
        model->setData(index, dateEdit->date(), Qt::EditRole);
    }

    void updateEditorGeometry(QWidget *editor,
                              const QStyleOptionViewItem &option,
                              const QModelIndex &) const override
    {
        editor->setGeometry(option.rect);
    }

    // 自定义显示格式
    QString displayText(const QVariant &value,
                        const QLocale &locale) const override
    {
        Q_UNUSED(locale)
        if (value.type() == QVariant::Date)
            return value.toDate().toString("yyyy年MM月dd日");
        return QStyledItemDelegate::displayText(value, locale);
    }
};
```

## 5. 委托的设置方式

View 提供多种粒度的委托设置 API：

```cpp
// ① 为整个视图设置委托（所有单元格共用）
view->setItemDelegate(delegate);

// ② 为某一列设置委托
view->setItemDelegateForColumn(columnIndex, delegate);

// ③ 为某一行设置委托
view->setItemDelegateForRow(rowIndex, delegate);
```

**优先级**：行委托 > 列委托 > 全局委托。也就是说，如果同时为某行和某列设置了不同委托，行委托优先。

### 混合使用示例

```cpp
auto *tableView = new QTableView;
tableView->setModel(model);

// 第0列：日期选择器
tableView->setItemDelegateForColumn(0, new DateDelegate(tableView));

// 第1列：SpinBox（数字输入）
tableView->setItemDelegateForColumn(1, new SpinBoxDelegate(tableView));

// 第2列：ComboBox（下拉选择）
tableView->setItemDelegateForColumn(2, new ComboBoxDelegate(levels, tableView));

// 第3列：进度条（只读显示）
tableView->setItemDelegateForColumn(3, new ProgressBarDelegate(tableView));
```

## 6. 编辑触发机制

View 控制什么操作会触发编辑模式（调用委托的 `createEditor`）：

```cpp
// 获取/设置编辑触发方式
view->editTriggers();
view->setEditTriggers(triggers);
```

| 触发方式 | 枚举值 | 说明 |
|---|---|---|
| 无触发 | `QAbstractItemView::NoEditTriggers` | 禁止编辑 |
| 当前项变化 | `QAbstractItemView::CurrentChanged` | 选中不同项时进入编辑 |
| 双击 | `QAbstractItemView::DoubleClicked` | 双击单元格（**默认**） |
| 单击选中项 | `QAbstractItemView::SelectedClicked` | 单击已选中的项 |
| 按 Enter/Return | `QAbstractItemView::EditKeyPressed` | 按编辑键 |
| 任意按键 | `QAbstractItemView::AnyKeyPressed` | 按任意键开始编辑 |
| 全部 | `QAbstractItemView::AllEditTriggers` | 以上所有方式 |

```cpp
// 组合使用（按位或）
view->setEditTriggers(QAbstractItemView::DoubleClicked
                      | QAbstractItemView::EditKeyPressed);
```

## 7. commitData 与 closeEditor 信号

委托通过两个信号与 View 交互，控制编辑完成的流程：

```cpp
// 告诉 View：编辑器中的数据已准备好，请写回 Model
emit commitData(editor);

// 告诉 View：请关闭并销毁编辑器
emit closeEditor(editor);

// closeEditor 的第二个参数可控制关闭后的行为
emit closeEditor(editor, QAbstractItemDelegate::SubmitModelCache);
emit closeEditor(editor, QAbstractItemDelegate::RevertModelCache);
```

### EndEditHint 枚举

| 枚举值 | 说明 |
|---|---|
| `NoHint` | 关闭编辑器，无额外操作 |
| `EditNextItem` | 关闭后自动打开下一项的编辑器 |
| `EditPreviousItem` | 关闭后自动打开上一项的编辑器 |
| `SubmitModelCache` | 提交模型缓存（用于缓存型 Model） |
| `RevertModelCache` | 回滚模型缓存 |

### 典型用法：编辑器中自动提交

```cpp
// 在 createEditor 中连接信号，实现"值改变即提交"
QWidget *createEditor(QWidget *parent, ...) const override
{
    auto *combo = new QComboBox(parent);
    combo->addItems(m_items);

    // 选择改变时立即提交数据并关闭编辑器
    connect(combo, QOverload<int>::of(&QComboBox::currentIndexChanged),
            this, [this, combo]() {
                // const_cast 因为 createEditor 是 const 方法
                auto *delegate = const_cast<ComboBoxDelegate *>(this);
                emit delegate->commitData(combo);
                emit delegate->closeEditor(combo);
            });
    return combo;
}
```

## 8. 自定义绘制进阶

### 8.1 paint() 中的 QStyleOptionViewItem

`QStyleOptionViewItem` 包含绘制所需的完整上下文信息：

```cpp
void paint(QPainter *painter,
           const QStyleOptionViewItem &option,
           const QModelIndex &index) const override
{
    // option 中的关键字段：
    // option.rect         → 单元格矩形区域
    // option.state        → 当前状态（选中、悬停、禁用等）
    // option.palette      → 调色板
    // option.font         → 字体
    // option.decorationPosition → 图标位置
    // option.displayAlignment   → 文本对齐方式

    // 常用状态判断：
    bool isSelected = option.state & QStyle::State_Selected;
    bool isHovered  = option.state & QStyle::State_MouseOver;
    bool isEnabled  = option.state & QStyle::State_Enabled;
    bool hasFocus   = option.state & QStyle::State_HasFocus;
}
```

### 8.2 绘制带图标和多行文本的 Item

```cpp
void paint(QPainter *painter,
           const QStyleOptionViewItem &option,
           const QModelIndex &index) const override
{
    painter->save();

    // 绘制背景
    if (option.state & QStyle::State_Selected) {
        painter->fillRect(option.rect, option.palette.highlight());
        painter->setPen(option.palette.highlightedText().color());
    } else if (option.state & QStyle::State_MouseOver) {
        painter->fillRect(option.rect, QColor(240, 240, 240));
        painter->setPen(option.palette.text().color());
    } else {
        painter->setPen(option.palette.text().color());
    }

    QRect rect = option.rect.adjusted(4, 4, -4, -4);

    // 绘制图标（左侧 48x48）
    QIcon icon = qvariant_cast<QIcon>(index.data(Qt::DecorationRole));
    if (!icon.isNull()) {
        QRect iconRect(rect.x(), rect.y(), 48, 48);
        icon.paint(painter, iconRect, Qt::AlignCenter);
        rect.setLeft(rect.left() + 56);  // 图标区域 + 间距
    }

    // 绘制标题（上半部分，粗体）
    QString title = index.data(Qt::DisplayRole).toString();
    QFont boldFont = option.font;
    boldFont.setBold(true);
    boldFont.setPointSize(boldFont.pointSize() + 1);
    painter->setFont(boldFont);
    painter->drawText(rect.adjusted(0, 0, 0, -rect.height() / 2),
                      Qt::AlignLeft | Qt::AlignVCenter, title);

    // 绘制描述（下半部分，小号灰色）
    QString desc = index.data(Qt::UserRole + 1).toString();
    QFont smallFont = option.font;
    smallFont.setPointSize(smallFont.pointSize() - 1);
    painter->setFont(smallFont);
    painter->setPen(QColor(128, 128, 128));
    painter->drawText(rect.adjusted(0, rect.height() / 2, 0, 0),
                      Qt::AlignLeft | Qt::AlignVCenter, desc);

    painter->restore();
}

QSize sizeHint(const QStyleOptionViewItem &,
               const QModelIndex &) const override
{
    return QSize(200, 56);  // 固定行高 56px
}
```

### 8.3 使用 initStyleOption 复用默认绘制

如果只需要修改部分绘制逻辑，可以先初始化默认样式，再覆盖部分内容：

```cpp
void paint(QPainter *painter,
           const QStyleOptionViewItem &option,
           const QModelIndex &index) const override
{
    QStyleOptionViewItem opt = option;
    initStyleOption(&opt, index);

    // 修改部分样式
    if (index.data(Qt::UserRole + 100).toBool()) {
        opt.font.setStrikeOut(true);    // 已完成的任务加删除线
        opt.palette.setColor(QPalette::Text, QColor(160, 160, 160));
    }

    // 让默认样式引擎完成绑定
    const QWidget *widget = option.widget;
    QStyle *style = widget ? widget->style() : QApplication::style();
    style->drawControl(QStyle::CE_ItemViewItem, &opt, painter, widget);
}
```

## 9. editorEvent — 无需创建编辑器的交互

当不需要弹出独立编辑器控件，而是在 `paint()` 区域直接响应鼠标事件时，重写 `editorEvent()`：

```cpp
bool editorEvent(QEvent *event,
                 QAbstractItemModel *model,
                 const QStyleOptionViewItem &option,
                 const QModelIndex &index) override
{
    // 实现复选框效果：单击切换勾选状态
    if (event->type() == QEvent::MouseButtonRelease) {
        auto *mouseEvent = static_cast<QMouseEvent *>(event);

        // 计算复选框区域
        QRect checkboxRect(option.rect.x() + 4, option.rect.y() + 4, 20, 20);

        if (checkboxRect.contains(mouseEvent->pos())) {
            bool checked = index.data(Qt::CheckStateRole).toBool();
            model->setData(index, !checked, Qt::CheckStateRole);
            return true;  // 事件已处理
        }
    }
    return QStyledItemDelegate::editorEvent(event, model, option, index);
}
```

**典型应用场景**：
- 复选框切换
- 评分组件（直接点击修改星级）
- 删除按钮（行内按钮）
- 超链接点击

## 10. 数据角色（Data Role）与委托的关系

Model 中的每个数据项可以存储多个角色的数据，委托根据不同角色获取不同数据：

| 角色 | 枚举值 | 委托中的用途 |
|---|---|---|
| `Qt::DisplayRole` | 0 | `paint()` 中获取显示文本 |
| `Qt::EditRole` | 2 | `setEditorData()` / `setModelData()` 读写数据 |
| `Qt::DecorationRole` | 1 | `paint()` 中绘制图标 |
| `Qt::ToolTipRole` | 3 | 鼠标悬停时的提示文本 |
| `Qt::FontRole` | 6 | 自定义字体 |
| `Qt::TextAlignmentRole` | 7 | 文本对齐方式 |
| `Qt::BackgroundRole` | 8 | 背景颜色/画刷 |
| `Qt::ForegroundRole` | 9 | 前景色（文字颜色） |
| `Qt::CheckStateRole` | 10 | 复选框状态 |
| `Qt::UserRole` | 0x100 | 自定义数据起始值 |

```cpp
// Model 中存储多种角色数据
model->setData(index, "张三", Qt::DisplayRole);            // 显示文本
model->setData(index, QIcon(":/avatar.png"), Qt::DecorationRole); // 图标
model->setData(index, "员工编号：001", Qt::ToolTipRole);      // 提示
model->setData(index, QFont("微软雅黑", 12), Qt::FontRole);   // 字体
model->setData(index, QColor(255, 0, 0), Qt::ForegroundRole); // 红色文字
model->setData(index, Qt::AlignCenter, Qt::TextAlignmentRole); // 居中

// 自定义角色
model->setData(index, employeeId, Qt::UserRole);             // 存储自定义数据
model->setData(index, department, Qt::UserRole + 1);
```

## 11. QStyledItemDelegate vs QItemDelegate 对比

| 特性 | QStyledItemDelegate | QItemDelegate |
|---|---|---|
| 绘制方式 | 通过 `QStyle` 系统 | 直接 `QPainter` 绘制 |
| 样式表支持 | ✅ 完整支持 | ❌ 不支持 |
| 平台原生外观 | ✅ 自动适配 | ❌ 固定外观 |
| 数据格式化 | `displayText()` + `QLocale` | 自行处理 |
| HTML 富文本 | 需手动处理 | 内置 `QTextDocument` 支持 |
| 推荐程度 | **推荐** | 仅兼容旧代码 |

> **注意**：`QItemDelegate` 内置支持 HTML 富文本渲染（通过 `QTextDocument`），这是它相对 `QStyledItemDelegate` 的一个小优势。如果需要在 `QStyledItemDelegate` 中渲染富文本，需要手动使用 `QTextDocument`。

## 12. 富文本渲染委托示例

```cpp
class RichTextDelegate : public QStyledItemDelegate
{
    Q_OBJECT
public:
    using QStyledItemDelegate::QStyledItemDelegate;

    void paint(QPainter *painter,
               const QStyleOptionViewItem &option,
               const QModelIndex &index) const override
    {
        QStyleOptionViewItem opt = option;
        initStyleOption(&opt, index);

        // 绘制背景和选中效果
        const QWidget *widget = opt.widget;
        QStyle *style = widget ? widget->style() : QApplication::style();
        style->drawPrimitive(QStyle::PE_PanelItemViewItem, &opt, painter, widget);

        // 使用 QTextDocument 渲染 HTML
        QTextDocument doc;
        doc.setHtml(opt.text);
        doc.setTextWidth(opt.rect.width());

        painter->save();
        painter->translate(opt.rect.topLeft());
        painter->setClipRect(opt.rect.translated(-opt.rect.topLeft()));
        doc.drawContents(painter);
        painter->restore();
    }

    QSize sizeHint(const QStyleOptionViewItem &option,
                   const QModelIndex &index) const override
    {
        QStyleOptionViewItem opt = option;
        initStyleOption(&opt, index);

        QTextDocument doc;
        doc.setHtml(opt.text);
        doc.setTextWidth(opt.rect.width());
        return QSize(doc.idealWidth(), doc.size().height());
    }
};
```

**使用方式：**

```cpp
model->setData(index, "<b>标题</b><br><i style='color:gray'>描述文字</i>");
view->setItemDelegate(new RichTextDelegate(view));
```

## 13. 实用技巧与注意事项

### 13.1 编辑器的父控件

`createEditor()` 中创建的编辑器 **必须** 以传入的 `parent` 为父对象。View 会自动管理编辑器的生命周期和定位。

```cpp
// ✅ 正确
auto *editor = new QSpinBox(parent);

// ❌ 错误：无父对象会导致编辑器浮动在窗口外
auto *editor = new QSpinBox;
```

### 13.2 避免在 const 方法中修改状态

`createEditor`、`setEditorData`、`paint` 等都是 `const` 方法，不应修改委托的成员变量。如需存储临时状态，使用 `mutable` 或通过信号/槽机制。

### 13.3 大量数据的绘制性能

- `paint()` 会在每次视图刷新时被频繁调用，**避免在其中执行耗时操作**（如文件 I/O、复杂计算）
- 使用 `QPixmapCache` 缓存复杂的绘制结果
- `sizeHint()` 也会频繁调用，保持轻量

```cpp
void paint(QPainter *painter, ...) const override
{
    // ✅ 使用缓存的 Pixmap
    QString key = QString("item_%1").arg(index.row());
    QPixmap pixmap;
    if (!QPixmapCache::find(key, &pixmap)) {
        pixmap = QPixmap(option.rect.size());
        // ... 执行复杂绘制到 pixmap
        QPixmapCache::insert(key, pixmap);
    }
    painter->drawPixmap(option.rect, pixmap);
}
```

### 13.4 编辑器事件过滤

委托默认安装了事件过滤器来处理 Tab/Esc/Enter 等按键。如果编辑器需要特殊键盘处理，可以重写 `eventFilter()`：

```cpp
bool eventFilter(QObject *editor, QEvent *event) override
{
    if (event->type() == QEvent::KeyPress) {
        auto *keyEvent = static_cast<QKeyEvent *>(event);
        if (keyEvent->key() == Qt::Key_Tab) {
            // 自定义 Tab 行为：而不是切换到下一个委托
            return true;  // 事件已处理
        }
    }
    return QStyledItemDelegate::eventFilter(editor, event);
}
```

### 13.5 在 QML 中使用委托

QML 中的委托是声明式的，概念相同但实现方式完全不同：

```qml
ListView {
    model: myModel

    delegate: Rectangle {
        width: ListView.view.width
        height: 60
        color: index % 2 === 0 ? "#f0f0f0" : "white"

        Row {
            anchors.fill: parent
            spacing: 10

            Image {
                source: model.avatarUrl
                width: 48; height: 48
            }

            Column {
                Text {
                    text: model.name
                    font.bold: true
                    font.pixelSize: 16
                }
                Text {
                    text: model.description
                    color: "gray"
                    font.pixelSize: 12
                }
            }
        }

        MouseArea {
            anchors.fill: parent
            onClicked: console.log("Clicked:", model.name)
        }
    }
}
```

> **QML 委托特点**：QML 中每个列表项都是一个独立的 QML 组件实例，比 C++ 中的 `paint()` 方法更直观，但对大量数据的性能不如 C++ 委托。

## 14. 完整综合示例：员工管理表格

将以上所有委托组合到一个完整的表格应用中：

```cpp
#include <QApplication>
#include <QTableView>
#include <QStandardItemModel>
#include <QHeaderView>
#include "spinboxdelegate.h"
#include "comboboxdelegate.h"
#include "datedelegate.h"
#include "progressbardelegate.h"

int main(int argc, char *argv[])
{
    QApplication app(argc, argv);

    // 创建模型
    auto *model = new QStandardItemModel(5, 5);
    model->setHorizontalHeaderLabels(
        {"姓名", "入职日期", "部门", "年龄", "任务进度"});

    // 填充示例数据
    QStringList names = {"张三", "李四", "王五", "赵六", "钱七"};
    QStringList depts = {"研发", "市场", "运营", "设计", "测试"};
    QList<int> ages = {28, 35, 24, 30, 27};
    QList<int> progress = {85, 60, 45, 90, 70};

    for (int row = 0; row < 5; ++row) {
        model->setItem(row, 0, new QStandardItem(names[row]));
        model->setItem(row, 1, new QStandardItem);
        model->setData(model->index(row, 1),
                       QDate(2020 + row, row + 1, 15));
        model->setItem(row, 2, new QStandardItem(depts[row]));
        model->setItem(row, 3, new QStandardItem(QString::number(ages[row])));
        model->setItem(row, 4, new QStandardItem(QString::number(progress[row])));
    }

    // 创建视图
    auto *tableView = new QTableView;
    tableView->setModel(model);
    tableView->setWindowTitle("员工管理系统");
    tableView->resize(800, 400);
    tableView->horizontalHeader()->setStretchLastSection(true);
    tableView->setSelectionBehavior(QAbstractItemView::SelectRows);
    tableView->setAlternatingRowColors(true);

    // 为各列设置不同的委托
    // 第0列：默认委托（文本输入）
    tableView->setItemDelegateForColumn(1, new DateDelegate(tableView));
    tableView->setItemDelegateForColumn(2,
        new ComboBoxDelegate({"研发", "市场", "运营", "设计", "测试", "管理"}, tableView));
    tableView->setItemDelegateForColumn(3, new SpinBoxDelegate(tableView));
    tableView->setItemDelegateForColumn(4, new ProgressBarDelegate(tableView));

    tableView->show();
    return app.exec();
}
```

## 15. 总结

| 要点 | 说明 |
|---|---|
| 推荐基类 | 始终继承 `QStyledItemDelegate` |
| 核心四函数 | `createEditor` → `setEditorData` → `setModelData` → `updateEditorGeometry` |
| 自定义显示 | 重写 `paint()` 和 `sizeHint()` |
| 无编辑器交互 | 重写 `editorEvent()` |
| 设置粒度 | 支持全局 / 按行 / 按列三种粒度 |
| 数据通信 | 通过 `commitData` + `closeEditor` 信号 |
| 性能注意 | `paint()` 中避免耗时操作，善用缓存 |
| QML 替代 | QML 使用声明式 `delegate` 组件，概念相同实现不同 |
