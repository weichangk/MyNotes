# 文本流与数据流

Qt 提供两种高层流式 I/O 类：`QTextStream`（文本流，处理人类可读的文本）和 `QDataStream`（数据流，处理二进制序列化）。两者都基于 `QIODevice`，可用于文件、内存缓冲区、网络套接字等所有 I/O 设备。

---

## 1. 对比概览

| 特性 | QTextStream | QDataStream |
|---|---|---|
| 数据格式 | 人类可读的文本 | 二进制（紧凑、高效） |
| 编码处理 | 自动编码转换（UTF-8 等） | 无编码概念（原始字节） |
| 跨平台 | 需注意换行符/编码差异 | 自带版本号，天然跨平台 |
| 精度 | 浮点数受文本格式限制 | 完全精确（IEEE 754） |
| 文件大小 | 较大（文本表示） | 较小（二进制表示） |
| 可读性 | 文本编辑器可直接查看 | 需专用程序解析 |
| 适用场景 | 配置文件、日志、CSV、用户可见文本 | 保存/加载应用数据、网络传输、缓存 |

```
QIODevice（基类）
    ├── QFile          ←─┐
    ├── QBuffer        ←──┤── QTextStream / QDataStream 都可附着
    ├── QProcess       ←──┤
    └── QTcpSocket     ←─┘
```

## 2. QTextStream：文本流

### 2.1 基本读写

```cpp
#include <QTextStream>
#include <QFile>

// ===== 写入 =====
QFile file("output.txt");
file.open(QIODevice::WriteOnly | QIODevice::Text);

QTextStream out(&file);
out << "Name: " << "Alice" << "\n";
out << "Age: " << 30 << "\n";
out << "Score: " << 95.5 << "\n";
out << "Active: " << true << "\n";   // 输出 "1"
file.close();

// 文件内容：
// Name: Alice
// Age: 30
// Score: 95.5
// Active: 1

// ===== 读取 =====
file.open(QIODevice::ReadOnly | QIODevice::Text);
QTextStream in(&file);

QString line1 = in.readLine();  // "Name: Alice"
QString all   = in.readAll();   // 剩余全部文本
file.close();
```

### 2.2 附着到不同设备

```cpp
// 附着到文件
QFile file("data.txt");
file.open(QIODevice::ReadWrite | QIODevice::Text);
QTextStream stream(&file);

// 附着到 QString（内存中操作）
QString buffer;
QTextStream stream(&buffer);
stream << "Hello " << 42 << " world";
// buffer == "Hello 42 world"

// 附着到 QByteArray
QByteArray bytes;
QTextStream stream(&bytes);
stream << "value: " << 3.14;
stream.flush();  // 必须 flush，数据才写入 bytes
// bytes == "value: 3.14"

// 附着到 stdout / stderr（控制台输出）
QTextStream cout(stdout);
QTextStream cerr(stderr);
cout << "Normal output" << Qt::endl;
cerr << "Error output" << Qt::endl;

// 附着到 stdin（控制台输入）
QTextStream cin(stdin);
QString name;
cout << "Enter name: " << Qt::flush;
name = cin.readLine();
```

### 2.3 编码设置

```cpp
QTextStream stream(&file);

// ===== Qt 5 =====
stream.setCodec("UTF-8");           // 默认取决于系统 locale
stream.setCodec("GBK");             // 中文 Windows 常用
stream.setCodec("Shift-JIS");       // 日文
stream.setCodec("ISO-8859-1");      // Latin1

stream.setCodec(QTextCodec::codecForName("UTF-8"));
stream.setGenerateByteOrderMark(true);  // 写入 BOM

// ===== Qt 6 =====
stream.setEncoding(QStringConverter::Utf8);     // 默认
stream.setEncoding(QStringConverter::Utf16);
stream.setEncoding(QStringConverter::Latin1);
stream.setGenerateByteOrderMark(true);

// 自动检测编码（Qt 5）
QTextCodec *codec = QTextCodec::codecForUtfText(file.peek(4), 
                                                 QTextCodec::codecForName("UTF-8"));
stream.setCodec(codec);
```

### 2.4 逐行读取

```cpp
QFile file("data.csv");
file.open(QIODevice::ReadOnly | QIODevice::Text);
QTextStream in(&file);

int lineNum = 0;
while (!in.atEnd()) {
    QString line = in.readLine();  // 不含末尾换行符
    lineNum++;
    
    // 跳过空行和注释
    if (line.trimmed().isEmpty() || line.startsWith('#'))
        continue;
    
    QStringList fields = line.split(',');
    qDebug() << "Line" << lineNum << ":" << fields;
}
```

### 2.5 逐词读取（>> 运算符）

`>>` 运算符以空白符（空格、Tab、换行）为分隔符读取 token：

```cpp
// 文件内容: "Alice 30 95.5"
QTextStream in(&file);

QString name;
int age;
double score;
in >> name >> age >> score;
// name="Alice", age=30, score=95.5

// 读取所有单词
QStringList words;
while (!in.atEnd()) {
    QString word;
    in >> word;
    if (!word.isEmpty())
        words.append(word);
}
```

### 2.6 格式控制

```cpp
QTextStream out(&file);

// ===== 数字基数 =====
out << Qt::hex << 255;             // "ff"
out << Qt::dec << 255;             // "255"
out << Qt::oct << 255;             // "377"
out << Qt::bin << 255;             // "11111111"（Qt 6.4+）

out << Qt::showbase << Qt::hex << 255;   // "0xff"
out << Qt::uppercasebase << Qt::hex << 255;  // "0XFF"
out << Qt::uppercasedigits << Qt::hex << 255; // "0xFF" → "FF"

// ===== 浮点数格式 =====
double pi = 3.14159265358979;

out.setRealNumberPrecision(3);
out << pi;                         // "3.14"（默认 SmartNotation）

out.setRealNumberNotation(QTextStream::FixedNotation);
out.setRealNumberPrecision(6);
out << pi;                         // "3.141593"

out.setRealNumberNotation(QTextStream::ScientificNotation);
out << pi;                         // "3.141593e+00"

out.setRealNumberNotation(QTextStream::SmartNotation);
out << pi;                         // "3.14159"（自动选择最短表示）

// ===== 对齐与填充 =====
out.setFieldWidth(10);
out.setFieldAlignment(QTextStream::AlignRight);
out.setPadChar('.');
out << "Hello";                    // ".....Hello"

out.setFieldAlignment(QTextStream::AlignLeft);
out << "Hello";                    // "Hello....."

out.setFieldAlignment(QTextStream::AlignCenter);
out << "Hi";                       // "....Hi...."

// ===== 重置字段宽度 =====
out.setFieldWidth(0);              // 关闭对齐

// ===== 正号显示 =====
out << Qt::forcesign << 42;        // "+42"
out << Qt::noforcesign << 42;      // "42"

// ===== 布尔值文本 =====
out << Qt::boolalpha << true;      // "true"（默认输出 "1"）
out << Qt::noboolalpha << true;    // "1"
```

### 2.7 操纵器（Manipulators）

```cpp
// Qt::endl — 写入换行符并刷新缓冲区
out << "First line" << Qt::endl;

// Qt::flush — 仅刷新缓冲区（不换行）
out << "prompt: " << Qt::flush;

// Qt::reset — 重置所有格式设置

// "\n" vs Qt::endl

```

### 2.8 状态检查

```cpp
QTextStream in(&file);

in.status();  // QTextStream::Ok / ReadPastEnd / ReadCorruptData / WriteFailed

// 读取数字失败不会抛异常，需检查 status
int val;
in >> val;
if (in.status() == QTextStream::ReadCorruptData) {
    qWarning() << "Failed to read integer";
    in.resetStatus();  // 重置状态后才能继续读取
}

// 自动的隐式 bool 转换（类似 std::stream）
if (in >> val) {
    // 读取成功
}
```

## 3. QDataStream：二进制数据流

### 3.1 基本读写

```cpp
#include <QDataStream>
#include <QFile>

// ===== 写入 =====
QFile file("data.bin");
file.open(QIODevice::WriteOnly);

QDataStream out(&file);
out.setVersion(QDataStream::Qt_5_15);  // 指定序列化版本

out << QString("Hello");      // 写入 QString
out << (qint32)42;            // 写入 32 位整数
out << 3.14;                  // 写入 double
out << true;                  // 写入 bool
out << QDateTime::currentDateTime();  // 写入日期时间
file.close();

// ===== 读取（顺序必须与写入一致） =====
file.open(QIODevice::ReadOnly);

QDataStream in(&file);
in.setVersion(QDataStream::Qt_5_15);  // 版本必须匹配！

QString str;
qint32 num;
double pi;
bool flag;
QDateTime dt;

in >> str >> num >> pi >> flag >> dt;
// str="Hello", num=42, pi=3.14, flag=true, dt=写入时的时间
file.close();
```

### 3.2 版本管理（关键）

`QDataStream` 的序列化格式会随 Qt 版本演进。**必须显式指定版本号**，否则不同 Qt 版本编译的程序可能无法互读数据。

```cpp
// 序列化版本常量
QDataStream::Qt_5_0     // Qt 5.0 格式
QDataStream::Qt_5_15    // Qt 5.15 格式
QDataStream::Qt_6_0     // Qt 6.0 格式
QDataStream::Qt_6_7     // Qt 6.7 格式

// 推荐做法：在文件头写入版本信息
void writeData(QFile &file) {
    QDataStream out(&file);

    // 文件头：魔数 + 版本号
    out << (quint32)0xA0B0C0D0;          // 魔数（标识文件格式）
    out << (qint32)QDataStream::Qt_5_15; // 流版本号
    out.setVersion(QDataStream::Qt_5_15);

    // 实际数据
    out << QString("data") << (qint32)100;
}

void readData(QFile &file) {
    QDataStream in(&file);

    // 读取并验证文件头
    quint32 magic;
    in >> magic;
    if (magic != 0xA0B0C0D0) {
        qWarning() << "Invalid file format";
        return;
    }

    qint32 version;
    in >> version;
    if (version < QDataStream::Qt_5_0 || version > QDataStream::Qt_6_7) {
        qWarning() << "Unsupported version:" << version;
        return;
    }
    in.setVersion(version);

    // 读取实际数据
    QString str;
    qint32 num;
    in >> str >> num;
}
```

```cpp

// 单精度（节省空间）
```

### 3.5 支持的内置类型

`QDataStream` 原生支持几乎所有 Qt 值类型的序列化：

| 类型 | 二进制格式 |
|---|---|
| `qint8/16/32/64`、`quint8/16/32/64` | 定长整数 |
| `float`、`double` | IEEE 754 |
| `bool` | 1 字节 |
| `QString` | 长度(quint32) + UTF-16 数据；空串写 0xFFFFFFFF |
| `QByteArray` | 长度(quint32) + 原始数据 |
| `QChar` | 2 字节 |
| `QDate` | Julian day (qint64) |
| `QTime` | 午夜以来毫秒 (quint32) |
| `QDateTime` | QDate + QTime + 时区信息 |
| `QPoint` / `QPointF` | x + y |
| `QSize` / `QSizeF` | width + height |
| `QRect` / `QRectF` | x + y + width + height |
| `QColor` | RGBA 各 quint16 |
| `QImage` / `QPixmap` | 图像格式（PNG 等）+ 数据 |
| `QFont` | 字体属性序列化 |
| `QVariant` | type id + 值数据 |
| `QList<T>` | 长度 + 元素序列 |
| `QMap<K,V>` | 长度 + 键值对序列 |
| `QHash<K,V>` | 长度 + 键值对序列 |
| `QSet<T>` | 长度 + 元素序列 |
| `QPair<T1,T2>` | first + second |

```cpp
// 容器类型序列化
QDataStream out(&file);

QStringList names = {"Alice", "Bob", "Charlie"};
out << names;  // QList<QString> 直接写入

QMap<QString, int> scores = {{"Alice", 95}, {"Bob", 87}};
out << scores;

QList<QPointF> points = {{1.0, 2.0}, {3.0, 4.0}};
out << points;

// 读取
QDataStream in(&file);
QStringList names2;
QMap<QString, int> scores2;
QList<QPointF> points2;
in >> names2 >> scores2 >> points2;
```

### 3.6 错误处理

```cpp
QDataStream in(&file);

// 状态检查
QDataStream::Status status = in.status();
// QDataStream::Ok              — 正常
// QDataStream::ReadPastEnd     — 读到文件末尾
// QDataStream::ReadCorruptData — 数据损坏
// QDataStream::WriteFailed     — 写入失败

in >> value;
if (in.status() != QDataStream::Ok) {
    qWarning() << "Read error occurred";
    in.resetStatus();  // 重置后可继续尝试
}

// 事务（Qt 5.7+）：原子性读取
in.startTransaction();
QString name;
qint32 age;
in >> name >> age;

if (in.commitTransaction()) {
    // 全部读取成功
    qDebug() << name << age;
} else {
    // 读取失败，流位置自动回滚
    qWarning() << "Incomplete data";
}
```

## 4. 自定义类型的序列化

### 4.1 QDataStream 自定义序列化

```cpp
struct Student {
    QString name;
    int age;
    double gpa;
    QStringList courses;
};

// 重载 << 和 >> 运算符
QDataStream &operator<<(QDataStream &out, const Student &s) {
    out << s.name << (qint32)s.age << s.gpa << s.courses;
    return out;
}

QDataStream &operator>>(QDataStream &in, Student &s) {
    qint32 age;
    in >> s.name >> age >> s.gpa >> s.courses;
    s.age = age;
    return in;
}

// 使用
Student alice{"Alice", 20, 3.8, {"Math", "CS", "Physics"}};

// 写入
QFile file("students.dat");
file.open(QIODevice::WriteOnly);
QDataStream out(&file);
out.setVersion(QDataStream::Qt_5_15);
out << alice;
file.close();

// 读取
file.open(QIODevice::ReadOnly);
QDataStream in(&file);
in.setVersion(QDataStream::Qt_5_15);
Student loaded;
in >> loaded;
// loaded.name="Alice", loaded.age=20, loaded.gpa=3.8
```

### 4.3 版本化自定义序列化（向后兼容）

```cpp
struct AppData {
    // v1 字段
    QString name;
    int score;
    // v2 新增字段
    QDateTime lastLogin;
    // v3 新增字段
    QMap<QString, QVariant> settings;
};

// 写入时总是写最新格式
QDataStream &operator<<(QDataStream &out, const AppData &d) {
    out << (quint8)3;  // 当前数据版本号
    out << d.name << (qint32)d.score;
    out << d.lastLogin;
    out << d.settings;
    return out;
}

// 读取时根据版本号适配
QDataStream &operator>>(QDataStream &in, AppData &d) {
    quint8 dataVersion;
    in >> dataVersion;

    // v1 字段（所有版本都有）
    qint32 score;
    in >> d.name >> score;
    d.score = score;

    // v2 字段
    if (dataVersion >= 2) {
        in >> d.lastLogin;
    } else {
        d.lastLogin = QDateTime();  // 默认值
    }

    // v3 字段
    if (dataVersion >= 3) {
        in >> d.settings;
    } else {
        d.settings.clear();
    }

    return in;
}
```

## 6. 实战案例

### 6.1 CSV 文件读写

```cpp
// 写入 CSV
void writeCsv(const QString &filePath,
              const QStringList &headers,
              const QList<QStringList> &rows) {
    QFile file(filePath);
    file.open(QIODevice::WriteOnly | QIODevice::Text);
    QTextStream out(&file);
    out.setCodec("UTF-8");  // Qt 5
    out.setGenerateByteOrderMark(true);  // Excel 需要 BOM 才能正确识别 UTF-8

    // 表头
    out << headers.join(",") << "\n";

    // 数据行
    for (const auto &row : rows) {
        QStringList escaped;
        for (const auto &field : row) {
            // CSV 转义：包含逗号/引号/换行的字段用双引号包裹
            if (field.contains(',') || field.contains('"') || field.contains('\n')) {
                escaped.append("\"" + QString(field).replace("\"", "\"\"") + "\"");
            } else {
                escaped.append(field);
            }
        }
        out << escaped.join(",") << "\n";
    }
}

// 读取 CSV（简易版：不处理引号内的逗号）
QList<QStringList> readCsv(const QString &filePath) {
    QList<QStringList> result;
    QFile file(filePath);
    file.open(QIODevice::ReadOnly | QIODevice::Text);
    QTextStream in(&file);

    while (!in.atEnd()) {
        QString line = in.readLine().trimmed();
        if (!line.isEmpty()) {
            result.append(line.split(','));
        }
    }
    return result;
}
```

### 6.2 简易 INI 解析器

```cpp
QMap<QString, QString> parseIni(const QString &filePath) {
    QMap<QString, QString> config;
    QFile file(filePath);
    file.open(QIODevice::ReadOnly | QIODevice::Text);
    QTextStream in(&file);

    QString section;
    while (!in.atEnd()) {
        QString line = in.readLine().trimmed();

        // 跳过空行和注释
        if (line.isEmpty() || line.startsWith(';') || line.startsWith('#'))
            continue;

        // 节（section）
        if (line.startsWith('[') && line.endsWith(']')) {
            section = line.mid(1, line.length() - 2);
            continue;
        }

        // 键值对
        int eq = line.indexOf('=');
        if (eq > 0) {
            QString key = line.left(eq).trimmed();
            QString val = line.mid(eq + 1).trimmed();
            QString fullKey = section.isEmpty() ? key : section + "/" + key;
            config[fullKey] = val;
        }
    }
    return config;
}

// 使用：文件内容
// [General]
// name = MyApp
// version = 1.0
// [Window]
// width = 800

auto cfg = parseIni("app.ini");
// cfg["General/name"] = "MyApp"
// cfg["Window/width"] = "800"
```

### 6.3 二进制文件格式（游戏存档示例）

```cpp
struct SaveData {
    static constexpr quint32 MAGIC = 0x53415645;  // "SAVE"
    static constexpr quint8 FORMAT_VERSION = 2;

    QString playerName;
    qint32 level;
    qint64 score;
    QDateTime saveTime;
    QMap<QString, qint32> inventory;  // 物品 → 数量
    QList<QPointF> checkpoints;

    bool save(const QString &filePath) {
        QFile file(filePath);
        if (!file.open(QIODevice::WriteOnly))
            return false;

        QDataStream out(&file);
        out.setVersion(QDataStream::Qt_5_15);

        // 文件头
        out << MAGIC;
        out << FORMAT_VERSION;

        // 数据
        out << playerName << level << score << saveTime;
        out << inventory << checkpoints;

        return out.status() == QDataStream::Ok;
    }

    bool load(const QString &filePath) {
        QFile file(filePath);
        if (!file.open(QIODevice::ReadOnly))
            return false;

        QDataStream in(&file);
        in.setVersion(QDataStream::Qt_5_15);

        // 验证文件头
        quint32 magic;
        in >> magic;
        if (magic != MAGIC) {
            qWarning() << "Not a save file";
            return false;
        }

        quint8 version;
        in >> version;
        if (version > FORMAT_VERSION) {
            qWarning() << "Save file too new:" << version;
            return false;
        }

        // 读取数据
        in >> playerName >> level >> score >> saveTime;

        if (version >= 2) {
            in >> inventory >> checkpoints;
        }

        return in.status() == QDataStream::Ok;
    }
};

// 使用
SaveData save;
save.playerName = "Hero";
save.level = 42;
save.score = 123456;
save.saveTime = QDateTime::currentDateTime();
save.inventory = {{"Sword", 1}, {"Potion", 5}};
save.checkpoints = {{100, 200}, {300, 400}};
save.save("game.sav");

SaveData loaded;
loaded.load("game.sav");
```

```cpp
#include <QBuffer>

// 写入内存
```

### 6.5 网络传输

```cpp
// 发送端：将数据打包到 QByteArray 后发送
QByteArray sendData;
QDataStream out(&sendData, QIODevice::WriteOnly);
out.setVersion(QDataStream::Qt_5_15);

// 先占位写长度（4字节），后面再填
out << (quint32)0;

// 写入有效载荷
out << QString("Hello") << (qint32)42;

// 回填实际长度（不含长度字段本身的 4 字节）
out.device()->seek(0);
out << (quint32)(sendData.size() - sizeof(quint32));

socket->write(sendData);

// 接收端：根据长度边界拆包
void onReadyRead() {
    QDataStream in(socket);
    in.setVersion(QDataStream::Qt_5_15);

    // 等待足够的数据
    if (m_blockSize == 0) {
        if (socket->bytesAvailable() < sizeof(quint32))
            return;
        in >> m_blockSize;
    }

    if (socket->bytesAvailable() < m_blockSize)
        return;

    // 读取有效载荷
    QString msg;
    qint32 num;
    in >> msg >> num;

    m_blockSize = 0;  // 为下一个包重置
}
```

### 6.6 格式化表格输出

```cpp
void printTable(const QList<QStringList> &rows, const QStringList &headers) {
    QString output;
    QTextStream out(&output);

    // 计算列宽
    QList<int> widths;
    for (int col = 0; col < headers.size(); col++) {
        int maxWidth = headers[col].length();
        for (const auto &row : rows) {
            if (col < row.size())
                maxWidth = qMax(maxWidth, row[col].length());
        }
        widths.append(maxWidth + 2);  // 加 2 留边距
    }

    // 表头
    for (int col = 0; col < headers.size(); col++) {
        out.setFieldWidth(widths[col]);
        out.setFieldAlignment(QTextStream::AlignLeft);
        out << headers[col];
    }
    out.setFieldWidth(0);
    out << "\n";

    // 分隔线
    for (int w : widths) {
        out << QString(w, '-');
    }
    out << "\n";

    // 数据行
    for (const auto &row : rows) {
        for (int col = 0; col < headers.size(); col++) {
            out.setFieldWidth(widths[col]);
            out << (col < row.size() ? row[col] : "");
        }
        out.setFieldWidth(0);
        out << "\n";
    }

    qDebug().noquote() << output;
}

// 使用
printTable({
    {"Alice", "30", "95.5"},
    {"Bob",   "25", "87.3"},
    {"Charlie", "35", "92.1"}
}, {"Name", "Age", "Score"});

// 输出：
// Name      Age  Score
// --------- ---- ------
// Alice     30   95.5
// Bob       25   87.3
// Charlie   35   92.1
```

## 7. 原始 I/O：readRawData / writeRawData

`QDataStream` 的 `readRawData` / `writeRawData` 绕过长度前缀和编码，直接读写原始字节。用于与非 Qt 程序交换二进制数据。

```cpp
// 写入：不加长度前缀
QDataStream out(&file);
char header[4] = {'R', 'I', 'F', 'F'};
out.writeRawData(header, 4);  // 精确写入 4 字节

// 写入固定长度字段
struct FileHeader {
    char magic[4];
    quint32 version;
    quint32 dataSize;
};
FileHeader hdr = {{'M','Y','F','M'}, 1, 1024};
out.writeRawData(reinterpret_cast<const char*>(&hdr), sizeof(hdr));
// ⚠ 注意字节序和对齐问题

// 读取
char magic[4];
in.readRawData(magic, 4);
if (memcmp(magic, "RIFF", 4) == 0) {
    // ...
}

// 对比 writeBytes / readBytes（会自动加 quint32 长度前缀）
out.writeBytes("Hello", 5);   // 写入 [00 00 00 05] [48 65 6C 6C 6F]
char *buf;
uint len;
in.readBytes(buf, len);       // 读取长度前缀 + 数据
delete[] buf;                 // 需要手动释放
```

## 8. 性能考虑

### 8.1 缓冲与刷新

```cpp
// QTextStream 内部有缓冲区，数据不会立即写入设备
QTextStream out(&file);
out << "data";        // 可能还在缓冲区中
out.flush();          // 强制刷新到设备
// 或 out << Qt::flush;

// QDataStream 没有内部缓冲（直接操作 QIODevice 的缓冲区）
// 但 QFile 本身有缓冲区
QFile file("data.bin");
file.open(QIODevice::WriteOnly);
QDataStream out(&file);
out << data;
file.flush();         // 刷新 QFile 的缓冲区到磁盘
```

### 8.2 大量小写入 vs 批量写入

```cpp
// ❌ 逐行 flush 效率低
for (int i = 0; i < 100000; i++) {
    out << "Line " << i << Qt::endl;  // endl 每次都 flush
}

// ✅ 用 "\n" 代替 endl，最后统一 flush
for (int i = 0; i < 100000; i++) {
    out << "Line " << i << "\n";
}
out.flush();  // 一次性 flush

// ✅ 或用 QBuffer 先写入内存，再一次性写入文件
QByteArray buffer;
QTextStream out(&buffer);
for (int i = 0; i < 100000; i++) {
    out << "Line " << i << "\n";
}
out.flush();
file.write(buffer);
```

### 8.3 QDataStream vs 直接 QFile::write

```cpp
// QDataStream 有开销：类型标记、长度前缀、字节序转换
// 对性能极端敏感的场景，可直接用 QFile::write

QFile file("binary.dat");
file.open(QIODevice::WriteOnly);

// 直接写入原始结构体（注意：不跨平台，有对齐问题）
struct RawPoint { float x, y, z; };
RawPoint points[1000];
file.write(reinterpret_cast<const char*>(points), sizeof(points));

// QDataStream 方式（安全、跨平台，但稍慢）
QDataStream out(&file);
for (int i = 0; i < 1000; i++) {
    out << points[i].x << points[i].y << points[i].z;
}
```

## 9. 常见问题

**问题 1：QDataStream 读取顺序错误导致数据混乱**

```cpp
// 写入顺序
out << name << age << score;

// ❌ 读取顺序不一致 → 数据错乱（无错误提示）
in >> age >> name >> score;

// ✅ 读取顺序必须与写入完全一致
in >> name >> age >> score;

// 建议：对复杂数据封装为 struct，统一 << >> 运算符
```

**问题 2：QDataStream 版本不匹配**

```cpp
// ❌ 写入时用 Qt 5.15 格式，读取时用 Qt 6.0 格式
// 可能导致某些类型（如 QDateTime）反序列化异常

// ✅ 总是显式设置相同版本
out.setVersion(QDataStream::Qt_5_15);
in.setVersion(QDataStream::Qt_5_15);
```

**问题 3：QTextStream 写入 QByteArray 后内容为空**

```cpp
// ❌ 忘记 flush
QByteArray data;
QTextStream out(&data);
out << "Hello";
qDebug() << data;  // 空！数据还在缓冲区

// ✅ 必须 flush 或关闭设备
out.flush();
qDebug() << data;  // "Hello"
```

**问题 4：整数类型宽度不确定**

```cpp
// ❌ int 大小因平台而异（可能是 32 或 64 位）
int val = 42;
out << val;  // 不可移植

// ✅ 使用定宽类型
qint32 val = 42;
out << val;  // 始终 4 字节

// 或显式转换
out << (qint32)val;
```

**问题 5：QTextStream 读取中文乱码**

```cpp
// ❌ Qt 5 默认使用系统 locale 编码
QTextStream in(&file);
QString text = in.readAll();  // 可能乱码

// ✅ 明确指定编码
QTextStream in(&file);
in.setCodec("UTF-8");  // Qt 5
QString text = in.readAll();
// Qt 6 默认 UTF-8，通常无此问题
```

**问题 6：QTextStream::readLine() 后 >> 行为异常**

```cpp
// readLine() 和 >> 混用可能导致意外
in.readLine();  // 读取并消费整行（含换行符）
in >> val;      // 继续读下一行的第一个 token — OK

// 但反过来：
in >> val;      // 读取一个 token，换行符未消费
in.readLine();  // 只读到行末剩余部分（可能为空）

// 建议：不要混用 readLine() 和 >>
```

## 10. 最佳实践

| 实践 | 说明 |
|---|---|
| 人类可读文件用 QTextStream | 配置文件、日志、CSV 等 |
| 应用数据用 QDataStream | 存档、缓存、网络传输等 |
| QDataStream 必须设版本号 | `setVersion()` 保证跨版本兼容 |
| 加魔数和版本头 | 二进制文件头部写标识+版本，方便校验 |
| 用定宽整数 | `qint32` 而非 `int`，确保跨平台一致 |
| 读写顺序一致 | QDataStream 按写入顺序读取，否则数据错乱 |
| QTextStream 明确编码 | Qt 5 显式 `setCodec("UTF-8")`，避免乱码 |
| 性能敏感用 `\n` | 避免 `Qt::endl` 频繁 flush |
| 自定义类型重载 `<<` `>>` | 集中序列化逻辑，支持版本演进 |
| 网络传输加长度前缀 | 先写 `quint32` 长度再写数据，接收端按长度拆包 |
