# 数据模型

Qt SQL 模块提供了三个核心数据模型类，将数据库表映射到 Qt 的 Model/View 架构中，使 `QTableView`、`QListView`、`QComboBox` 等控件能直接展示和编辑数据库内容，无需手写 SQL 即可完成常见的 CRUD 操作。

| 模型类 | 读写模式 | 特点 |
|---|---|---|
| `QSqlQueryModel` | **只读** | 执行任意 SQL 查询，结果作为只读模型 |
| `QSqlTableModel` | **读写** | 映射单张表，支持增删改，可选提交策略 |
| `QSqlRelationalTableModel` | **读写** | 在 `QSqlTableModel` 基础上支持外键关联（JOIN） |

三者的继承关系：

```
QAbstractTableModel
  └── QSqlQueryModel          （只读）
        └── QSqlTableModel    （读写，单表）
              └── QSqlRelationalTableModel  （读写，外键关联）
```

---

## 1. QSqlQueryModel — 只读查询模型

### 1.1 基本用法

```cpp
#include <QSqlQueryModel>
#include <QTableView>

QSqlQueryModel *model = new QSqlQueryModel(this);
model->setQuery("SELECT id, name, age, email FROM employees ORDER BY name");

// 检查查询是否成功
if (model->lastError().isValid()) {
    qWarning() << "Query error:" << model->lastError().text();
    return;
}

// 设置列标题
model->setHeaderData(0, Qt::Horizontal, tr("ID"));
model->setHeaderData(1, Qt::Horizontal, tr("姓名"));
model->setHeaderData(2, Qt::Horizontal, tr("年龄"));
model->setHeaderData(3, Qt::Horizontal, tr("邮箱"));

QTableView *view = new QTableView;
view->setModel(model);
view->resizeColumnsToContents();
view->show();
```

### 1.2 获取数据

```cpp
// 行数和列数
int rows = model->rowCount();
int cols = model->columnCount();

// 获取某个单元格的值
QModelIndex idx = model->index(0, 1);  // 第 0 行，第 1 列
QString name = model->data(idx).toString();

// 获取底层 QSqlRecord
QSqlRecord record = model->record(0);  // 第 0 行
QString email = record.value("email").toString();
int age = record.value("age").toInt();

// 遍历所有行
for (int row = 0; row < model->rowCount(); ++row) {
    QSqlRecord r = model->record(row);
    qDebug() << r.value("name").toString() << r.value("age").toInt();
}
```

### 1.3 带参数的查询

```cpp
// 方式一：直接拼接（小心 SQL 注入，仅限内部使用）
model->setQuery(QString("SELECT * FROM employees WHERE department = '%1'").arg(dept));

// 方式二：使用 QSqlQuery 参数绑定（推荐）
QSqlQuery query;
query.prepare("SELECT * FROM employees WHERE department = :dept AND age > :age");
query.bindValue(":dept", "Engineering");
query.bindValue(":age", 25);
query.exec();

model->setQuery(std::move(query));  // Qt 6
// Qt 5: model->setQuery(query);
```

### 1.4 刷新数据

```cpp
// 重新执行查询
model->setQuery(model->query().executedQuery());

// 或保存查询语句，重新设置
QString sql = "SELECT * FROM employees";
model->setQuery(sql);  // 每次调用都重新执行
```

### 1.5 将只读模型变为可写

通过继承 `QSqlQueryModel` 并重写 `flags()` 和 `setData()`：

```cpp
class EditableSqlModel : public QSqlQueryModel {
    Q_OBJECT
public:
    using QSqlQueryModel::QSqlQueryModel;

    Qt::ItemFlags flags(const QModelIndex &index) const override {
        Qt::ItemFlags f = QSqlQueryModel::flags(index);
        if (index.column() == 1 || index.column() == 2) {  // name 和 age 列可编辑
            f |= Qt::ItemIsEditable;
        }
        return f;
    }

    bool setData(const QModelIndex &index, const QVariant &value, int role) override {
        if (role != Qt::EditRole) return false;

        // 获取该行的主键
        int id = QSqlQueryModel::data(this->index(index.row(), 0)).toInt();

        QSqlQuery query;
        bool ok = false;

        switch (index.column()) {
        case 1:  // name
            query.prepare("UPDATE employees SET name = ? WHERE id = ?");
            query.addBindValue(value.toString());
            query.addBindValue(id);
            ok = query.exec();
            break;
        case 2:  // age
            query.prepare("UPDATE employees SET age = ? WHERE id = ?");
            query.addBindValue(value.toInt());
            query.addBindValue(id);
            ok = query.exec();
            break;
        }

        if (ok) {
            // 刷新模型
            setQuery(this->query().executedQuery());
            emit dataChanged(index, index, {role});
        }
        return ok;
    }
};
```

## 2. QSqlTableModel — 单表读写模型

### 2.1 基本用法

```cpp
#include <QSqlTableModel>

QSqlTableModel *model = new QSqlTableModel(this);
model->setTable("employees");
model->setSort(1, Qt::AscendingOrder);  // 按第 1 列（name）升序

// 设置提交策略
model->setEditStrategy(QSqlTableModel::OnManualSubmit);

// 设置列标题
model->setHeaderData(0, Qt::Horizontal, tr("ID"));
model->setHeaderData(1, Qt::Horizontal, tr("姓名"));
model->setHeaderData(2, Qt::Horizontal, tr("年龄"));
model->setHeaderData(3, Qt::Horizontal, tr("部门"));

// 执行查询（必须调用）
if (!model->select()) {
    qWarning() << "Select failed:" << model->lastError().text();
}

QTableView *view = new QTableView;
view->setModel(model);
view->setColumnHidden(0, true);  // 隐藏 ID 列
view->resizeColumnsToContents();
```

### 2.2 三种提交策略

```cpp
enum EditStrategy {
    OnFieldChange,    // 字段修改后立即提交（每按下回车/离开单元格就 UPDATE）
    OnRowChange,      // 行切换时提交（编辑完整行后统一 UPDATE）
    OnManualSubmit    // 手动调用 submitAll() 才提交（支持批量操作和撤销）
};
```

| 策略 | 提交时机 | 可撤销 | 适用场景 |
|---|---|---|---|
| `OnFieldChange` | 单元格编辑完成 | 否 | 简单的即时修改 |
| `OnRowChange` | 光标移到另一行 | 当前行可撤销 | 行级编辑 |
| `OnManualSubmit` | 手动 `submitAll()` | `revertAll()` 撤销全部 | 批量编辑、事务操作 |

```cpp
// OnManualSubmit 的典型工作流
model->setEditStrategy(QSqlTableModel::OnManualSubmit);

// ... 用户在界面上编辑了多行数据 ...

// 提交所有修改
if (!model->submitAll()) {
    qWarning() << "Submit failed:" << model->lastError().text();
    model->revertAll();  // 回滚所有修改
}
```

### 2.3 过滤与排序

```cpp
// 设置过滤条件（相当于 WHERE 子句）
model->setFilter("department = 'Engineering' AND age > 25");
model->select();  // 重新查询

// 排序
model->setSort(2, Qt::DescendingOrder);  // 按 age 降序
model->select();

// 清除过滤
model->setFilter("");
model->select();

// 动态搜索
void onSearchTextChanged(const QString &text) {
    if (text.isEmpty()) {
        model->setFilter("");
    } else {
        // 注意：用 QSqlField 或参数化方式防止 SQL 注入
        QString escaped = text;
        escaped.replace("'", "''");  // 基本转义
        model->setFilter(QString("name LIKE '%%1%'").arg(escaped));
    }
    model->select();
}
```

### 2.4 增删改操作

#### 插入行

```cpp
// 方式一：在末尾插入空行，然后填充
int row = model->rowCount();
model->insertRow(row);
model->setData(model->index(row, 1), "Alice");
model->setData(model->index(row, 2), 28);
model->setData(model->index(row, 3), "Design");

// OnManualSubmit 模式需要手动提交
model->submitAll();

// 方式二：使用 QSqlRecord 插入
QSqlRecord record = model->record();
record.setValue("name", "Bob");
record.setValue("age", 32);
record.setValue("department", "Marketing");
// record.remove(record.indexOf("id"));  // 如果 id 是自增列，可移除

if (!model->insertRecord(-1, record)) {  // -1 表示追加到末尾
    qWarning() << model->lastError();
}
model->submitAll();
```

#### 删除行

```cpp
// 删除选中的行
QModelIndexList selected = view->selectionModel()->selectedRows();

// 逆序删除，避免索引偏移
std::sort(selected.begin(), selected.end(),
          [](const QModelIndex &a, const QModelIndex &b) {
    return a.row() > b.row();
});

for (const QModelIndex &idx : selected) {
    model->removeRow(idx.row());
}

model->submitAll();
```

#### 修改数据

```cpp
// 用户通过 QTableView 直接编辑即可
// 或代码修改：
model->setData(model->index(0, 1), "New Name");
model->submitAll();

// 使用 QSqlRecord 修改
QSqlRecord record = model->record(0);
record.setValue("name", "Updated Name");
record.setValue("age", 30);
model->setRecord(0, record);
model->submitAll();
```

### 2.5 事务支持

```cpp
void batchUpdate(QSqlTableModel *model) {
    QSqlDatabase db = model->database();

    db.transaction();

    // 多项修改
    model->setData(model->index(0, 2), 99);
    model->setData(model->index(1, 2), 88);
    model->insertRow(model->rowCount());
    model->setData(model->index(model->rowCount() - 1, 1), "NewPerson");

    if (model->submitAll()) {
        db.commit();
        qDebug() << "Transaction committed";
    } else {
        db.rollback();
        model->revertAll();
        qWarning() << "Transaction rolled back:" << model->lastError().text();
    }
}
```

## 3. QSqlRelationalTableModel — 外键关联模型

### 3.1 问题场景

假设有两张表：

```sql
CREATE TABLE departments (
    id INTEGER PRIMARY KEY,
    name TEXT NOT NULL
);

CREATE TABLE employees (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL,
    age INTEGER,
    dept_id INTEGER REFERENCES departments(id)
);
```

直接用 `QSqlTableModel` 查看 `employees` 表，`dept_id` 列只会显示数字 `1, 2, 3 ...`，用户看不到部门名称。

### 3.2 用 QSqlRelationalTableModel 自动关联

```cpp
#include <QSqlRelationalTableModel>
#include <QSqlRelationalDelegate>

QSqlRelationalTableModel *model = new QSqlRelationalTableModel(this);
model->setTable("employees");
model->setEditStrategy(QSqlTableModel::OnManualSubmit);

// 设置外键关联：employees.dept_id → departments.id，显示 departments.name
model->setRelation(3, QSqlRelation("departments", "id", "name"));
// 参数：(列索引, QSqlRelation(关联表, 关联表主键列, 显示列))

model->setHeaderData(0, Qt::Horizontal, tr("ID"));
model->setHeaderData(1, Qt::Horizontal, tr("姓名"));
model->setHeaderData(2, Qt::Horizontal, tr("年龄"));
model->setHeaderData(3, Qt::Horizontal, tr("部门"));  // 现在显示部门名称

model->select();

QTableView *view = new QTableView;
view->setModel(model);

// ⚠ 关键：设置关联委托，使编辑时出现下拉框选择部门
view->setItemDelegate(new QSqlRelationalDelegate(view));

view->show();
```

效果：
- 查看时，`dept_id` 列显示的是部门名称（如 "Engineering"）而非数字
- 编辑时，该列会出现下拉框（`QComboBox`），列出所有部门供选择

### 3.3 多外键关联

```cpp
// employees 表有 dept_id 和 manager_id 两个外键
model->setTable("employees");
model->setRelation(3, QSqlRelation("departments", "id", "name"));     // dept_id → 部门名
model->setRelation(4, QSqlRelation("employees", "id", "name"));       // manager_id → 经理名
model->select();
```

### 3.4 关联模型的过滤

```cpp
// ⚠ 注意：setFilter 中的列名需要使用"表名.列名"形式
// 因为关联查询会 JOIN 多张表，列名可能冲突
model->setFilter("departments.name = 'Engineering'");
// 或
model->setFilter("employees.age > 30");
model->select();
```

### 3.5 获取关联表的子模型

```cpp
// 获取 dept_id 列关联的 departments 表模型
QSqlTableModel *deptModel = model->relationModel(3);

// 可以用这个子模型填充独立的 QComboBox
QComboBox *combo = new QComboBox;
combo->setModel(deptModel);
combo->setModelColumn(deptModel->fieldIndex("name"));  // 显示 name 列
```

## 4. 自定义显示

### 4.1 重写 data() 自定义单元格内容

```cpp
class CustomEmployeeModel : public QSqlTableModel {
    Q_OBJECT
public:
    using QSqlTableModel::QSqlTableModel;

    QVariant data(const QModelIndex &index, int role = Qt::DisplayRole) const override {
        // 年龄列着色
        if (index.column() == 2 && role == Qt::ForegroundRole) {
            int age = QSqlTableModel::data(index, Qt::DisplayRole).toInt();
            if (age > 50) return QColor(Qt::red);
            if (age < 25) return QColor(Qt::blue);
        }

        // 年龄列添加"岁"后缀
        if (index.column() == 2 && role == Qt::DisplayRole) {
            return QString("%1 岁").arg(QSqlTableModel::data(index, role).toInt());
        }

        // 姓名列添加图标
        if (index.column() == 1 && role == Qt::DecorationRole) {
            return QIcon(":/icons/person.png");
        }

        // 对齐方式
        if (role == Qt::TextAlignmentRole) {
            if (index.column() == 2) return int(Qt::AlignCenter);
        }

        return QSqlTableModel::data(index, role);
    }
};
```

### 4.2 自定义委托进行高级编辑

```cpp
class AgeSpinBoxDelegate : public QStyledItemDelegate {
    Q_OBJECT
public:
    using QStyledItemDelegate::QStyledItemDelegate;

    QWidget *createEditor(QWidget *parent, const QStyleOptionViewItem &,
                          const QModelIndex &) const override {
        auto *editor = new QSpinBox(parent);
        editor->setRange(18, 120);
        editor->setSuffix(" 岁");
        return editor;
    }

    void setEditorData(QWidget *editor, const QModelIndex &index) const override {
        int value = index.model()->data(index, Qt::EditRole).toInt();
        static_cast<QSpinBox *>(editor)->setValue(value);
    }

    void setModelData(QWidget *editor, QAbstractItemModel *model,
                      const QModelIndex &index) const override {
        model->setData(index, static_cast<QSpinBox *>(editor)->value(), Qt::EditRole);
    }
};

// 对年龄列应用委托
```

## 5. 代理模型配合使用

### 5.1 QSortFilterProxyModel 实现搜索过滤

```cpp
#include <QSortFilterProxyModel>

QSqlTableModel *sourceModel = new QSqlTableModel(this);
sourceModel->setTable("employees");
sourceModel->select();

QSortFilterProxyModel *proxyModel = new QSortFilterProxyModel(this);
proxyModel->setSourceModel(sourceModel);
proxyModel->setFilterKeyColumn(1);  // 按姓名列过滤
proxyModel->setFilterCaseSensitivity(Qt::CaseInsensitive);

QTableView *view = new QTableView;
view->setModel(proxyModel);
view->setSortingEnabled(true);  // 允许点击列头排序

// 搜索框
QLineEdit *searchBox = new QLineEdit;
connect(searchBox, &QLineEdit::textChanged, proxyModel, [proxyModel](const QString &text) {
    proxyModel->setFilterRegularExpression(text);
});
```

### 5.2 多列搜索

```cpp
class MultiColumnFilterProxy : public QSortFilterProxyModel {
    Q_OBJECT
public:
    using QSortFilterProxyModel::QSortFilterProxyModel;

    void setFilterColumns(const QVector<int> &columns) {
        m_filterColumns = columns;
        invalidateFilter();
    }

protected:
    bool filterAcceptsRow(int sourceRow, const QModelIndex &sourceParent) const override {
        QString pattern = filterRegularExpression().pattern();
        if (pattern.isEmpty()) return true;

        for (int col : m_filterColumns) {
            QModelIndex idx = sourceModel()->index(sourceRow, col, sourceParent);
            if (sourceModel()->data(idx).toString().contains(pattern, Qt::CaseInsensitive)) {
                return true;
            }
        }
        return false;
    }

private:
    QVector<int> m_filterColumns;
};

// 使用
auto *proxy = new MultiColumnFilterProxy(this);
proxy->setSourceModel(model);
proxy->setFilterColumns({1, 3});  // 名字和部门列都参与搜索
```

## 6. 实战案例：完整的员工管理界面

```cpp
class EmployeeManager : public QWidget {
    Q_OBJECT
public:
    EmployeeManager(QWidget *parent = nullptr) : QWidget(parent) {
        setupDatabase();
        setupModel();
        setupUI();
    }

private:
    void setupDatabase() {
        QSqlDatabase db = QSqlDatabase::addDatabase("QSQLITE");
        db.setDatabaseName("employees.db");
        if (!db.open()) {
            QMessageBox::critical(this, "错误", "无法打开数据库");
            return;
        }

        QSqlQuery query;
        query.exec("CREATE TABLE IF NOT EXISTS departments ("
                   "id INTEGER PRIMARY KEY, name TEXT NOT NULL)");
        query.exec("CREATE TABLE IF NOT EXISTS employees ("
                   "id INTEGER PRIMARY KEY AUTOINCREMENT, "
                   "name TEXT NOT NULL, age INTEGER, "
                   "dept_id INTEGER REFERENCES departments(id))");

        // 插入示例数据
        query.exec("INSERT OR IGNORE INTO departments VALUES(1, '研发部')");
        query.exec("INSERT OR IGNORE INTO departments VALUES(2, '市场部')");
        query.exec("INSERT OR IGNORE INTO departments VALUES(3, '设计部')");
    }

    void setupModel() {
        m_model = new QSqlRelationalTableModel(this);
        m_model->setTable("employees");
        m_model->setRelation(3, QSqlRelation("departments", "id", "name"));
        m_model->setEditStrategy(QSqlTableModel::OnManualSubmit);
        m_model->setHeaderData(0, Qt::Horizontal, "ID");
        m_model->setHeaderData(1, Qt::Horizontal, "姓名");
        m_model->setHeaderData(2, Qt::Horizontal, "年龄");
        m_model->setHeaderData(3, Qt::Horizontal, "部门");
        m_model->select();

        // 代理模型（排序和过滤）
        m_proxy = new QSortFilterProxyModel(this);
        m_proxy->setSourceModel(m_model);
        m_proxy->setFilterKeyColumn(1);
        m_proxy->setFilterCaseSensitivity(Qt::CaseInsensitive);
    }

    void setupUI() {
        auto *layout = new QVBoxLayout(this);

        // === 工具栏 ===
        auto *toolbar = new QHBoxLayout;

        auto *searchEdit = new QLineEdit;
        searchEdit->setPlaceholderText("搜索姓名...");
        connect(searchEdit, &QLineEdit::textChanged,
                m_proxy, &QSortFilterProxyModel::setFilterFixedString);

        auto *addBtn = new QPushButton("添加");
        auto *delBtn = new QPushButton("删除");
        auto *saveBtn = new QPushButton("保存");
        auto *revertBtn = new QPushButton("撤销");
        auto *refreshBtn = new QPushButton("刷新");

        toolbar->addWidget(searchEdit);
        toolbar->addStretch();
        toolbar->addWidget(addBtn);
        toolbar->addWidget(delBtn);
        toolbar->addWidget(saveBtn);
        toolbar->addWidget(revertBtn);
        toolbar->addWidget(refreshBtn);

        // === 表格视图 ===
        m_view = new QTableView;
        m_view->setModel(m_proxy);
        m_view->setItemDelegate(new QSqlRelationalDelegate(m_view));
        m_view->setSelectionBehavior(QAbstractItemView::SelectRows);
        m_view->setSelectionMode(QAbstractItemView::ExtendedSelection);
        m_view->setColumnHidden(0, true);      // 隐藏 ID
        m_view->horizontalHeader()->setStretchLastSection(true);
        m_view->setSortingEnabled(true);
        m_view->resizeColumnsToContents();

        // === 状态栏 ===
        m_statusLabel = new QLabel;
        updateStatus();

        layout->addLayout(toolbar);
        layout->addWidget(m_view);
        layout->addWidget(m_statusLabel);

        // === 信号连接 ===
        connect(addBtn, &QPushButton::clicked, this, &EmployeeManager::addEmployee);
        connect(delBtn, &QPushButton::clicked, this, &EmployeeManager::deleteSelected);
        connect(saveBtn, &QPushButton::clicked, this, &EmployeeManager::saveChanges);
        connect(revertBtn, &QPushButton::clicked, this, &EmployeeManager::revertChanges);
        connect(refreshBtn, &QPushButton::clicked, this, &EmployeeManager::refresh);
    }

    void addEmployee() {
        int row = m_model->rowCount();
        m_model->insertRow(row);
        m_model->setData(m_model->index(row, 1), "新员工");
        m_model->setData(m_model->index(row, 2), 25);

        // 映射到代理模型的索引，定位到新行
        QModelIndex proxyIdx = m_proxy->mapFromSource(m_model->index(row, 1));
        m_view->setCurrentIndex(proxyIdx);
        m_view->edit(proxyIdx);
        updateStatus();
    }

    void deleteSelected() {
        QModelIndexList selected = m_view->selectionModel()->selectedRows();
        if (selected.isEmpty()) return;

        if (QMessageBox::question(this, "确认", "确定删除选中的行？")
            != QMessageBox::Yes) return;

        // 逆序删除
        QVector<int> rows;
        for (const auto &idx : selected) {
            rows.append(m_proxy->mapToSource(idx).row());
        }
        std::sort(rows.begin(), rows.end(), std::greater<int>());

        for (int row : rows) {
            m_model->removeRow(row);
        }
        updateStatus();
    }

    void saveChanges() {
        if (m_model->submitAll()) {
            m_statusLabel->setText("保存成功");
        } else {
            QMessageBox::warning(this, "保存失败", m_model->lastError().text());
            m_model->revertAll();
        }
        updateStatus();
    }

    void revertChanges() {
        m_model->revertAll();
        m_statusLabel->setText("已撤销所有修改");
        updateStatus();
    }

    void refresh() {
        m_model->select();
        updateStatus();
    }

    void updateStatus() {
        m_statusLabel->setText(QString("共 %1 条记录").arg(m_proxy->rowCount()));
    }

    QSqlRelationalTableModel *m_model = nullptr;
    QSortFilterProxyModel *m_proxy = nullptr;
    QTableView *m_view = nullptr;
    QLabel *m_statusLabel = nullptr;
};
```

## 7. 分页加载

当数据量很大时，一次加载全部数据会导致内存和性能问题。`QSqlQueryModel` 支持延迟加载（`fetchMore`），但也可以自行实现分页：

### 7.1 利用 fetchMore（自动延迟加载）

```cpp
QSqlQueryModel *model = new QSqlQueryModel;
model->setQuery("SELECT * FROM big_table");  // 百万行

// QSqlQueryModel 默认只拉取 256 行
qDebug() << model->rowCount();  // 可能 <= 256

// 手动拉取更多
while (model->canFetchMore()) {
    model->fetchMore();
}
qDebug() << model->rowCount();  // 全部行数
```

> `QTableView` 滚动到底部时会自动调用 `fetchMore()`。

### 7.2 手动分页

```cpp
class PaginatedModel : public QObject {
    Q_OBJECT
public:
    PaginatedModel(const QString &table, int pageSize = 50, QObject *parent = nullptr)
        : QObject(parent), m_table(table), m_pageSize(pageSize) {

        m_model = new QSqlQueryModel(this);

        // 获取总行数
        QSqlQuery countQuery(QString("SELECT COUNT(*) FROM %1").arg(table));
        if (countQuery.next()) {
            m_totalRows = countQuery.value(0).toInt();
        }

        loadPage(0);
    }

    QSqlQueryModel *model() const { return m_model; }
    int currentPage() const { return m_currentPage; }
    int totalPages() const { return (m_totalRows + m_pageSize - 1) / m_pageSize; }

public slots:
    void loadPage(int page) {
        if (page < 0 || page >= totalPages()) return;
        m_currentPage = page;
        int offset = page * m_pageSize;
        m_model->setQuery(QString("SELECT * FROM %1 LIMIT %2 OFFSET %3")
                          .arg(m_table).arg(m_pageSize).arg(offset));
        emit pageChanged(page, totalPages());
    }

    void nextPage() { loadPage(m_currentPage + 1); }
    void prevPage() { loadPage(m_currentPage - 1); }
    void firstPage() { loadPage(0); }
    void lastPage() { loadPage(totalPages() - 1); }

signals:
    void pageChanged(int current, int total);

private:
    QSqlQueryModel *m_model;
    QString m_table;
    int m_pageSize;
    int m_currentPage = 0;
    int m_totalRows = 0;
};

// 使用
auto *paged = new PaginatedModel("employees", 50);
view->setModel(paged->model());

connect(nextBtn, &QPushButton::clicked, paged, &PaginatedModel::nextPage);
connect(prevBtn, &QPushButton::clicked, paged, &PaginatedModel::prevPage);
connect(paged, &PaginatedModel::pageChanged, [pageLabel](int cur, int total) {
    pageLabel->setText(QString("第 %1/%2 页").arg(cur + 1).arg(total));
});
```

## 8. QSqlTableModel 与 QML

### 8.1 在 QML 中使用 SQL 模型

```cpp
class SqlContactModel : public QSqlTableModel {
    Q_OBJECT
public:
    SqlContactModel(QObject *parent = nullptr) : QSqlTableModel(parent) {
        setTable("contacts");
        select();
    }

    // QML 使用 roleName 访问数据，需要提供 roleNames
    QHash<int, QByteArray> roleNames() const override {
        QHash<int, QByteArray> roles;
        for (int i = 0; i < columnCount(); ++i) {
            roles[Qt::UserRole + i] = record().fieldName(i).toUtf8();
        }
        return roles;
    }

    QVariant data(const QModelIndex &index, int role) const override {
        if (role >= Qt::UserRole) {
            return QSqlTableModel::data(
                this->index(index.row(), role - Qt::UserRole), Qt::DisplayRole);
        }
        return QSqlTableModel::data(index, role);
    }
};
```

## 10. 性能优化

### 10.1 减少无效刷新

```cpp
// ❌ 频繁调用 select()
for (int i = 0; i < 100; ++i) {
    model->setData(model->index(i, 1), newVal);
    model->submitAll();  // 每次都会触发 select 刷新

// ✅ 批量修改后统一提交
model->setEditStrategy(QSqlTableModel::OnManualSubmit);
for (int i = 0; i < 100; ++i) {
```

## 11. 常见问题

**问题 1：`select()` 返回 false**

```cpp
if (!model->select()) {
    qWarning() << model->lastError().text();
    // "No query" — 忘记调用 setTable()
    // "no such table" — 表不存在
    // "unable to open database" — 数据库连接未打开
}
```

**问题 2：`submitAll()` 失败**

```cpp
if (!model->submitAll()) {
    qWarning() << model->lastError().text();
    // "UNIQUE constraint failed" — 唯一键冲突
    // "NOT NULL constraint" — 必填字段为空
    // "FOREIGN KEY constraint" — 外键指向不存在的记录
    model->revertAll();
}
```

**问题 3：关联模型列索引偏移**

```cpp
// QSqlRelationalTableModel 的 setRelation 会改变列的含义
// 设置关联后，data() 返回的是关联表的显示列值
// 要获取原始外键 ID，用：
QModelIndex idx = model->index(row, 3);  // dept_id 列
int displayDept = model->data(idx).toString();   // "研发部"（关联后）

// 底层 QSqlRecord 仍保存原始值
QSqlRecord rec = model->record(row);
int deptId = rec.value("dept_id").toInt();  // 1（原始 ID）
```

**问题 4：修改后界面不刷新**

```cpp
// OnManualSubmit 模式下，submitAll() 会自动调用 select() 刷新
// 如果用的是 QSqlQueryModel，需要手动重新 setQuery()

// 如果用了代理模型，确保信号正确传递
proxy->invalidate();  // 强制重新过滤/排序
```

**问题 5：QSqlRelationalDelegate 不在下拉框中显示选项**

```cpp
// 确保：
// 1. 调用了 model->setRelation(col, QSqlRelation(...))
// 2. model->select() 成功
// 3. view->setItemDelegate(new QSqlRelationalDelegate(view))
// 4. 关联目标表有数据
```

## 12. 三种模型对比总结

| 特性 | QSqlQueryModel | QSqlTableModel | QSqlRelationalTableModel |
|---|---|---|---|
| 读取数据 | ✅ 任意 SQL | ✅ 单表 | ✅ 单表 + 关联表 |
| 编辑数据 | ❌ 只读（需继承重写） | ✅ | ✅ |
| 插入/删除行 | ❌ | ✅ | ✅ |
| 外键显示 | ❌ | ❌ 显示原始 ID | ✅ 显示关联表字段 |
| 编辑时下拉框 | ❌ | ❌ | ✅（配合 QSqlRelationalDelegate） |
| 提交策略 | — | 三种策略可选 | 三种策略可选 |
| 过滤/排序 | SQL 中写 WHERE/ORDER BY | `setFilter()` / `setSort()` | 同 QSqlTableModel |
| 性能 | 最高（一次查询） | 较高 | 有 JOIN 开销 |
| 灵活性 | 最高（任意 SQL） | 单表操作 | 单表 + 关联 |
| 适用场景 | 复杂查询、报表展示 | 单表 CRUD | 含外键的表 CRUD |

## 13. 最佳实践

| 实践 | 说明 |
|---|---|
| 优先用 `OnManualSubmit` | 支持批量操作、事务和撤销，用户体验最好 |
| `submitAll` 后检查返回值 | 失败时调用 `revertAll()` 并提示用户 |
| SQL 注入防护 | `setFilter()` 中避免直接拼接用户输入，用转义或参数绑定 |
| 大数据量用分页或延迟加载 | 避免一次性加载数十万行到内存 |
| 隐藏主键列 | `view->setColumnHidden(0, true)` 让用户界面更简洁 |
| 搜索用代理模型 | `QSortFilterProxyModel` 比 `setFilter` 更灵活，不重新查库 |
| 支持事务 | 多步修改用 `transaction() / commit() / rollback()` 保护 |
| 重写 `roleNames()` 给 QML 用 | `QSqlTableModel` 默认不提供 QML 可用的角色名 |
| 关联模型必须设置 `QSqlRelationalDelegate` | 否则外键列编辑时只能手打 ID，无法选择关联值 |
