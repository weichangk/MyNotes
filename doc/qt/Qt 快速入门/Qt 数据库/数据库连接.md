# 数据库连接

Qt SQL 模块通过**驱动插件**机制支持多种数据库后端（SQLite、MySQL、PostgreSQL、Oracle、SQL Server 等），并以 `QSqlDatabase` 作为统一的连接管理入口。理解连接的创建、命名、生命周期和线程安全规则是使用 Qt 数据库的第一步。

## 2. 创建数据库连接

### 2.1 默认连接

```cpp
// 创建默认连接（连接名为 QSqlDatabase::defaultConnection，即 "qt_sql_default_connection"）
QSqlDatabase db = QSqlDatabase::addDatabase("QSQLITE");
db.setDatabaseName("myapp.db");

if (!db.open()) {
    qCritical() << "Database open failed:" << db.lastError().text();
    return;
}

// 后续任何地方获取默认连接（不传连接名）
QSqlDatabase db2 = QSqlDatabase::database();
QSqlQuery query;  // 自动使用默认连接
query.exec("SELECT * FROM users");
```

### 2.2 命名连接

```cpp
// 创建命名连接
QSqlDatabase db = QSqlDatabase::addDatabase("QSQLITE", "my_connection");
db.setDatabaseName("data.db");
db.open();

// 获取命名连接
QSqlDatabase db2 = QSqlDatabase::database("my_connection");

// 使用命名连接执行查询
QSqlQuery query(QSqlDatabase::database("my_connection"));
query.exec("SELECT * FROM products");
```

### 2.3 addDatabase 详解

```cpp
// 函数签名
static QSqlDatabase QSqlDatabase::addDatabase(
    const QString &type,                   // 驱动标识
    const QString &connectionName = defaultConnection  // 连接名
);
```

> **重要**：`addDatabase` 只是**注册**一个连接配置，此时并未真正建立网络/文件连接。只有调用 `open()` 后才真正连接。

```cpp
// ⚠ 对同一个连接名重复调用 addDatabase 会产生警告
QSqlDatabase db1 = QSqlDatabase::addDatabase("QSQLITE", "conn1");
QSqlDatabase db2 = QSqlDatabase::addDatabase("QSQLITE", "conn1");  // 警告！覆盖旧连接
// QSqlDatabasePrivate::addDatabase: duplicate connection name 'conn1', old connection removed.

// ✅ 先检查是否已存在
if (QSqlDatabase::contains("conn1")) {
    QSqlDatabase db = QSqlDatabase::database("conn1");
} else {
    QSqlDatabase db = QSqlDatabase::addDatabase("QSQLITE", "conn1");
    db.setDatabaseName("data.db");
    db.open();
}
```

## 4. 连接生命周期管理

### 4.1 打开与关闭

```cpp
QSqlDatabase db = QSqlDatabase::addDatabase("QSQLITE", "work");
db.setDatabaseName("work.db");

// 打开
if (!db.open()) {
    qCritical() << db.lastError();
    return;
}

// 检查是否打开
qDebug() << db.isOpen();       // true
qDebug() << db.isValid();      // true（连接对象有效）

// 关闭（释放底层连接，但连接名仍注册）
db.close();
qDebug() << db.isOpen();       // false
qDebug() << db.isValid();      // true（对象仍有效，可再次 open）

// 重新打开
db.open();
```

### 4.2 完全移除连接

```cpp
{
    QSqlDatabase db = QSqlDatabase::database("work");
    db.close();
}   // db 变量离开作用域

// 移除连接注册（必须在所有 QSqlDatabase 对象析构后调用）
QSqlDatabase::removeDatabase("work");
```

> **常见错误**：在 `QSqlDatabase` 对象仍存活时调用 `removeDatabase`：

```cpp
// ❌ 错误写法——db 仍在作用域中
QSqlDatabase db = QSqlDatabase::database("work");
db.close();
QSqlDatabase::removeDatabase("work");
// 警告: QSqlDatabasePrivate::removeDatabase: connection 'work' is still in use, all queries will cease to work.

// ✅ 正确写法——确保对象先析构
{
    QSqlDatabase db = QSqlDatabase::database("work");
    db.close();
}   // db 在这里析构
QSqlDatabase::removeDatabase("work");  // 安全
```

### 4.3 连接管理器封装

```cpp
class DatabaseManager {
public:
    static DatabaseManager &instance() {
        static DatabaseManager inst;
        return inst;
    }

    // 初始化主连接
    bool init(const QString &dbPath) {
        if (QSqlDatabase::contains("main")) {
            return QSqlDatabase::database("main").isOpen();
        }
        QSqlDatabase db = QSqlDatabase::addDatabase("QSQLITE", "main");
        db.setDatabaseName(dbPath);
        if (!db.open()) {
            qCritical() << "DB open failed:" << db.lastError().text();
            return false;
        }
        initSchema(db);
        return true;
    }

    // 获取连接（供非主线程使用时需复制连接，见 §7）
    QSqlDatabase database() const {
        return QSqlDatabase::database("main");
    }

    // 关闭并移除所有连接
    void shutdown() {
        QStringList names = QSqlDatabase::connectionNames();
        for (const QString &name : names) {
            {
                QSqlDatabase db = QSqlDatabase::database(name);
                db.close();
            }
            QSqlDatabase::removeDatabase(name);
        }
    }

private:
    DatabaseManager() = default;
    ~DatabaseManager() { shutdown(); }

    void initSchema(QSqlDatabase &db) {
        QSqlQuery query(db);
        query.exec("CREATE TABLE IF NOT EXISTS users ("
                   "id INTEGER PRIMARY KEY AUTOINCREMENT, "
                   "name TEXT NOT NULL, "
                   "email TEXT UNIQUE, "
                   "created_at TEXT DEFAULT (datetime('now')))");
    }

    Q_DISABLE_COPY(DatabaseManager)
};

// 使用
DatabaseManager::instance().init("app.db");
QSqlQuery query(DatabaseManager::instance().database());
query.exec("SELECT * FROM users");
```

## 5. 连接检测与重连

### 5.1 检测连接是否有效

```cpp
QSqlDatabase db = QSqlDatabase::database();

// 方式一：isOpen()（只检查对象状态，不实际测试网络）
if (!db.isOpen()) { /* ... */ }

// 方式二：执行轻量查询（真正测试网络连通性）
bool isAlive(const QSqlDatabase &db) {
    if (!db.isOpen()) return false;
    QSqlQuery query(db);
    // SQLite
    return query.exec("SELECT 1");
    // MySQL: SELECT 1 或 SELECT NOW()
    // PostgreSQL: SELECT 1
    // Oracle: SELECT 1 FROM DUAL
}
```

### 5.2 自动重连

```cpp
class ConnectionKeeper : public QObject {
    Q_OBJECT
public:
    ConnectionKeeper(const QString &connName, int intervalMs = 30000,
                     QObject *parent = nullptr)
        : QObject(parent), m_connName(connName) {

        m_timer = new QTimer(this);
        m_timer->setInterval(intervalMs);
        connect(m_timer, &QTimer::timeout, this, &ConnectionKeeper::checkAndReconnect);
        m_timer->start();
    }

private slots:
    void checkAndReconnect() {
        QSqlDatabase db = QSqlDatabase::database(m_connName);
        if (!db.isValid()) return;

        QSqlQuery query(db);
        if (!query.exec("SELECT 1")) {
            qWarning() << "Connection lost, attempting reconnect...";

            db.close();
            if (db.open()) {
                qInfo() << "Reconnected successfully";
                emit reconnected();
            } else {
                qCritical() << "Reconnect failed:" << db.lastError().text();
                emit reconnectFailed();
            }
        }
    }

signals:
    void reconnected();
    void reconnectFailed();

private:
    QTimer *m_timer;
    QString m_connName;
};
```

### 5.3 连接超时与错误处理

```cpp
bool connectWithRetry(QSqlDatabase &db, int maxRetries = 3, int delayMs = 2000) {
    for (int i = 0; i < maxRetries; ++i) {
        if (db.open()) {
            qInfo() << "Connected on attempt" << (i + 1);
            return true;
        }
        qWarning() << QString("Attempt %1/%2 failed: %3")
                       .arg(i + 1).arg(maxRetries).arg(db.lastError().text());

        if (i < maxRetries - 1) {
            QThread::msleep(delayMs);
        }
    }
    qCritical() << "All connection attempts failed";
    return false;
}

// 使用
QSqlDatabase db = QSqlDatabase::addDatabase("QMYSQL");
db.setHostName("192.168.1.100");
db.setDatabaseName("mydb");
db.setUserName("root");
db.setPassword("secret");
db.setConnectOptions("MYSQL_OPT_CONNECT_TIMEOUT=3");

if (!connectWithRetry(db, 3, 2000)) {
    // 显示离线模式或退出
}
```

## 7. 多线程与数据库连接

### 7.1 核心规则

> **Qt 官方文档明确规定：一个 `QSqlDatabase` 连接只能在创建它的线程中使用。不能跨线程共享连接。**

```cpp
// ❌ 错误：在主线程创建连接，在子线程使用
QSqlDatabase db = QSqlDatabase::addDatabase("QSQLITE");
db.setDatabaseName("data.db");
db.open();

QtConcurrent::run([&]() {
    QSqlQuery query(db);  // ❌ 跨线程使用连接
    query.exec("SELECT * FROM users");
});
```

### 7.2 正确方案：每个线程创建自己的连接

```cpp
// 方案一：在子线程中创建独立连接

void workerFunction() {
    // 使用唯一连接名（通常用线程 ID）
    QString connName = QString("worker_%1")
        .arg(reinterpret_cast<quintptr>(QThread::currentThread()), 0, 16);

    {
        QSqlDatabase db = QSqlDatabase::addDatabase("QSQLITE", connName);
        db.setDatabaseName("data.db");

        if (db.open()) {
            QSqlQuery query(db);
            query.exec("SELECT COUNT(*) FROM users");
            if (query.next()) {
                qDebug() << "User count:" << query.value(0).toInt();
            }
            db.close();
        }
    }

    QSqlDatabase::removeDatabase(connName);
}

// 使用
QFuture<void> future = QtConcurrent::run(workerFunction);
```

### 7.3 线程安全的连接池

```cpp
class ConnectionPool {
public:
    static ConnectionPool &instance() {
        static ConnectionPool pool;
        return pool;
    }

    // 初始化（主线程调用一次）
    void init(const QString &driver, const QString &dbName,
              const QString &host = {}, int port = -1,
              const QString &user = {}, const QString &password = {},
              int maxConns = 10) {
        m_driver = driver;
        m_dbName = dbName;
        m_host = host;
        m_port = port;
        m_user = user;
        m_password = password;
        m_maxConns = maxConns;
    }

    // 获取连接（线程安全）
    QSqlDatabase acquire() {
        QMutexLocker locker(&m_mutex);

        // 复用空闲连接
        if (!m_idle.isEmpty()) {
            QString name = m_idle.dequeue();
            QSqlDatabase db = QSqlDatabase::database(name);
            if (db.isOpen()) {
                m_active.insert(name);
                return db;
            }
            // 连接已断开，移除
            QSqlDatabase::removeDatabase(name);
        }

        // 创建新连接
        if (m_nextId < m_maxConns) {
            QString name = QString("pool_%1").arg(m_nextId++);
            QSqlDatabase db = QSqlDatabase::addDatabase(m_driver, name);
            db.setDatabaseName(m_dbName);
            if (!m_host.isEmpty()) db.setHostName(m_host);
            if (m_port > 0) db.setPort(m_port);
            if (!m_user.isEmpty()) db.setUserName(m_user);
            if (!m_password.isEmpty()) db.setPassword(m_password);

            if (db.open()) {
                m_active.insert(name);
                return db;
            }
            qWarning() << "Pool: open failed:" << db.lastError().text();
            QSqlDatabase::removeDatabase(name);
        }

        qWarning() << "Pool: no available connections (max:" << m_maxConns << ")";
        return QSqlDatabase();  // 无效连接
    }

    // 归还连接
    void release(const QSqlDatabase &db) {
        QMutexLocker locker(&m_mutex);
        QString name = db.connectionName();
        if (m_active.remove(name)) {
            m_idle.enqueue(name);
        }
    }

    // 关闭所有
    void shutdown() {
        QMutexLocker locker(&m_mutex);
        for (const QString &name : m_active + QSet<QString>(m_idle.begin(), m_idle.end())) {
            {
                QSqlDatabase db = QSqlDatabase::database(name);
                db.close();
            }
            QSqlDatabase::removeDatabase(name);
        }
        m_active.clear();
        m_idle.clear();
    }

private:
    ConnectionPool() = default;
    ~ConnectionPool() { shutdown(); }
    Q_DISABLE_COPY(ConnectionPool)

    QMutex m_mutex;
    QQueue<QString> m_idle;
    QSet<QString> m_active;
    int m_nextId = 0;
    int m_maxConns = 10;
    QString m_driver, m_dbName, m_host, m_user, m_password;
    int m_port = -1;
};

// RAII 连接守卫
class ScopedConnection {
public:
    ScopedConnection() : m_db(ConnectionPool::instance().acquire()) {}
    ~ScopedConnection() {
        if (m_db.isValid())
            ConnectionPool::instance().release(m_db);
    }

    QSqlDatabase &database() { return m_db; }
    bool isValid() const { return m_db.isValid() && m_db.isOpen(); }

    // 禁止拷贝
    ScopedConnection(const ScopedConnection &) = delete;
    ScopedConnection &operator=(const ScopedConnection &) = delete;

private:
    QSqlDatabase m_db;
};

// 使用
ConnectionPool::instance().init("QMYSQL", "mydb", "localhost", 3306, "root", "secret", 10);

// 在任意线程中
void doQuery() {
    ScopedConnection conn;
    if (!conn.isValid()) return;

    QSqlQuery query(conn.database());
    query.exec("SELECT * FROM users LIMIT 10");
    while (query.next()) {
        qDebug() << query.value("name").toString();
    }
}   // 自动归还连接

// 并发调用
QFuture<void> f1 = QtConcurrent::run(doQuery);
QFuture<void> f2 = QtConcurrent::run(doQuery);
QFuture<void> f3 = QtConcurrent::run(doQuery);
```

## 8. 事务管理

### 8.1 基本事务操作

```cpp
QSqlDatabase db = QSqlDatabase::database();

// 检查数据库是否支持事务
if (!db.driver()->hasFeature(QSqlDriver::Transactions)) {
    qWarning() << "Database does not support transactions";
    return;
}

// 开始事务
if (!db.transaction()) {
    qWarning() << "Transaction start failed:" << db.lastError().text();
    return;
}

QSqlQuery query(db);

bool ok = true;
ok &= query.exec("INSERT INTO orders (customer_id, total) VALUES (1, 99.99)");

// 获取自增 ID
int orderId = query.lastInsertId().toInt();

ok &= query.exec(QString("INSERT INTO order_items (order_id, product_id, qty) "
                          "VALUES (%1, 100, 2)").arg(orderId));
ok &= query.exec(QString("UPDATE products SET stock = stock - 2 WHERE id = 100"));

if (ok) {
    db.commit();
    qDebug() << "Transaction committed, order ID:" << orderId;
} else {
    db.rollback();
    qWarning() << "Transaction rolled back:" << query.lastError().text();
}
```

### 8.2 RAII 事务守卫

```cpp
class TransactionGuard {
public:
    explicit TransactionGuard(QSqlDatabase db = QSqlDatabase::database())
        : m_db(db), m_committed(false) {
        m_active = m_db.transaction();
        if (!m_active) {
            qWarning() << "Failed to start transaction:" << m_db.lastError().text();
        }
    }

    ~TransactionGuard() {
        if (m_active && !m_committed) {
            m_db.rollback();
            qDebug() << "Transaction auto-rolled back (guard destroyed)";
        }
    }

    bool isActive() const { return m_active; }

    bool commit() {
        if (m_active && !m_committed) {
            m_committed = m_db.commit();
            if (!m_committed) {
                qWarning() << "Commit failed:" << m_db.lastError().text();
                m_db.rollback();
            }
            return m_committed;
        }
        return false;
    }

    // 禁止拷贝
    TransactionGuard(const TransactionGuard &) = delete;
    TransactionGuard &operator=(const TransactionGuard &) = delete;

private:
    QSqlDatabase m_db;
    bool m_active;
    bool m_committed;
};

// 使用——异常安全
void transferMoney(int fromId, int toId, double amount) {
    TransactionGuard tx;
    if (!tx.isActive()) return;

    QSqlQuery query;
    query.prepare("UPDATE accounts SET balance = balance - :amount WHERE id = :id AND balance >= :amount");
    query.bindValue(":amount", amount);
    query.bindValue(":id", fromId);

    if (!query.exec() || query.numRowsAffected() != 1) {
        qWarning() << "Debit failed";
        return;  // 析构时自动回滚
    }

    query.prepare("UPDATE accounts SET balance = balance + :amount WHERE id = :id");
    query.bindValue(":amount", amount);
    query.bindValue(":id", toId);

    if (!query.exec()) {
        qWarning() << "Credit failed";
        return;  // 析构时自动回滚
    }

    tx.commit();  // 成功时显式提交
}
```

        break;

## 12. 实战案例：应用启动时的数据库初始化

```cpp
class AppDatabase {
public:
    static bool initialize(const QString &appDataPath) {
        QString dbPath = appDataPath + "/appdata.db";

        // 确保目录存在
        QDir().mkpath(appDataPath);

        QSqlDatabase db = QSqlDatabase::addDatabase("QSQLITE");
        db.setDatabaseName(dbPath);
        db.setConnectOptions("QSQLITE_BUSY_TIMEOUT=5000");

        if (!db.open()) {
            qCritical() << "Cannot open database:" << db.lastError().text();
            return false;
        }

        // 启用 WAL 模式和外键
        QSqlQuery query(db);
        query.exec("PRAGMA journal_mode=WAL");
        query.exec("PRAGMA foreign_keys=ON");

        // 执行迁移
        if (!migrate(db)) {
            qCritical() << "Database migration failed";
            return false;
        }

        qInfo() << "Database initialized at" << dbPath;
        return true;
    }

private:
    static bool migrate(QSqlDatabase &db) {
        QSqlQuery query(db);

        // 版本管理表
        query.exec("CREATE TABLE IF NOT EXISTS db_version ("
                   "version INTEGER PRIMARY KEY)");

        query.exec("SELECT MAX(version) FROM db_version");
        int currentVersion = query.next() ? query.value(0).toInt() : 0;

        // 迁移脚本
        struct Migration {
            int version;
            const char *sql;
        };

        Migration migrations[] = {
            {1, "CREATE TABLE users ("
                "id INTEGER PRIMARY KEY AUTOINCREMENT, "
                "name TEXT NOT NULL, "
                "email TEXT UNIQUE NOT NULL, "
                "created_at TEXT DEFAULT (datetime('now')))"},

            {2, "CREATE TABLE settings ("
                "key TEXT PRIMARY KEY, "
                "value TEXT)"},

            {3, "ALTER TABLE users ADD COLUMN avatar_path TEXT"},

            {4, "CREATE INDEX idx_users_email ON users(email)"},
        };

        for (const auto &m : migrations) {
            if (m.version <= currentVersion) continue;

            TransactionGuard tx(db);
            if (!tx.isActive()) return false;

            if (!query.exec(m.sql)) {
                qWarning() << "Migration" << m.version << "failed:"
                           << query.lastError().text();
                return false;
            }

            query.prepare("INSERT INTO db_version (version) VALUES (:v)");
            query.bindValue(":v", m.version);
            if (!query.exec()) return false;

            tx.commit();
            qInfo() << "Applied migration" << m.version;
        }

        return true;
    }
};

// 在 main() 中调用
int main(int argc, char *argv[]) {
    QApplication app(argc, argv);

    QString dataDir = QStandardPaths::writableLocation(QStandardPaths::AppDataLocation);
    if (!AppDatabase::initialize(dataDir)) {
        QMessageBox::critical(nullptr, "错误", "数据库初始化失败");
        return 1;
    }

    MainWindow w;
    w.show();
    return app.exec();
}
```

## 13. 常见问题

**问题 1：`QSqlDatabase: QMYSQL driver not loaded`**

```
// 原因：缺少 MySQL 驱动插件（qsqlmysql.dll / libqsqlmysql.so）
// 排查步骤：
qDebug() << QSqlDatabase::drivers();  // 查看可用驱动列表
qDebug() << QCoreApplication::libraryPaths();  // 查看插件搜索路径

// 解决：
// 1. 确认 Qt 安装时编译了对应驱动
// 2. 确认 MySQL 客户端库（libmysql.dll / libmysqlclient.so）在 PATH 中
// 3. Windows 下将 libmysql.dll 复制到可执行文件目录
```

**问题 2：`removeDatabase` 警告 "connection is still in use"**

```cpp
// 原因：调用 removeDatabase 时仍有 QSqlDatabase 变量引用该连接
// 解决：将 QSqlDatabase 变量限制在内部作用域（见 §4.2）
```

**问题 3：子线程中查询报错或崩溃**

```cpp
// 原因：跨线程使用了主线程创建的连接
// 解决：每个线程创建自己的连接（见 §7.2）
// 使用连接池统一管理（见 §7.3）
```

**问题 4：SQLite 报 "database is locked"**

```cpp
// 原因：多个连接/进程同时写入
// 解决：
// 1. 使用 WAL 模式: PRAGMA journal_mode=WAL
// 2. 设置忙等超时: PRAGMA busy_timeout=5000 或 QSQLITE_BUSY_TIMEOUT=5000
// 3. 写操作使用事务，减少锁持有时间
// 4. 单写者模式：所有写操作集中在一个线程
```

**问题 5：中文乱码**

```cpp
// MySQL：连接后执行
QSqlQuery(db).exec("SET NAMES 'utf8mb4'");

// 或在 setConnectOptions 中设置
db.setConnectOptions("MYSQL_SET_CHARSET_NAME=utf8mb4");

// PostgreSQL：通常默认 UTF-8
// SQLite：原生支持 UTF-8
```

**问题 6：重复调用 `addDatabase` 同一连接名**

```cpp
// 先检查再创建
QSqlDatabase db;
if (QSqlDatabase::contains("myconn")) {
    db = QSqlDatabase::database("myconn");
} else {
    db = QSqlDatabase::addDatabase("QSQLITE", "myconn");
    db.setDatabaseName("data.db");
}
if (!db.isOpen()) db.open();
```

## 14. 最佳实践

| 实践 | 说明 |
|---|---|
| 每线程一个连接 | 绝不跨线程共享 `QSqlDatabase`，使用连接池管理 |
| 正确移除连接 | `QSqlDatabase` 变量离开作用域后再调用 `removeDatabase()` |
| 先判断驱动再连接 | `QSqlDatabase::isDriverAvailable("QMYSQL")` |
| 用 `open()` 返回值判断 | 不要假设连接一定成功 |
| SQLite 启用 WAL | 多连接/多线程场景下显著提升并发性能 |
| SQLite 启用外键 | `PRAGMA foreign_keys=ON`（默认关闭） |
| 网络数据库设置超时 | 避免 UI 阻塞在连接尝试上 |
| 用事务包裹批量操作 | 性能提升数十倍（SQLite 尤其明显） |
| RAII 管理事务 | `TransactionGuard` 模式防止忘记回滚 |
| 数据库迁移版本化 | 用版本号管理 schema 变更，支持增量升级 |
| 敏感信息不硬编码 | 密码从配置文件/环境变量读取，不写在源码中 |
