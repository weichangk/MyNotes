# 画笔与画刷

## 概述

`QPen`（画笔）和 `QBrush`（画刷）是 `QPainter` 绑定系统中最核心的两个样式对象：

- **QPen** — 控制**轮廓/描边**：线条颜色、宽度、样式（实线/虚线）、端点形状、连接方式
- **QBrush** — 控制**填充**：纯色、图案、渐变、纹理

```
QPen 控制的区域              QBrush 控制的区域
┌────────────────┐          ┌────────────────┐
│ ██████████████ │          │                │
│ █            █ │          │ ░░░░░░░░░░░░░░ │
│ █            █ │ ← 边框   │ ░░░░░░░░░░░░░░ │ ← 内部填充
│ █            █ │          │ ░░░░░░░░░░░░░░ │
│ ██████████████ │          │                │
└────────────────┘          └────────────────┘
```

它们共同依赖 `QColor` 提供颜色支持。本文将由浅入深地讲解这三者的完整用法。

## 1. QColor —— 颜色基础

### 1.1 构造方式

```cpp
// RGB（0~255）
QColor red(255, 0, 0);
QColor semiTransparent(100, 150, 200, 128); // 带 Alpha

// 十六进制字符串
QColor hex("#FF6600");
QColor hexAlpha("#80FF6600"); // #AARRGGBB

// 预定义颜色
QColor blue = Qt::blue;
QColor dark = Qt::darkGray;

// 颜色名
QColor named("steelblue");
QColor namedCss("rgba(255, 100, 50, 0.5)");

// HSV（色相 0~359、饱和度 0~255、明度 0~255）
QColor hsvColor = QColor::fromHsv(210, 200, 240);

// HSL（色相 0~359、饱和度 0~255、亮度 0~255）
QColor hslColor = QColor::fromHsl(210, 200, 180);

// CMYK
QColor cmykColor = QColor::fromCmyk(200, 100, 0, 50);

// 浮点 RGB（0.0~1.0）
QColor floatColor = QColor::fromRgbF(0.3, 0.6, 0.9, 1.0);
```

### 1.2 颜色分量操作

```cpp
QColor c(100, 150, 200, 255);

// 读取
int r = c.red();        // 100
int g = c.green();      // 150
int b = c.blue();       // 200
int a = c.alpha();      // 255
qreal rf = c.redF();    // ≈0.392

// 修改
c.setRed(200);
c.setAlpha(128);        // 设为半透明
c.setRgb(255, 128, 0);
c.setAlphaF(0.5);       // 浮点设置

// HSV 颜色空间转换
int h, s, v;
c.getHsv(&h, &s, &v);
```

### 1.3 颜色变换

```cpp
QColor base(60, 150, 255);

// 变亮/变暗（参数 >100 变亮，<100 变暗）
QColor lighter = base.lighter(150); // 亮 50%
QColor darker  = base.darker(200);  // 暗 100%

// 使用 HSV 微调
QColor adjusted = QColor::fromHsv(
    base.hsvHue(),
    base.hsvSaturation() * 0.5,    // 降低饱和度
    base.value()
);
```

### 1.4 颜色插值

```cpp
// 在两种颜色之间线性插值
QColor colorA(255, 0, 0);    // 红
QColor colorB(0, 0, 255);    // 蓝

// 手动插值（t = 0.0~1.0）
auto lerp = [](const QColor &a, const QColor &b, qreal t) -> QColor {
    return QColor(
        a.red()   + (b.red()   - a.red())   * t,
        a.green() + (b.green() - a.green()) * t,
        a.blue()  + (b.blue()  - a.blue())  * t,
        a.alpha() + (b.alpha() - a.alpha()) * t
    );
};

QColor mid = lerp(colorA, colorB, 0.5); // 紫色
```

## 2. QPen —— 画笔详解

### 2.1 构造方式

```cpp
// 默认画笔（黑色、宽度 1、实线）
QPen pen;

// 快速构造
QPen pen1(Qt::red);                           // 红色实线
QPen pen2(QColor(50, 100, 200));              // 自定义颜色
QPen pen3(QBrush(Qt::green), 2);              // 画刷填充 + 宽度
QPen pen4(Qt::blue, 3, Qt::DashLine);         // 颜色 + 宽度 + 线型
QPen pen5(Qt::red, 2, Qt::SolidLine, Qt::RoundCap, Qt::RoundJoin);
//         颜色    宽度    线型         端点样式       连接样式
```

### 2.2 线宽（Width）

```cpp
QPen pen;
pen.setWidth(3);       // 整数像素宽度
pen.setWidthF(2.5);    // 浮点宽度（开启抗锯齿时更平滑）

// ⚠️ 宽度 0 的特殊含义
pen.setWidth(0);
// "cosmetic pen"——始终为 1 像素宽，不受坐标变换（缩放）影响
// 常用于辅助线、网格线等需要固定视觉宽度的场景
```

**Cosmetic Pen vs Non-cosmetic Pen**:

```cpp
QPainter painter(this);

// cosmetic（宽度 0）：缩放不影响线宽
QPen cosmeticPen(Qt::black, 0);
painter.setPen(cosmeticPen);
painter.scale(3, 3);
painter.drawLine(0, 0, 50, 50); // 线宽仍为 1 像素

// non-cosmetic（宽度 > 0）：缩放同比例放大线宽
QPen normalPen(Qt::black, 2);
painter.setPen(normalPen);
painter.scale(3, 3);
painter.drawLine(0, 0, 50, 50); // 线宽变为 6 像素

// 显式设置
pen.setCosmetic(true);   // 强制为 cosmetic
pen.setCosmetic(false);  // 强制为 non-cosmetic
```

### 2.3 线型（Style）

```cpp
QPainter painter(this);
painter.setRenderHint(QPainter::Antialiasing);

struct StyleInfo {
    Qt::PenStyle style;
    QString name;
};

QVector<StyleInfo> styles = {
    {Qt::SolidLine,      "SolidLine      ───────────────"},
    {Qt::DashLine,       "DashLine       ── ── ── ── ──"},
    {Qt::DotLine,        "DotLine        ·  ·  ·  ·  · "},
    {Qt::DashDotLine,    "DashDotLine    ── · ── · ── ·"},
    {Qt::DashDotDotLine, "DashDotDotLine ── ·· ── ·· ──"},
};

int y = 30;
for (const auto &info : styles) {
    QPen pen(Qt::darkBlue, 3, info.style);
    painter.setPen(pen);
    painter.drawLine(200, y, 550, y);

    painter.setPen(Qt::black);
    painter.drawText(10, y + 5, info.name);
    y += 35;
}
```

### 2.4 自定义虚线模式

```cpp
QPen pen(Qt::black, 3);
pen.setStyle(Qt::CustomDashLine);

// 模式数组：{画线长度, 空白长度, 画线长度, 空白长度, ...}
// 单位是画笔宽度的倍数
pen.setDashPattern({10, 5, 2, 5}); // 长线-空-短线-空 循环

// 设置虚线偏移量（用于动画效果）
pen.setDashOffset(0);  // 从模式的起始位置开始

painter.setPen(pen);
painter.drawLine(20, 50, 400, 50);
```

**蚂蚁线动画（选区虚线流动效果）**：

```cpp
class MarchingAnts : public QWidget
{
    Q_OBJECT
public:
    MarchingAnts(QWidget *parent = nullptr) : QWidget(parent) {
        auto *timer = new QTimer(this);
        connect(timer, &QTimer::timeout, this, [this]() {
            m_offset += 0.5;
            if (m_offset > 12) m_offset = 0;
            update();
        });
        timer->start(50);
    }

protected:
    void paintEvent(QPaintEvent *) override {
        QPainter painter(this);

        QPen pen(Qt::black, 1);
        pen.setStyle(Qt::CustomDashLine);
        pen.setDashPattern({6, 6});
        pen.setDashOffset(m_offset);

        painter.setPen(pen);
        painter.setBrush(Qt::NoBrush);
        painter.drawRect(50, 50, 200, 150);
    }

private:
    qreal m_offset = 0;
};
```

### 2.5 端点样式（Cap Style）

端点样式决定线段两端的外观：

```cpp
QPainter painter(this);
painter.setRenderHint(QPainter::Antialiasing);

Qt::PenCapStyle caps[] = {Qt::FlatCap, Qt::SquareCap, Qt::RoundCap};
QString names[] = {"FlatCap", "SquareCap", "RoundCap"};

int y = 40;
for (int i = 0; i < 3; ++i) {
    // 粗线展示端点效果
    QPen pen(Qt::darkBlue, 16, Qt::SolidLine, caps[i]);
    painter.setPen(pen);
    painter.drawLine(150, y, 350, y);

    // 细红线标注实际线段范围
    painter.setPen(QPen(Qt::red, 1));
    painter.drawLine(150, y - 20, 150, y + 20);
    painter.drawLine(350, y - 20, 350, y + 20);

    // 标签
    painter.setPen(Qt::black);
    painter.drawText(10, y + 5, names[i]);

    y += 50;
}
```

图示对比：

```
                   线段实际范围
                   ├──────────┤

FlatCap:         │ ██████████ │      端点齐平，不超出
SquareCap:     ██│ ██████████ │██    两端各延伸半个线宽
RoundCap:      ◖│ ██████████ │◗    两端各加半圆
```

- **FlatCap**（默认）：端点在端点坐标处截断
- **SquareCap**：端点延伸 `width/2` 的正方形
- **RoundCap**：端点延伸 `width/2` 的半圆

### 2.6 连接样式（Join Style）

连接样式决定两条线段交汇处的外观：

```cpp
QPainter painter(this);
painter.setRenderHint(QPainter::Antialiasing);

Qt::PenJoinStyle joins[] = {Qt::BevelJoin, Qt::MiterJoin, Qt::RoundJoin};
QString names[] = {"BevelJoin", "MiterJoin", "RoundJoin"};

int xOffset = 0;
for (int i = 0; i < 3; ++i) {
    QPen pen(Qt::darkBlue, 16, Qt::SolidLine, Qt::FlatCap, joins[i]);
    painter.setPen(pen);
    painter.setBrush(Qt::NoBrush);

    // 绘制 V 形折线展示接合效果
    QPolygon points;
    points << QPoint(30 + xOffset, 100)
           << QPoint(80 + xOffset, 30)
           << QPoint(130 + xOffset, 100);
    painter.drawPolyline(points);

    painter.setPen(Qt::black);
    painter.setFont(QFont("Arial", 9));
    painter.drawText(30 + xOffset, 120, names[i]);

    xOffset += 160;
}
```

图示对比：

```
BevelJoin          MiterJoin          RoundJoin
    ╱╲                ╱╲                ╱╲
   ╱──╲              ╱  ╲              ╱  ╲
  ╱    ╲            ╱    ╲            ╱    ╲
 斜切接合            尖角接合           圆弧接合
```

### 2.7 Miter Limit

当两条线以很小的角度相交时，`MiterJoin` 的尖角会变得极长。`miterLimit` 限制尖角长度：

```cpp
QPen pen(Qt::black, 10);
pen.setJoinStyle(Qt::MiterJoin);
pen.setMiterLimit(2.0); // 尖角最大长度为线宽的 2 倍（默认也是 2.0）
// 超过限制时自动退化为 BevelJoin
```

### 2.8 用 QBrush 作为画笔颜色

画笔不仅支持纯色，还可以使用画刷（渐变、纹理）作为描边颜色：

```cpp
// 渐变描边
QLinearGradient grad(0, 0, 400, 0);
grad.setColorAt(0, Qt::red);
grad.setColorAt(1, Qt::blue);

QPen gradPen(QBrush(grad), 5);
painter.setPen(gradPen);
painter.drawLine(20, 50, 400, 50);

// 纹理描边
QPixmap texture(":/pattern.png");
QPen texPen(QBrush(texture), 8);
painter.setPen(texPen);
painter.drawEllipse(50, 80, 300, 200);
```

## 3. QBrush —— 画刷详解

### 3.1 构造方式

```cpp
// 无填充
QBrush noBrush;                              // 默认 Qt::NoBrush
QBrush noBrush2(Qt::NoBrush);

// 纯色填充
QBrush solidBrush(Qt::blue);
QBrush solidBrush2(QColor(100, 200, 100));
QBrush solidBrush3(Qt::green, Qt::SolidPattern);

// 图案填充
QBrush patternBrush(Qt::red, Qt::CrossPattern);

// 渐变填充
QLinearGradient grad(0, 0, 100, 0);
grad.setColorAt(0, Qt::white);
grad.setColorAt(1, Qt::black);
QBrush gradBrush(grad);

// 纹理填充
QBrush textureBrush(QPixmap("tile.png"));
QBrush textureBrush2(QImage("tile.png"));
```

### 3.2 图案样式大全

```cpp
QPainter painter(this);
painter.setRenderHint(QPainter::Antialiasing);

struct PatternInfo {
    Qt::BrushStyle style;
    QString name;
};

QVector<PatternInfo> patterns = {
    {Qt::SolidPattern,          "Solid"},
    {Qt::Dense1Pattern,         "Dense1"},
    {Qt::Dense2Pattern,         "Dense2"},
    {Qt::Dense3Pattern,         "Dense3"},
    {Qt::Dense4Pattern,         "Dense4"},
    {Qt::Dense5Pattern,         "Dense5"},
    {Qt::Dense6Pattern,         "Dense6"},
    {Qt::Dense7Pattern,         "Dense7"},
    {Qt::HorPattern,            "Horizontal"},
    {Qt::VerPattern,            "Vertical"},
    {Qt::CrossPattern,          "Cross"},
    {Qt::BDiagPattern,          "BDiag ╲"},
    {Qt::FDiagPattern,          "FDiag ╱"},
    {Qt::DiagCrossPattern,      "DiagCross ╳"},
};

int col = 0, row = 0;
for (const auto &info : patterns) {
    int x = 20 + col * 120;
    int y = 20 + row * 90;

    painter.setPen(QPen(Qt::gray, 1));
    painter.setBrush(QBrush(Qt::darkBlue, info.style));
    painter.drawRect(x, y, 80, 50);

    painter.setPen(Qt::black);
    painter.setFont(QFont("Arial", 8));
    painter.drawText(x, y + 65, info.name);

    col++;
    if (col >= 5) { col = 0; row++; }
}
```

## 4. 渐变详解

Qt 支持三种渐变类型，都继承自 `QGradient`。

### 4.1 线性渐变（QLinearGradient）

沿一条直线方向过渡颜色：

```cpp
// 从 (x1,y1) 到 (x2,y2) 的方向
QLinearGradient grad(0, 0, 300, 0); // 水平从左到右

// 添加颜色停止点（位置 0.0~1.0）
grad.setColorAt(0.0, QColor(255, 100, 100));  // 起点：浅红
grad.setColorAt(0.3, QColor(255, 255, 100));  // 30%：黄
grad.setColorAt(0.7, QColor(100, 255, 100));  // 70%：绿
grad.setColorAt(1.0, QColor(100, 100, 255));  // 终点：蓝

painter.setBrush(grad);
painter.setPen(Qt::NoPen);
painter.drawRect(20, 20, 300, 80);
```

**不同方向**：

```cpp
// 水平
QLinearGradient h(0, 0, width(), 0);

// 垂直
QLinearGradient v(0, 0, 0, height());

// 对角线
QLinearGradient d(0, 0, width(), height());

// 任意角度（通过起终点坐标控制）
double angle = 45 * M_PI / 180;
double len = 300;
QLinearGradient angled(0, 0, len * cos(angle), len * sin(angle));
```

### 4.2 径向渐变（QRadialGradient）

从一个焦点向外扩散的圆形渐变：

```cpp
// 基本用法：center + radius
QRadialGradient grad(QPointF(150, 150), 100);
grad.setColorAt(0, Qt::white);        // 中心
grad.setColorAt(0.5, Qt::cyan);       // 中间
grad.setColorAt(1, Qt::darkBlue);     // 边缘

painter.setBrush(grad);
painter.drawEllipse(50, 50, 200, 200);
```

```cpp
// 高级：焦点偏移（模拟球体光照）
QRadialGradient sphere(QPointF(150, 150), 100, QPointF(120, 120));
//                     中心               半径    焦点（高光位置）
sphere.setColorAt(0, QColor(255, 255, 255));  // 高光
sphere.setColorAt(0.3, QColor(200, 50, 50));  // 表面色
sphere.setColorAt(1, QColor(80, 0, 0));       // 边缘暗色

painter.setPen(Qt::NoPen);
painter.setBrush(sphere);
painter.drawEllipse(50, 50, 200, 200);
// 效果：一个有光泽的红色球体
```

```cpp
// 焦点半径（Qt 5.12+）
QRadialGradient grad(center, outerRadius, focalPoint, focalRadius);
// focalRadius > 0 时形成环形渐变
```

### 4.3 锥形渐变（QConicalGradient）

围绕一个中心点按角转变颜色：

```cpp
// center + startAngle（0° = 3点钟方向，逆时针递增）
QConicalGradient grad(QPointF(150, 150), 0);
grad.setColorAt(0.0,  Qt::red);
grad.setColorAt(0.17, Qt::yellow);
grad.setColorAt(0.33, Qt::green);
grad.setColorAt(0.5,  Qt::cyan);
grad.setColorAt(0.67, Qt::blue);
grad.setColorAt(0.83, Qt::magenta);
grad.setColorAt(1.0,  Qt::red);     // 首尾相同实现平滑过渡

painter.setPen(Qt::NoPen);
painter.setBrush(grad);
painter.drawEllipse(50, 50, 200, 200);
// 效果：色轮
```

### 4.4 扩展模式（Spread）

当绘制区域超出渐变定义的范围时，扩展模式决定如何填充：

```cpp
QLinearGradient grad(100, 0, 200, 0); // 只在 x=100~200 范围定义
grad.setColorAt(0, Qt::red);
grad.setColorAt(1, Qt::blue);

// 三种扩展模式
grad.setSpread(QGradient::PadSpread);     // 用最近的端点颜色填充（默认）
grad.setSpread(QGradient::RepeatSpread);  // 重复渐变
grad.setSpread(QGradient::ReflectSpread); // 镜像重复
```

图示对比（渐变定义区域在中间）：

```
PadSpread:     红红红红|红───────蓝|蓝蓝蓝蓝
RepeatSpread:  红──蓝|红───────蓝|红──蓝
ReflectSpread: 蓝──红|红───────蓝|蓝──红
```

> **注意**：`QConicalGradient` 不支持 Spread 模式（它天然是 360° 环绕的）。

### 4.5 坐标模式

```cpp
QLinearGradient grad;

// 逻辑坐标模式（默认）：坐标是绝对像素坐标
grad.setCoordinateMode(QGradient::LogicalMode);

// 对象边界模式：坐标为 0.0~1.0 相对于绘制对象的边界框
grad.setCoordinateMode(QGradient::ObjectBoundingMode);
// 此模式下 (0,0) 是左上角，(1,1) 是右下角

// 对象模式（Qt 5.12+）：类似 ObjectBoundingMode 但更精确
grad.setCoordinateMode(QGradient::ObjectMode);
```

```cpp
// 使用 ObjectBoundingMode 让渐变自动适配不同大小的形状
QLinearGradient grad(0, 0, 1, 0); // 从左到右
grad.setCoordinateMode(QGradient::ObjectBoundingMode);
grad.setColorAt(0, Qt::red);
grad.setColorAt(1, Qt::blue);

painter.setBrush(grad);
painter.drawRect(20, 20, 100, 50);   // 小矩形：渐变自适应

```

        }
    }

### 5.3 仪表盘刻度

```cpp
void drawGauge(QPainter &painter, const QRect &area, double value)
{
    painter.setRenderHint(QPainter::Antialiasing);
    painter.save();

    QPointF center = QRectF(area).center();
    double radius = qMin(area.width(), area.height()) / 2.0 - 10;

    painter.translate(center);

    // 背景圆
    QRadialGradient bgGrad(0, 0, radius);
    bgGrad.setColorAt(0, QColor(50, 50, 60));
    bgGrad.setColorAt(1, QColor(30, 30, 35));
    painter.setPen(QPen(QColor(80, 80, 90), 2));
    painter.setBrush(bgGrad);
    painter.drawEllipse(QPointF(0, 0), radius, radius);

    // 刻度
    for (int i = 0; i <= 50; ++i) {
        double angle = 225 - i * 270.0 / 50; // 从左下到右下，270°范围
        double rad = angle * M_PI / 180.0;

        bool major = (i % 10 == 0);
        double innerR = radius * (major ? 0.78 : 0.85);
        double outerR = radius * 0.92;

        QPen tickPen(major ? Qt::white : QColor(150, 150, 160),
                     major ? 2 : 1);
        painter.setPen(tickPen);
        painter.drawLine(QPointF(innerR * cos(rad), -innerR * sin(rad)),
                         QPointF(outerR * cos(rad), -outerR * sin(rad)));

        // 数字标签
        if (major) {
            painter.setFont(QFont("Arial", 10, QFont::Bold));
            painter.setPen(Qt::white);
            double labelR = radius * 0.68;
            QString label = QString::number(i * 2);
            QRectF textRect(-15, -8, 30, 16);
            painter.save();
            painter.translate(labelR * cos(rad), -labelR * sin(rad));
            painter.drawText(textRect, Qt::AlignCenter, label);
            painter.restore();
        }
    }

    // 指针
    double angle = (225 - value / 100.0 * 270) * M_PI / 180.0;
    double needleLen = radius * 0.72;

    QPen needlePen(QColor(255, 60, 60), 3, Qt::SolidLine, Qt::RoundCap);
    painter.setPen(needlePen);
    painter.drawLine(QPointF(0, 0),
                     QPointF(needleLen * cos(angle), -needleLen * sin(angle)));

    // 中心圆点
    QRadialGradient centerGrad(0, 0, 8);
    centerGrad.setColorAt(0, QColor(220, 220, 230));
    centerGrad.setColorAt(1, QColor(100, 100, 110));
    painter.setPen(Qt::NoPen);
    painter.setBrush(centerGrad);
    painter.drawEllipse(QPointF(0, 0), 8, 8);

    painter.restore();
}
```

## 6. QPen 与 QBrush 的关系

### 6.1 组合使用

```cpp
QPainter painter(this);

// 画笔 + 画刷 = 有边框 + 有填充
painter.setPen(QPen(Qt::black, 2));
painter.setBrush(QColor(200, 220, 255));
painter.drawRect(20, 20, 120, 80);

// 只画笔 = 只有边框
painter.setPen(QPen(Qt::red, 2));
painter.setBrush(Qt::NoBrush);
painter.drawRect(160, 20, 120, 80);

// 只画刷 = 只有填充
painter.setPen(Qt::NoPen);
painter.setBrush(Qt::green);
painter.drawRect(300, 20, 120, 80);
```

### 6.2 绘制时的顺序

```
1. 用 QBrush 填充图形内部
2. 用 QPen 描绘图形轮廓（边框覆盖在填充之上）
```

> **注意**：当画笔宽度较大时，轮廓会向图形**内外双向扩展**（各 width/2），这可能导致图形看起来比指定尺寸略大。

```cpp
// 画笔宽度对图形大小的影响
QPen pen(Qt::black, 10);
painter.setPen(pen);
painter.drawRect(50, 50, 100, 100);
// 实际绘制的矩形外边缘为 (45,45)-(155,155)，比指定的多出 5 像素

// 如需精确控制，用 adjusted 补偿
int halfPen = pen.width() / 2;
painter.drawRect(QRect(50, 50, 100, 100).adjusted(halfPen, halfPen, -halfPen, -halfPen));
```

## 7. 全局透明度

```cpp
QPainter painter(this);

painter.setOpacity(1.0);   // 完全不透明（默认）
painter.setBrush(Qt::red);
painter.drawRect(20, 20, 150, 100);

painter.setOpacity(0.5);   // 50% 透明
painter.setBrush(Qt::blue);
painter.drawRect(80, 60, 150, 100); // 蓝色半透明矩形叠加在红色上

painter.setOpacity(0.2);   // 20% 透明
painter.setBrush(Qt::green);
painter.drawRect(140, 100, 150, 100);
```

> `setOpacity()` 是**全局乘法因子**，它会与颜色自身的 Alpha 通道相乘。例如 `opacity=0.5` + `color.alpha()=128` → 实际 Alpha = `0.5 × 128/255 ≈ 0.25`。

## 9. 总结

| 要点 | 说明 |
|---|---|
| **QPen 管轮廓** | 颜色、宽度、线型、端点、连接、Miter 限制 |
| **QBrush 管填充** | 纯色、图案（14种）、渐变（3种）、纹理 |
| **QColor** | 支持 RGB/HSV/HSL/CMYK 多色彩空间，Alpha 通道控制透明度 |
| **渐变** | Linear（线性）、Radial（径向）、Conical（锥形），支持多停止点 |
| **Spread 模式** | Pad（填充）、Repeat（重复）、Reflect（镜像）控制渐变越界行为 |
| **Cosmetic Pen** | 宽度 0 的线条不受坐标缩放影响，始终 1 像素 |
| **自定义虚线** | `setDashPattern()` + `setDashOffset()` 实现任意虚线和动画 |
| **全局透明度** | `setOpacity()` 是乘法因子，与颜色 Alpha 叠加 |
| **绘制顺序** | 先填充（Brush）后描边（Pen），边框向内外各扩展 width/2 |
