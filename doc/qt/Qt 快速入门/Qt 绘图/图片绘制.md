# 图片绘制

## 概述

Qt 提供三种核心图像类用于绘制：**QPixmap**（屏幕显示优化）、**QImage**（像素级操作优化）、**QBitmap**（单色位图）。它们都是 `QPaintDevice` 的子类，既可以作为绑定目标（往图片上画），也可以被 `QPainter` 绘制到其他设备上。

```
QPaintDevice
  ├─ QWidget         ← 直接绘制到控件
  ├─ QPixmap         ← 屏幕显示优化，GPU 友好
  │    └─ QBitmap    ← 1-bit 单色位图（QPixmap 子类）
  ├─ QImage          ← CPU 像素级操作，跨平台一致
  └─ QPicture        ← 记录/回放 QPainter 指令（矢量）
```

## 1. QPixmap —— 屏幕显示首选

### 1.1 特点

| 特性 | 说明 |
|---|---|
| 存储位置 | 服务端（GPU/显存），平台优化 |
| 像素访问 | **不支持**直接逐像素操作 |
| 线程安全 | ❌ 只能在 GUI 线程使用 |
| 适用场景 | 屏幕绘制、控件背景、图标缓存 |
| 隐式共享 | ✅ 赋值/传参零拷贝 |

### 1.2 创建方式

```cpp
// 从文件加载
QPixmap pix(":/images/photo.png");
QPixmap pix2("C:/photos/test.jpg");

// 指定大小创建空白图
QPixmap blank(400, 300);
blank.fill(Qt::transparent);    // 透明背景
blank.fill(Qt::white);          // 白色背景

// 从资源系统加载
QPixmap icon(":/icons/save.svg");

// 延迟加载（仅在需要时才真正解码）
QPixmap lazy;
lazy.load("big_photo.jpg");

// 从 QImage 转换
QImage img("photo.png");
QPixmap fromImg = QPixmap::fromImage(img);
```

### 1.3 绘制 QPixmap

```cpp
void MyWidget::paintEvent(QPaintEvent *) {
    QPainter painter(this);
    QPixmap pix(":/images/landscape.jpg");

    // 方式 1：指定左上角位置
    painter.drawPixmap(10, 10, pix);

    // 方式 2：指定目标矩形（自动缩放）
    painter.drawPixmap(QRect(10, 10, 200, 150), pix);

    // 方式 3：指定目标矩形 + 源矩形（裁剪绘制）
    painter.drawPixmap(
        QRect(10, 10, 200, 150),    // 目标区域
        pix,
        QRect(100, 50, 400, 300)    // 从源图中裁剪的区域
    );

    // 方式 4：平铺绘制
    painter.drawTiledPixmap(rect(), pix, QPoint(0, 0));
}
```

### 1.4 在 QPixmap 上绘图（离屏绑定）

```cpp
QPixmap canvas(400, 300);
canvas.fill(Qt::white);

QPainter painter(&canvas);
painter.setRenderHint(QPainter::Antialiasing);

// 绘制圆角矩形
painter.setBrush(QColor(60, 150, 255));
painter.setPen(QPen(Qt::white, 2));
painter.drawRoundedRect(20, 20, 360, 260, 15, 15);

// 绘制文本
painter.setPen(Qt::white);
painter.setFont(QFont("Microsoft YaHei", 24, QFont::Bold));
painter.drawText(canvas.rect(), Qt::AlignCenter, "Hello Qt");
painter.end();

// 保存到文件
canvas.save("output.png");

// 用作控件背景
label->setPixmap(canvas);
```

### 1.5 缩放

```cpp
QPixmap original("photo.jpg");

// 等比缩放到指定大小内
QPixmap scaled = original.scaled(200, 150, Qt::KeepAspectRatio,
                                 Qt::SmoothTransformation);

// 拉伸到精确大小
QPixmap stretched = original.scaled(200, 150, Qt::IgnoreAspectRatio,
                                    Qt::SmoothTransformation);

// 等比缩放（按宽度）
QPixmap byWidth = original.scaledToWidth(300, Qt::SmoothTransformation);

// 等比缩放（按高度）
QPixmap byHeight = original.scaledToHeight(200, Qt::SmoothTransformation);
```

**缩放模式对比**：

| `Qt::TransformationMode` | 效果 | 速度 |
|---|---|---|
| `Qt::FastTransformation` | 最近邻插值，有锯齿 | 快 |
| `Qt::SmoothTransformation` | 双线性插值，平滑 | 慢 |

### 1.6 常用操作

```cpp
QPixmap pix("photo.png");

// 尺寸查询
int w = pix.width();
int h = pix.height();
QSize sz = pix.size();
QRect r = pix.rect();           // QRect(0, 0, w, h)
bool empty = pix.isNull();      // 是否为空
int depth = pix.depth();        // 位深（通常 32）
bool alpha = pix.hasAlpha();    // 是否有透明通道

// 裁剪（返回新 QPixmap）
QPixmap cropped = pix.copy(QRect(50, 50, 200, 150));

// 创建蒙版
QBitmap mask = pix.createMaskFromColor(Qt::white, Qt::MaskInColor);
pix.setMask(mask);               // 白色区域变透明

// QPixmap ↔ QImage 互转
QImage img = pix.toImage();
QPixmap pix2 = QPixmap::fromImage(img);
```

## 2. QImage —— 像素级操作首选

### 2.1 特点

| 特性 | 说明 |
|---|---|
| 存储位置 | 客户端（CPU 内存） |
| 像素访问 | ✅ `pixel()` / `setPixel()` / `scanLine()` / `bits()` |
| 线程安全 | ✅ 可在非 GUI 线程中操作 |
| 适用场景 | 图像处理、像素操作、格式转换、文件 I/O |
| 隐式共享 | ✅ 写时复制（COW） |

### 2.2 像素格式

```cpp
// 常用格式
QImage img1(400, 300, QImage::Format_ARGB32);           // 带 Alpha，32 位
QImage img2(400, 300, QImage::Format_ARGB32_Premultiplied); // 预乘 Alpha（绘制更快）
QImage img3(400, 300, QImage::Format_RGB32);             // 无 Alpha，32 位
QImage img4(400, 300, QImage::Format_Grayscale8);        // 灰度，8 位
QImage img5(400, 300, QImage::Format_RGB888);            // 24 位 RGB（无 Alpha，省内存）
```

| 格式 | 每像素位数 | Alpha | 说明 |
|---|---|---|---|
| `Format_ARGB32` | 32 | ✅ | 通用格式 |
| `Format_ARGB32_Premultiplied` | 32 | ✅ | QPainter 绑定最快 |
| `Format_RGB32` | 32 | ❌ | 不透明图片 |
| `Format_RGB888` | 24 | ❌ | 省内存 |
| `Format_Grayscale8` | 8 | ❌ | 灰度图 |
| `Format_Indexed8` | 8 | 可选 | 调色板模式 |
| `Format_Mono` | 1 | ❌ | 黑白 |

> **性能提示**：用 `Format_ARGB32_Premultiplied` 做离屏绘制目标，QPainter 的混合运算最快。

### 2.3 像素访问

```cpp
QImage img(256, 256, QImage::Format_ARGB32);

// ──── 方式 1：pixel() / setPixel()（慢，但简单） ────
for (int y = 0; y < img.height(); ++y) {
    for (int x = 0; x < img.width(); ++x) {
        img.setPixel(x, y, qRgba(x, y, 128, 255));
    }
}
QRgb color = img.pixel(100, 50);  // 读取像素
int r = qRed(color);
int g = qGreen(color);
int b = qBlue(color);
int a = qAlpha(color);

// ──── 方式 2：scanLine()（快，推荐） ────
for (int y = 0; y < img.height(); ++y) {
    QRgb *line = reinterpret_cast<QRgb *>(img.scanLine(y));
    for (int x = 0; x < img.width(); ++x) {
        line[x] = qRgba(x, y, 128, 255);
    }
}

// ──── 方式 3：bits()（最快，一维数组访问） ────
uchar *data = img.bits();
int bytesPerLine = img.bytesPerLine();
for (int y = 0; y < img.height(); ++y) {
    QRgb *line = reinterpret_cast<QRgb *>(data + y * bytesPerLine);
    for (int x = 0; x < img.width(); ++x) {
        line[x] = qRgba(x, y, 128, 255);
    }
}

// ──── 方式 4：pixelColor() / setPixelColor()（Qt 5.6+，QColor 版本） ────
QColor c = img.pixelColor(100, 50);
img.setPixelColor(100, 50, QColor(255, 0, 0, 200));
```

**性能对比**：

```
pixel()/setPixel()        ████░░░░░░  慢（每次调用有边界检查+格式转换）
scanLine()                ████████░░  快（每行一次指针获取）
bits()                    ██████████  最快（一次获取，直接内存偏移）
pixelColor()/setPixelColor() ███░░░░░░░  最慢（QColor 构造开销）
```

### 2.4 图像处理示例

#### 灰度化

```cpp
QImage toGrayscale(const QImage &src) {
    QImage result = src.convertToFormat(QImage::Format_ARGB32);
    for (int y = 0; y < result.height(); ++y) {
        QRgb *line = reinterpret_cast<QRgb *>(result.scanLine(y));
        for (int x = 0; x < result.width(); ++x) {
            int gray = qGray(line[x]);  // 加权灰度：0.299R + 0.587G + 0.114B
            int alpha = qAlpha(line[x]);
            line[x] = qRgba(gray, gray, gray, alpha);
        }
    }
    return result;
}
```

#### 亮度/对比度调节

```cpp
QImage adjustBrightnessContrast(const QImage &src, int brightness, double contrast) {
    // brightness: -255 ~ 255, contrast: 0.0 ~ 3.0
    QImage result = src.convertToFormat(QImage::Format_ARGB32);
    for (int y = 0; y < result.height(); ++y) {
        QRgb *line = reinterpret_cast<QRgb *>(result.scanLine(y));
        for (int x = 0; x < result.width(); ++x) {
            int r = qBound(0, int((qRed(line[x]) - 128) * contrast + 128 + brightness), 255);
            int g = qBound(0, int((qGreen(line[x]) - 128) * contrast + 128 + brightness), 255);
            int b = qBound(0, int((qBlue(line[x]) - 128) * contrast + 128 + brightness), 255);
            line[x] = qRgba(r, g, b, qAlpha(line[x]));
        }
    }
    return result;
}
```

#### 颜色反转

```cpp
QImage invertColors(const QImage &src) {
    QImage result = src.copy();
    result.invertPixels(QImage::InvertRgb); // 只反转 RGB，保留 Alpha
    return result;
}
```

#### 简单模糊（均值模糊）

```cpp
QImage boxBlur(const QImage &src, int radius) {
    QImage result = src.convertToFormat(QImage::Format_ARGB32);
    QImage temp = result.copy();
    int w = src.width(), h = src.height();
    int div = (2 * radius + 1) * (2 * radius + 1);

    for (int y = radius; y < h - radius; ++y) {
        QRgb *dstLine = reinterpret_cast<QRgb *>(result.scanLine(y));
        for (int x = radius; x < w - radius; ++x) {
            int sumR = 0, sumG = 0, sumB = 0, sumA = 0;
            for (int ky = -radius; ky <= radius; ++ky) {
                const QRgb *srcLine = reinterpret_cast<const QRgb *>(temp.constScanLine(y + ky));
                for (int kx = -radius; kx <= radius; ++kx) {
                    QRgb px = srcLine[x + kx];
                    sumR += qRed(px);
                    sumG += qGreen(px);
                    sumB += qBlue(px);
                    sumA += qAlpha(px);
                }
            }
            dstLine[x] = qRgba(sumR / div, sumG / div, sumB / div, sumA / div);
        }
    }
    return result;
}
```

## 3. QBitmap —— 单色位图

`QBitmap` 是 `QPixmap` 的子类，深度固定为 1 位（黑/白），主要用于蒙版和形状定义。

```cpp
// 从图片生成蒙版
QPixmap pix("photo.png");
QBitmap mask = pix.createMaskFromColor(QColor(0, 0, 0), Qt::MaskOutColor);

// 应用蒙版（黑色区域变透明）
pix.setMask(mask);

// 创建自定义蒙版
QBitmap customMask(100, 100);
customMask.fill(Qt::color0); // 全透明
QPainter mp(&customMask);
mp.setBrush(Qt::color1);     // 不透明
mp.drawEllipse(0, 0, 100, 100); // 圆形区域可见
mp.end();
pix.setMask(customMask);

// 用于窗口形状（圆形窗口）
QBitmap windowMask(size());
windowMask.fill(Qt::color0);
QPainter p(&windowMask);
p.setBrush(Qt::color1);
p.drawEllipse(rect());
p.end();
setMask(windowMask);
```

> `Qt::color0` = 透明/0 值，`Qt::color1` = 不透明/1 值。

## 4. QPicture —— 矢量记录/回放

`QPicture` 记录 `QPainter` 的绘制指令序列，可以保存到文件并在任何设备上回放，实现分辨率无关的矢量图形。

```cpp
// 录制
QPicture picture;
QPainter recorder(&picture);
recorder.setRenderHint(QPainter::Antialiasing);
recorder.setPen(QPen(Qt::blue, 3));
recorder.setBrush(QColor(100, 200, 255, 100));
recorder.drawRoundedRect(10, 10, 180, 130, 15, 15);
recorder.drawText(50, 80, "Vector!");
recorder.end();

// 保存到文件
picture.save("drawing.pic");

// 加载并回放
QPicture loaded;
loaded.load("drawing.pic");

QPainter painter(this);
painter.drawPicture(0, 0, loaded);        // 原始位置
painter.translate(250, 0);
painter.scale(1.5, 1.5);
painter.drawPicture(0, 0, loaded);        // 放大 1.5 倍（无损）

// 获取包围盒
QRect bounds = picture.boundingRect();
```

### 5.2 QImageReader —— 精细控制加载

```cpp
QImageReader reader("photo.jpg");

// 查询信息（不加载全图）

reader.setAutoTransform(true);
QImage corrected = reader.read();

// 错误处理
if (!reader.canRead()) {
    qWarning() << "无法读取:" << reader.errorString();
}
```

```cpp
QImage img = createImage();
```

## 6. 高 DPI 图片绘制

### 6.1 基本原则

```
普通屏 (DPR=1)     高清屏 (DPR=2)        超高清 (DPR=3)
100×100 逻辑像素    100×100 逻辑像素       100×100 逻辑像素
100×100 物理像素    200×200 物理像素       300×300 物理像素

图片如果只有 100×100:   在 DPR=2 上被放大 → 模糊
图片如果有 200×200:     在 DPR=2 上原尺寸显示 → 清晰
```

### 6.2 正确创建高 DPI QPixmap

```cpp
void MyWidget::paintEvent(QPaintEvent *) {
    qreal dpr = devicePixelRatioF();

    // ✅ 正确：创建的物理像素数 = 逻辑大小 × DPR
    int logicalW = 200, logicalH = 150;
    QPixmap hires(logicalW * dpr, logicalH * dpr);
    hires.setDevicePixelRatio(dpr);
    hires.fill(Qt::transparent);

    QPainter px(&hires);
    px.setRenderHint(QPainter::Antialiasing);
    px.setBrush(Qt::blue);
    px.drawEllipse(0, 0, logicalW, logicalH); // 用逻辑坐标绑定
    px.end();

    // 绘制到 widget（显示为 200×150 逻辑像素）
    QPainter painter(this);
    painter.drawPixmap(10, 10, hires);

    // ❌ 错误：不考虑 DPR
    QPixmap lowres(200, 150);   // 只有 200×150 物理像素
    // 在 DPR=2 屏幕上，Qt 会把它拉伸到 400×300 物理像素 → 模糊
}
```

### 6.3 加载多分辨率资源

```cpp
// 方式 1：@2x 命名约定
// 资源文件中放置：
//   images/icon.png       (64x64)
//   images/icon@2x.png    (128x128)
//   images/icon@3x.png    (192x192)

QPixmap icon(":/images/icon.png");
// Qt 自动根据 DPR 选择 @2x 或 @3x 版本

// 方式 2：手动加载高清版本
qreal dpr = qApp->devicePixelRatio();
QString path = dpr >= 2 ? ":/images/icon@2x.png" : ":/images/icon.png";
QPixmap icon(path);
icon.setDevicePixelRatio(dpr);
```

### 6.4 高 DPI QImage

```cpp

```

## 7. 图片变换与合成

### 7.1 旋转

```cpp
QPixmap original("photo.jpg");

// 方式 1：QTransform
QTransform t;
t.rotate(45);
QPixmap rotated = original.transformed(t, Qt::SmoothTransformation);

// 方式 2：在绘制时旋转
QPainter painter(this);
painter.translate(200, 200);
painter.rotate(45);
painter.drawPixmap(-original.width() / 2, -original.height() / 2, original);
```

> `transformed()` 返回的 QPixmap 会自动扩大以容纳旋转后的图像，旋转后多出的部分为透明。

### 7.2 镜像翻转

```cpp
QImage img("photo.png");

// 水平镜像
QImage hFlipped = img.mirrored(true, false);

// 垂直镜像
QImage vFlipped = img.mirrored(false, true);

// 水平 + 垂直（180° 旋转）
QImage rotated180 = img.mirrored(true, true);

// QPixmap 版本
QPixmap pix("photo.png");
QTransform flipH;
flipH.scale(-1, 1);
QPixmap hFlippedPix = pix.transformed(flipH);
```

### 7.3 合成模式（图片叠加）

```cpp
QPainter painter(this);

QPixmap background("bg.png");
QPixmap overlay("overlay.png");

// 先绘制背景
painter.drawPixmap(0, 0, background);

// 设置合成模式后再绘制叠加层
painter.setCompositionMode(QPainter::CompositionMode_Multiply);
painter.drawPixmap(0, 0, overlay);

// 恢复默认合成模式
painter.setCompositionMode(QPainter::CompositionMode_SourceOver);
```

**常用合成模式**：

| 模式 | 效果 |
|---|---|
| `SourceOver` | 默认，前景覆盖背景（含 Alpha 混合） |
| `DestinationOver` | 背景在前 |
| `Source` | 完全替换 |
| `Clear` | 清除区域 |
| `Multiply` | 正片叠底（变暗） |
| `Screen` | 滤色（变亮） |
| `Overlay` | 叠加（增强对比） |
| `SoftLight` | 柔光 |
| `HardLight` | 强光 |
| `Difference` | 差值 |

### 7.4 透明度绘制

```cpp
QPainter painter(this);
QPixmap pix("photo.jpg");
```

### 8.4 图片查看器（缩放+拖拽）

```cpp
class ImageViewer : public QWidget
{
    Q_OBJECT
public:
    void setImage(const QPixmap &pixmap) {
        m_pixmap = pixmap;
        m_scale = 1.0;
        m_offset = QPointF(0, 0);
        fitInView();
        update();
    }

protected:
    void paintEvent(QPaintEvent *) override {
        QPainter painter(this);
        painter.setRenderHint(QPainter::SmoothPixmapTransform);
        painter.fillRect(rect(), QColor(30, 30, 30));

        if (m_pixmap.isNull()) return;

        painter.translate(m_offset);
        painter.scale(m_scale, m_scale);
        painter.drawPixmap(0, 0, m_pixmap);

        // 显示缩放比例
        painter.resetTransform();
        painter.setPen(QColor(200, 200, 200));
        painter.setFont(QFont("Arial", 10));
        painter.drawText(10, height() - 10,
                         QString("%1%").arg(int(m_scale * 100)));
    }

    void wheelEvent(QWheelEvent *event) override {
        QPointF mousePos = event->position();
        QPointF before = (mousePos - m_offset) / m_scale;

        double factor = event->angleDelta().y() > 0 ? 1.15 : 1.0 / 1.15;
        m_scale = qBound(0.05, m_scale * factor, 30.0);

        QPointF after = (mousePos - m_offset) / m_scale;
        m_offset += (after - before) * m_scale;
        update();
    }

    void mousePressEvent(QMouseEvent *event) override {
        if (event->button() == Qt::LeftButton) {
            m_dragging = true;
            m_lastPos = event->pos();
            setCursor(Qt::ClosedHandCursor);
        }
    }

    void mouseMoveEvent(QMouseEvent *event) override {
        if (m_dragging) {
            m_offset += event->pos() - m_lastPos;
            m_lastPos = event->pos();
            update();
        }
    }

    void mouseReleaseEvent(QMouseEvent *event) override {
        if (event->button() == Qt::LeftButton) {
            m_dragging = false;
            setCursor(Qt::ArrowCursor);
        }
    }

    void mouseDoubleClickEvent(QMouseEvent *) override {
        fitInView();
        update();
    }

private:
    void fitInView() {
        if (m_pixmap.isNull()) return;
        double sx = (double)width() / m_pixmap.width();
        double sy = (double)height() / m_pixmap.height();
        m_scale = qMin(sx, sy) * 0.95;
        m_offset.setX((width() - m_pixmap.width() * m_scale) / 2);
        m_offset.setY((height() - m_pixmap.height() * m_scale) / 2);
    }

    QPixmap m_pixmap;
    QPointF m_offset;
    double m_scale = 1.0;
    bool m_dragging = false;
    QPointF m_lastPos;
};
```

## 9. QPixmap 缓存机制

### 9.1 QPixmapCache

Qt 提供全局的 `QPixmapCache` 用于缓存 QPixmap，避免重复加载和创建：

```cpp
// 存入缓存
QPixmap pix(":/icons/large_icon.png");
QPixmapCache::insert("myIcon", pix);

// 从缓存读取
QPixmap cached;
if (QPixmapCache::find("myIcon", &cached)) {
    painter.drawPixmap(0, 0, cached);
} else {
    // 缓存未命中，重新加载
    cached.load(":/icons/large_icon.png");
    QPixmapCache::insert("myIcon", cached);
    painter.drawPixmap(0, 0, cached);
}

// 设置缓存大小（KB，默认 10240 = 10MB）
QPixmapCache::setCacheLimit(51200); // 50MB

// 清空缓存
QPixmapCache::clear();
```

### 9.2 使用 Key 的方式（更高效）

```cpp
QPixmapCache::Key cacheKey;

// 存入并获取 Key
QPixmap pix(createExpensivePixmap());
```

### 9.3 控件中的缓存模式

在自定义控件 `paintEvent` 中缓存复杂绘制结果：

```cpp
class CachedWidget : public QWidget {
    QPixmap m_cache;
    bool m_cacheDirty = true;

    void invalidateCache() {
        m_cacheDirty = true;
        update();
    }

    void paintEvent(QPaintEvent *) override {
        if (m_cacheDirty || m_cache.size() != size() * devicePixelRatioF()) {
            qreal dpr = devicePixelRatioF();
            m_cache = QPixmap(size() * dpr);
            m_cache.setDevicePixelRatio(dpr);
            m_cache.fill(Qt::transparent);

            QPainter cp(&m_cache);
            cp.setRenderHint(QPainter::Antialiasing);
            drawComplexContent(&cp); // 昂贵的绘制操作
            cp.end();

            m_cacheDirty = false;
        }

        QPainter painter(this);
        painter.drawPixmap(0, 0, m_cache);
    }

    // 只在数据变化时调用 invalidateCache()
    void setData(const Data &data) {
        m_data = data;
        invalidateCache();
    }
};
```

## 10. 性能优化

### 10.1 对比表

| 操作 | 建议 |
|---|---|
| 显示图片 | 用 `QPixmap`，不要用 `QImage` |
| 像素操作 | 用 `QImage`，处理完转 `QPixmap` 显示 |
| 离屏绑定 | `QPixmap`（GPU）或 `QImage`（Format_ARGB32_Premultiplied） |
| 非 GUI 线程处理 | 必须用 `QImage`（`QPixmap` 不可在非 GUI 线程使用） |
| 缩放 | 预缩放到目标大小缓存，不要每帧实时缩放 |
| 格式选择 | 离屏手绘 → `Format_ARGB32_Premultiplied`（混合最快） |
| 大图绘制 | 只绘制可见区域，不要绘制到屏幕外 |

### 10.2 避免常见性能陷阱

```cpp
// ❌ 每帧都加载和缩放
void paintEvent(QPaintEvent *) {
    QPainter p(this);
    QPixmap pix("big_photo.jpg");                   // 每帧磁盘 I/O
    p.drawPixmap(0, 0, pix.scaled(width(), height())); // 每帧缩放
}

// ✅ 加载一次，缓存缩放结果
class MyWidget : public QWidget {
    QPixmap m_original;
    QPixmap m_scaled;

    void setImage(const QString &path) {
        m_original.load(path);
        updateScaled();
    }

    void resizeEvent(QResizeEvent *) override {
        updateScaled();
    }

    void paintEvent(QPaintEvent *) override {
        QPainter p(this);
        if (!m_scaled.isNull())
            p.drawPixmap(0, 0, m_scaled);
    }

    void updateScaled() {
        if (!m_original.isNull())
            m_scaled = m_original.scaled(size(), Qt::KeepAspectRatio,
                                         Qt::SmoothTransformation);
        update();
    }
};
```

## 11. QPixmap vs QImage vs QPicture 对比

| 特性 | QPixmap | QImage | QPicture |
|---|---|---|---|
| 存储 | GPU/显存 | CPU 内存 | 指令序列 |
| 线程安全 | ❌ GUI 线程 | ✅ 任意线程 | ✅ 任意线程 |
| 像素访问 | ❌ | ✅ scanLine/bits | ❌ |
| 显示效率 | ⭐⭐⭐ 最快 | ⭐⭐ 需转换 | ⭐ 需重放 |
| 缩放无损 | ❌ 位图 | ❌ 位图 | ✅ 矢量 |
| Alpha 支持 | ✅ | ✅ | ✅ |
| 隐式共享 | ✅ | ✅（COW） | ✅ |
| 文件 I/O | load/save | load/save | load/save |
| 典型用途 | 界面显示、图标 | 像素处理、多线程 | 矢量录制回放 |

## 12. 总结

| 要点 | 说明 |
|---|---|
| **显示选 QPixmap** | GPU 加速，绘制最快，但只能在 GUI 线程使用 |
| **处理选 QImage** | CPU 内存，支持像素级操作，多线程安全 |
| **QBitmap** | 单色蒙版，是 QPixmap 的 1-bit 子类 |
| **QPicture** | 矢量指令录制/回放，分辨率无关 |
| **高 DPI** | QPixmap/QImage 需乘 DPR 创建，并调 `setDevicePixelRatio()` |
| **像素性能** | `scanLine()` > `bits()` > `pixel()` > `pixelColor()` |
| **离屏格式** | `Format_ARGB32_Premultiplied` 混合运算最快 |
| **缓存** | `QPixmapCache` 全局缓存；控件可用脏标记+离屏 QPixmap 避免重复绘制 |
| **多线程** | 工作线程处理 QImage，GUI 线程再转 QPixmap 显示 |
| **缩放** | 预缩放到目标大小并缓存，不要在 `paintEvent` 中实时缩放 |
