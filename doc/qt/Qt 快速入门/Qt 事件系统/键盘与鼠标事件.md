# 键盘与鼠标事件

## 1. 概述

键盘事件和鼠标事件是 Qt 事件系统中最常用的用户输入事件。Qt 将操作系统原生的输入消息封装为 `QKeyEvent` 和 `QMouseEvent` 等事件对象，通过事件分发机制传递给具有焦点的控件。

处理这些事件的基本方式是重写 `QWidget` 中对应的虚函数：

| 事件类型 | 事件类 | 重写函数 |
|---------|--------|---------|
| 键盘按下 | `QKeyEvent` | `keyPressEvent()` |
| 键盘释放 | `QKeyEvent` | `keyReleaseEvent()` |
| 鼠标按下 | `QMouseEvent` | `mousePressEvent()` |
| 鼠标释放 | `QMouseEvent` | `mouseReleaseEvent()` |
| 鼠标移动 | `QMouseEvent` | `mouseMoveEvent()` |
| 鼠标双击 | `QMouseEvent` | `mouseDoubleClickEvent()` |
| 鼠标滚轮 | `QWheelEvent` | `wheelEvent()` |
| 鼠标进入 | `QEnterEvent` | `enterEvent()` |
| 鼠标离开 | `QEvent` | `leaveEvent()` |

## 2. 键盘事件

### 2.1 QKeyEvent 类详解

`QKeyEvent` 描述了一次键盘按键事件，包含按键值、修饰键状态、文本内容等信息：

```cpp
class QKeyEvent : public QInputEvent
{
public:
    int key() const;              // 按键码（Qt::Key 枚举值）
    Qt::KeyboardModifiers modifiers() const; // 修饰键状态
    QString text() const;         // 按键对应的文本（如 "a"、"A"）
    bool isAutoRepeat() const;    // 是否为长按自动重复
    int count() const;            // 重复次数
    quint32 nativeScanCode() const;    // 原生扫描码
    quint32 nativeVirtualKey() const;  // 原生虚拟键码
    quint32 nativeModifiers() const;   // 原生修饰键
    bool matches(QKeySequence::StandardKey key) const; // 匹配标准快捷键
};
```

### 2.2 键盘按下与释放

重写 `keyPressEvent()` 和 `keyReleaseEvent()` 来处理键盘输入：

```cpp
class MyWidget : public QWidget
{
    Q_OBJECT
public:
    explicit MyWidget(QWidget *parent = nullptr) : QWidget(parent)
    {
        setFocusPolicy(Qt::StrongFocus); // 必须设置焦点策略才能接收键盘事件
    }

protected:
    void keyPressEvent(QKeyEvent *event) override
    {
        switch (event->key()) {
        case Qt::Key_Up:
            qDebug() << "上方向键按下";
            break;
        case Qt::Key_Down:
            qDebug() << "下方向键按下";
            break;
        case Qt::Key_Left:
            qDebug() << "左方向键按下";
            break;
        case Qt::Key_Right:
            qDebug() << "右方向键按下";
            break;
        case Qt::Key_Escape:
            qDebug() << "Esc 键按下，关闭窗口";
            close();
            break;
        default:
            // 重要：不处理的按键交给基类处理
            QWidget::keyPressEvent(event);
            break;
        }
    }

    void keyReleaseEvent(QKeyEvent *event) override
    {
        if (event->key() == Qt::Key_Space) {
            qDebug() << "空格键释放";
        } else {
            QWidget::keyReleaseEvent(event);
        }
    }
};
```

> **重要提示：** 控件必须拥有键盘焦点才能接收键盘事件。通过 `setFocusPolicy()` 设置焦点策略：
> - `Qt::NoFocus` — 不接受焦点（默认值，QLabel 等）
> - `Qt::TabFocus` — 仅通过 Tab 键获得焦点
> - `Qt::ClickFocus` — 仅通过鼠标点击获得焦点
> - `Qt::StrongFocus` — Tab 和鼠标点击都能获得焦点（推荐）
> - `Qt::WheelFocus` — StrongFocus + 鼠标滚轮

### 2.3 修饰键检测

修饰键（Ctrl / Shift / Alt / Meta）通过 `modifiers()` 获取，返回 `Qt::KeyboardModifiers` 标志位组合：

```cpp
void MyWidget::keyPressEvent(QKeyEvent *event)
{
    // 方式一：通过 modifiers() 按位判断
    if (event->modifiers() & Qt::ControlModifier) {
        if (event->key() == Qt::Key_S) {
            qDebug() << "Ctrl+S：保存";
            save();
            return;
        }
        if (event->key() == Qt::Key_Z) {
            if (event->modifiers() & Qt::ShiftModifier) {
                qDebug() << "Ctrl+Shift+Z：重做";
                redo();
            } else {
                qDebug() << "Ctrl+Z：撤销";
                undo();
            }
            return;
        }
    }

    // 方式二：使用 matches() 匹配标准快捷键（推荐，跨平台兼容）
    if (event->matches(QKeySequence::Copy)) {
        qDebug() << "复制操作";
        return;
    }
    if (event->matches(QKeySequence::Paste)) {
        qDebug() << "粘贴操作";
        return;
    }

    QWidget::keyPressEvent(event);
}
```

常用修饰键标志：

| 标志 | 说明 | Windows/Linux | macOS |
|-----|------|--------------|-------|
| `Qt::ControlModifier` | Ctrl | Ctrl | ⌘ Command |
| `Qt::ShiftModifier` | Shift | Shift | ⇧ Shift |
| `Qt::AltModifier` | Alt | Alt | ⌥ Option |
| `Qt::MetaModifier` | Meta | Win 键 | ^ Control |
| `Qt::KeypadModifier` | 数字小键盘 | NumPad | NumPad |

> **跨平台注意：** macOS 上 `Qt::ControlModifier` 映射到 ⌘ Command 键，`Qt::MetaModifier` 映射到 ^ Control 键。使用 `matches(QKeySequence::StandardKey)` 可以自动处理平台差异。

### 2.4 区分长按与单次按下

长按时系统会持续产生 `keyPressEvent`，通过 `isAutoRepeat()` 区分：

```cpp
void MyWidget::keyPressEvent(QKeyEvent *event)
{
    if (event->key() == Qt::Key_Space) {
        if (!event->isAutoRepeat()) {
            qDebug() << "空格键首次按下 —— 开始动作";
            startAction();
        }
        // isAutoRepeat() == true 时忽略，避免重复触发
    }
}

void MyWidget::keyReleaseEvent(QKeyEvent *event)
{
    if (event->key() == Qt::Key_Space) {
        if (!event->isAutoRepeat()) {
            qDebug() << "空格键真正释放 —— 结束动作";
            stopAction();
        }
    }
}
```

### 2.5 组合键与多键同时按下

Qt 的键盘事件每次只报告一个键。要检测多键同时按下，需使用状态集合记录：

```cpp
class GameWidget : public QWidget
{
    Q_OBJECT
public:
    GameWidget(QWidget *parent = nullptr) : QWidget(parent)
    {
        setFocusPolicy(Qt::StrongFocus);
        // 启动定时器驱动游戏循环
        startTimer(16); // ~60 FPS
    }

protected:
    void keyPressEvent(QKeyEvent *event) override
    {
        if (!event->isAutoRepeat())
            m_pressedKeys.insert(event->key());
    }

    void keyReleaseEvent(QKeyEvent *event) override
    {
        if (!event->isAutoRepeat())
            m_pressedKeys.remove(event->key());
    }

    void timerEvent(QTimerEvent *) override
    {
        // 在游戏循环中读取当前按键状态
        int dx = 0, dy = 0;
        if (m_pressedKeys.contains(Qt::Key_W)) dy -= 5;
        if (m_pressedKeys.contains(Qt::Key_S)) dy += 5;
        if (m_pressedKeys.contains(Qt::Key_A)) dx -= 5;
        if (m_pressedKeys.contains(Qt::Key_D)) dx += 5;

        if (dx != 0 || dy != 0) {
            m_playerPos += QPoint(dx, dy);
            update(); // 触发重绘
        }
    }

private:
    QSet<int> m_pressedKeys;
    QPoint m_playerPos{100, 100};
};
```

### 2.6 QShortcut 与 QAction 快捷键

对于全局或窗口级快捷键，优先使用 `QShortcut` 或 `QAction` 而非重写 `keyPressEvent()`：

```cpp
// 方式一：QShortcut（适合简单快捷键绑定）
auto *shortcut = new QShortcut(QKeySequence("Ctrl+N"), this);
connect(shortcut, &QShortcut::activated, this, &MyWidget::createNew);

// 带上下文参数（控制快捷键生效范围）
auto *globalShortcut = new QShortcut(QKeySequence("F5"), this);
globalShortcut->setContext(Qt::ApplicationShortcut); // 整个应用程序范围内生效
connect(globalShortcut, &QShortcut::activated, this, &MyWidget::refresh);

// 方式二：QAction（适合菜单/工具栏/快捷键三合一）
auto *saveAction = new QAction("保存(&S)", this);
saveAction->setShortcut(QKeySequence::Save); // 标准快捷键
saveAction->setShortcutContext(Qt::WindowShortcut);
connect(saveAction, &QAction::triggered, this, &MyWidget::save);
addAction(saveAction); // 必须添加到控件才能生效
```

快捷键上下文（`Qt::ShortcutContext`）：

| 值 | 说明 |
|---|------|
| `Qt::WidgetShortcut` | 仅当控件拥有焦点时生效 |
| `Qt::WidgetWithChildrenShortcut` | 控件或其子控件拥有焦点时生效 |
| `Qt::WindowShortcut` | 控件所在窗口为活动窗口时生效（默认） |
| `Qt::ApplicationShortcut` | 应用程序中任何窗口活动时都生效 |

## 3. 鼠标事件

### 3.1 QMouseEvent 类详解

`QMouseEvent` 描述鼠标按下、释放、移动和双击事件：

```cpp
class QMouseEvent : public QInputEvent
{
public:
    // 位置信息
    QPoint pos() const;          // 相对于接收控件的坐标（整数）
    QPointF localPos() const;    // 相对于接收控件的坐标（浮点，更精确）
    QPoint globalPos() const;    // 相对于屏幕的全局坐标
    QPointF screenPos() const;   // 全局坐标（浮点）
    QPointF windowPos() const;   // 相对于窗口的坐标

    // 按钮信息
    Qt::MouseButton button() const;       // 触发此事件的按钮（Press/Release 时有效）
    Qt::MouseButtons buttons() const;     // 事件发生时所有处于按下状态的按钮

    // 修饰键
    Qt::KeyboardModifiers modifiers() const; // Ctrl/Shift/Alt 等修饰键状态

    // Qt 5.4 引入的来源信息
    Qt::MouseEventSource source() const;  // 事件来源（真实鼠标/触摸模拟等）
    Qt::MouseEventFlags flags() const;    // 事件标志
};
```

**`button()` 与 `buttons()` 的区别：**
- `button()` 返回**引发当前事件的那个按钮**，仅在 Press/Release 事件中有意义；在 Move 事件中返回 `Qt::NoButton`。
- `buttons()` 返回事件发生时**所有处于按下状态的按钮集合**，在 Move 事件中特别有用。

```cpp
void MyWidget::mousePressEvent(QMouseEvent *event)
{
    // button() 是刚刚按下的那个按钮
    qDebug() << "按下了:" << event->button();       // 如 Qt::LeftButton

    // buttons() 是当前所有按着的按钮
    qDebug() << "当前按着:" << event->buttons();     // 如 Qt::LeftButton | Qt::RightButton
}

void MyWidget::mouseMoveEvent(QMouseEvent *event)
{
    // Move 事件中 button() 总是 Qt::NoButton
    qDebug() << "button():" << event->button();      // Qt::NoButton

    // 用 buttons() 判断拖拽时哪些键按着
    if (event->buttons() & Qt::LeftButton) {
        qDebug() << "左键拖拽中，位置:" << event->pos();
    }
}
```

### 3.2 鼠标按键枚举

```cpp
enum Qt::MouseButton {
    NoButton     = 0x00000000,  // 无按钮
    LeftButton   = 0x00000001,  // 左键
    RightButton  = 0x00000002,  // 右键
    MiddleButton = 0x00000004,  // 中键（滚轮按下）
    BackButton   = 0x00000008,  // 后退侧键（鼠标第4键）
    ForwardButton= 0x00000010,  // 前进侧键（鼠标第5键）
};
```

### 3.3 鼠标按下、释放与双击

```cpp
class ClickWidget : public QWidget
{
    Q_OBJECT

protected:
    void mousePressEvent(QMouseEvent *event) override
    {
        if (event->button() == Qt::LeftButton) {
            qDebug() << "左键按下 at" << event->pos();
            m_dragStartPos = event->pos();
        } else if (event->button() == Qt::RightButton) {
            qDebug() << "右键按下 —— 弹出上下文菜单";
            showContextMenu(event->globalPos());
        } else if (event->button() == Qt::MiddleButton) {
            qDebug() << "中键按下 at" << event->pos();
        }
    }

    void mouseReleaseEvent(QMouseEvent *event) override
    {
        if (event->button() == Qt::LeftButton) {
            qDebug() << "左键释放 at" << event->pos();
        }
    }

    void mouseDoubleClickEvent(QMouseEvent *event) override
    {
        if (event->button() == Qt::LeftButton) {
            qDebug() << "左键双击 at" << event->pos();
            // 注意：双击事件序列为：
            // Press → Release → DoubleClick → Release
        }
    }

private:
    QPoint m_dragStartPos;

    void showContextMenu(const QPoint &globalPos)
    {
        QMenu menu;
        menu.addAction("剪切", this, []{ qDebug() << "剪切"; });
        menu.addAction("复制", this, []{ qDebug() << "复制"; });
        menu.addAction("粘贴", this, []{ qDebug() << "粘贴"; });
        menu.exec(globalPos);
    }
};
```

> **双击事件序列：** 当用户双击时，Qt 产生的事件序列为：
> `mousePressEvent` → `mouseReleaseEvent` → `mouseDoubleClickEvent` → `mouseReleaseEvent`
> 第一次点击产生 Press+Release，第二次点击产生 DoubleClick+Release。如果同时重写了 `mousePressEvent` 和 `mouseDoubleClickEvent`，需要注意首次点击仍会触发 Press。

### 3.4 鼠标移动与追踪

默认情况下，`mouseMoveEvent()` 仅在**鼠标按下拖拽**时触发。开启鼠标追踪后，即使不按键也会触发：

```cpp
class TrackWidget : public QWidget
{
    Q_OBJECT
public:
    TrackWidget(QWidget *parent = nullptr) : QWidget(parent)
    {
        // 开启鼠标追踪：不按任何键也能收到 mouseMoveEvent
        setMouseTracking(true);
    }

protected:
    void mouseMoveEvent(QMouseEvent *event) override
    {
        // 不按任何键时 buttons() == Qt::NoButton
        if (event->buttons() == Qt::NoButton) {
            qDebug() << "鼠标悬停移动:" << event->pos();
            updateHoverEffect(event->pos());
        }

        // 按下左键拖拽
        if (event->buttons() & Qt::LeftButton) {
            qDebug() << "左键拖拽:" << event->pos();
        }
    }

private:
    void updateHoverEffect(const QPoint &pos)
    {
        // 更新悬停高亮效果
    }
};
```

### 3.5 实现拖拽移动窗口

一个经典应用场景 —— 无边框窗口的鼠标拖拽移动：

```cpp
class FramelessWindow : public QWidget
{
    Q_OBJECT
public:
    FramelessWindow(QWidget *parent = nullptr) : QWidget(parent)
    {
        setWindowFlags(Qt::FramelessWindowHint); // 无边框
        setFixedSize(400, 300);
    }

protected:
    void mousePressEvent(QMouseEvent *event) override
    {
        if (event->button() == Qt::LeftButton) {
            m_dragging = true;
            // 记录鼠标在窗口内的偏移量
            m_dragOffset = event->globalPos() - frameGeometry().topLeft();
            event->accept();
        }
    }

    void mouseMoveEvent(QMouseEvent *event) override
    {
        if (m_dragging && (event->buttons() & Qt::LeftButton)) {
            // 将窗口移动到新位置
            move(event->globalPos() - m_dragOffset);
            event->accept();
        }
    }

    void mouseReleaseEvent(QMouseEvent *event) override
    {
        if (event->button() == Qt::LeftButton) {
            m_dragging = false;
            event->accept();
        }
    }

private:
    bool m_dragging = false;
    QPoint m_dragOffset;
};
```

### 3.6 鼠标进入与离开

`enterEvent()` 和 `leaveEvent()` 在鼠标进入/离开控件区域时触发，常用于悬停高亮效果：

```cpp
class HoverButton : public QPushButton
{
    Q_OBJECT
public:
    HoverButton(const QString &text, QWidget *parent = nullptr)
        : QPushButton(text, parent)
    {
        // 为了获得流畅的过渡，也可以用 setAttribute(Qt::WA_Hover)
    }

protected:
    void enterEvent(QEvent *event) override
    {
        // 鼠标进入时改变样式
        setStyleSheet("background-color: #3498db; color: white; border-radius: 4px;");
        QPushButton::enterEvent(event);
    }

    void leaveEvent(QEvent *event) override
    {
        // 鼠标离开时恢复样式
        setStyleSheet("background-color: #ecf0f1; color: black; border-radius: 4px;");
        QPushButton::leaveEvent(event);
    }
};
```

> **更推荐的方式：** 悬停效果通常用 QSS（Qt Style Sheet）的 `:hover` 伪状态实现，无需重写事件函数：
> ```css
> QPushButton {
>     background-color: #ecf0f1;
>     border-radius: 4px;
> }
> QPushButton:hover {
>     background-color: #3498db;
>     color: white;
> }
> ```

## 4. 滚轮事件

### 4.1 QWheelEvent 类详解

`QWheelEvent` 描述鼠标滚轮（包括触控板双指滑动）事件：

```cpp
class QWheelEvent : public QInputEvent
{
public:
    // Qt 5.14+ 推荐 API
    QPoint angleDelta() const;     // 滚动角度增量（1/8 度单位）
    QPoint pixelDelta() const;     // 像素级增量（高精度设备，如触控板）
    QPointF position() const;      // 相对于接收控件的位置
    QPointF globalPosition() const;// 全局坐标
    Qt::MouseButtons buttons() const;
    Qt::KeyboardModifiers modifiers() const;
    Qt::ScrollPhase phase() const; // 滚动阶段（开始/更新/结束）
    bool inverted() const;         // 是否为"自然滚动"方向

    // Qt 5.14 之前的旧 API（已废弃）
    // int delta() const;
    // Qt::Orientation orientation() const;
};
```

**angleDelta() 说明：**
- 垂直滚动：`angleDelta().y()` 非零，正值向上滚，负值向下滚
- 水平滚动：`angleDelta().x()` 非零（Shift+滚轮 或触控板水平滑动）
- 传统鼠标一格滚轮 = 120（即 15 度 × 8）
- 高精度滚轮/触控板可能产生更小的增量

### 4.2 滚轮事件处理

```cpp
class ZoomWidget : public QWidget
{
    Q_OBJECT
public:
    ZoomWidget(QWidget *parent = nullptr) : QWidget(parent) {}

protected:
    void wheelEvent(QWheelEvent *event) override
    {
        // 获取滚动角度增量
        QPoint numDegrees = event->angleDelta() / 8;
        QPoint numSteps = numDegrees / 15; // 标准鼠标一格 = 1 步

        if (!numSteps.isNull()) {
            if (event->modifiers() & Qt::ControlModifier) {
                // Ctrl + 滚轮：缩放
                if (numSteps.y() > 0) {
                    zoomIn();
                } else {
                    zoomOut();
                }
            } else {
                // 普通滚轮：滚动内容
                scrollContent(numSteps.y() * 20); // 每步滚动 20 像素
            }
        }

        event->accept();
    }

    void paintEvent(QPaintEvent *) override
    {
        QPainter painter(this);
        painter.setRenderHint(QPainter::Antialiasing);

        // 以 m_scale 绘制内容
        painter.translate(width() / 2.0, height() / 2.0);
        painter.scale(m_scale, m_scale);
        painter.drawEllipse(QPointF(0, 0), 50, 50);

        painter.resetTransform();
        painter.drawText(10, 20, QString("缩放: %1%").arg(int(m_scale * 100)));
    }

private:
    double m_scale = 1.0;
    int m_scrollOffset = 0;

    void zoomIn()
    {
        m_scale = qMin(m_scale * 1.1, 10.0);
        update();
    }

    void zoomOut()
    {
        m_scale = qMax(m_scale / 1.1, 0.1);
        update();
    }

    void scrollContent(int delta)
    {
        m_scrollOffset += delta;
        update();
    }
};
```

### 4.3 高精度滚动（触控板）

触控板发出的滚轮事件与传统鼠标有差异，推荐使用 `pixelDelta()` 获取更平滑的滚动体验：

```cpp
void MyScrollArea::wheelEvent(QWheelEvent *event)
{
    int delta = 0;

    // 优先使用 pixelDelta（触控板提供像素级精度）
    if (!event->pixelDelta().isNull()) {
        delta = event->pixelDelta().y();
    } else {
        // 回退到 angleDelta（传统鼠标）
        delta = event->angleDelta().y() / 8; // 转为度数
    }

    m_scrollPosition -= delta;
    m_scrollPosition = qBound(0, m_scrollPosition, m_contentHeight - height());
    update();

    event->accept();
}
```

## 5. 坐标系统

鼠标事件涉及多个坐标系：

```
┌─────── 屏幕（Screen） ──────────────────────────┐
│ globalPos() / screenPos()                        │
│                                                   │
│   ┌──── 窗口（Window） ─────────────────┐        │
│   │ windowPos()                          │        │
│   │                                      │        │
│   │   ┌── 控件（Widget） ──────┐        │        │
│   │   │ pos() / localPos()      │        │        │
│   │   │                         │        │        │
│   │   │    ● (x, y)            │        │        │
│   │   │                         │        │        │
│   │   └─────────────────────────┘        │        │
│   │                                      │        │
│   └──────────────────────────────────────┘        │
│                                                   │
└───────────────────────────────────────────────────┘
```

坐标转换工具函数：

```cpp
void MyWidget::mousePressEvent(QMouseEvent *event)
{
    QPoint widgetPos = event->pos();       // 控件坐标
    QPoint windowPos = event->windowPos().toPoint(); // 窗口坐标
    QPoint screenPos = event->globalPos(); // 屏幕坐标

    // 手动转换
    QPoint toScreen = mapToGlobal(widgetPos);   // 控件坐标 → 屏幕坐标
    QPoint toWidget = mapFromGlobal(screenPos); // 屏幕坐标 → 控件坐标

    // 转换到父控件坐标
    QPoint toParent = mapToParent(widgetPos);
    QPoint fromParent = mapFromParent(toParent);

    // 控件间的坐标转换
    QPoint inOther = mapTo(otherWidget, widgetPos);    // 到某个祖先控件坐标
    QPoint fromOther = mapFrom(otherWidget, inOther);  // 从某个祖先控件坐标
}
```

## 6. 事件的 accept() 与 ignore()

### 6.1 传播机制

鼠标事件和键盘事件都支持在父子控件链上传播：

- 调用 `event->accept()` — **接受事件**，事件停止传播（默认行为）
- 调用 `event->ignore()` — **忽略事件**，事件继续向父控件传播

```cpp
// 子控件
class ChildWidget : public QWidget
{
protected:
    void mousePressEvent(QMouseEvent *event) override
    {
        if (event->button() == Qt::LeftButton) {
            qDebug() << "子控件处理左键点击";
            event->accept(); // 事件到此为止，不再传给父控件
        } else {
            qDebug() << "子控件不处理此按钮，传给父控件";
            event->ignore(); // 传递给父控件处理
        }
    }

    void keyPressEvent(QKeyEvent *event) override
    {
        if (event->key() == Qt::Key_Return) {
            qDebug() << "子控件处理 Enter";
            event->accept();
        } else {
            // 调用基类实现 = 默认 ignore 并传播
            QWidget::keyPressEvent(event);
        }
    }
};

// 父控件
class ParentWidget : public QWidget
{
protected:
    void mousePressEvent(QMouseEvent *event) override
    {
        // 只有子控件 ignore() 后才会到达这里
        qDebug() << "父控件收到鼠标事件（子控件未处理）";
    }
};
```

### 6.2 closeEvent 中的 accept/ignore

`accept()` 和 `ignore()` 在关闭事件中有特殊语义：

```cpp
void MyWindow::closeEvent(QCloseEvent *event)
{
    if (hasUnsavedChanges()) {
        auto ret = QMessageBox::question(this, "提示",
            "有未保存的更改，确定关闭？",
            QMessageBox::Save | QMessageBox::Discard | QMessageBox::Cancel);

        switch (ret) {
        case QMessageBox::Save:
            save();
            event->accept();  // 保存后允许关闭
            break;
        case QMessageBox::Discard:
            event->accept();  // 丢弃更改，允许关闭
            break;
        case QMessageBox::Cancel:
            event->ignore();  // 取消关闭，窗口保持打开
            break;
        default:
            event->ignore();
            break;
        }
    } else {
        event->accept();
    }
}
```

## 7. 鼠标抓取

### 7.1 grabMouse / releaseMouse

`grabMouse()` 使控件独占所有鼠标事件，即使鼠标移到该控件外部：

```cpp
class GrabWidget : public QWidget
{
protected:
    void mousePressEvent(QMouseEvent *event) override
    {
        if (event->button() == Qt::LeftButton) {
            grabMouse();         // 抓取鼠标，独占所有鼠标事件
            // grabMouse(Qt::ClosedHandCursor); // 抓取并改变光标
            qDebug() << "开始抓取鼠标";
        }
    }

    void mouseReleaseEvent(QMouseEvent *event) override
    {
        if (event->button() == Qt::LeftButton) {
            releaseMouse();      // 释放鼠标抓取
            qDebug() << "释放鼠标抓取";
        }
    }

    void mouseMoveEvent(QMouseEvent *event) override
    {
        // 抓取后，即使鼠标移出控件区域，事件仍会发送到此控件
        qDebug() << "鼠标位置（可能在控件外）:" << event->globalPos();
    }
};
```

> **注意：** `grabMouse()` 是底层操作，通常不需要直接使用。Qt 在鼠标按下后会自动进行隐式抓取，直到所有按钮释放。实际开发中更推荐使用 Qt 的拖放框架（`QDrag`）。

### 7.2 grabKeyboard / releaseKeyboard

类似地，`grabKeyboard()` 使控件独占键盘输入：

## 8. 综合实例

### 8.1 可拖拽调整大小的控件

```cpp
class ResizableWidget : public QWidget
{
    Q_OBJECT
public:
    ResizableWidget(QWidget *parent = nullptr) : QWidget(parent)
    {
        setMouseTracking(true);
        setMinimumSize(100, 100);
        resize(300, 200);
    }

protected:
    void paintEvent(QPaintEvent *) override
    {
        QPainter painter(this);
        painter.fillRect(rect(), QColor("#2c3e50"));
        painter.setPen(Qt::white);
        painter.drawText(rect(), Qt::AlignCenter,
            QString("大小: %1 × %2").arg(width()).arg(height()));

        // 绘制右下角调整手柄
        painter.setPen(QPen(Qt::gray, 1));
        int g = m_gripSize;
        for (int i = 0; i < 3; ++i) {
            painter.drawLine(width() - g + i * 4, height(),
                            width(), height() - g + i * 4);
        }
    }

    void mousePressEvent(QMouseEvent *event) override
    {
        if (event->button() == Qt::LeftButton) {
            m_edge = hitTest(event->pos());
            if (m_edge != Edge::None) {
                m_resizing = true;
                m_resizeOrigin = event->globalPos();
                m_originalGeometry = geometry();
            } else {
                // 非边缘区域：拖拽移动
                m_moving = true;
                m_moveOffset = event->globalPos() - pos();
            }
            event->accept();
        }
    }

    void mouseMoveEvent(QMouseEvent *event) override
    {
        if (m_resizing) {
            QPoint delta = event->globalPos() - m_resizeOrigin;
            QRect newGeo = m_originalGeometry;

            if (m_edge & Edge::Right)
                newGeo.setRight(newGeo.right() + delta.x());
            if (m_edge & Edge::Bottom)
                newGeo.setBottom(newGeo.bottom() + delta.y());
            if (m_edge & Edge::Left)
                newGeo.setLeft(newGeo.left() + delta.x());
            if (m_edge & Edge::Top)
                newGeo.setTop(newGeo.top() + delta.y());

            // 确保最小尺寸
            if (newGeo.width() >= minimumWidth() && newGeo.height() >= minimumHeight())
                setGeometry(newGeo);

        } else if (m_moving) {
            move(event->globalPos() - m_moveOffset);
        } else {
            // 悬停时更新光标
            updateCursorShape(event->pos());
        }
        event->accept();
    }

    void mouseReleaseEvent(QMouseEvent *event) override
    {
        m_resizing = false;
        m_moving = false;
        event->accept();
    }

private:
    enum Edge {
        None   = 0x00,
        Left   = 0x01,
        Top    = 0x02,
        Right  = 0x04,
        Bottom = 0x08
    };

    int m_gripSize = 12;
    bool m_resizing = false;
    bool m_moving = false;
    int m_edge = Edge::None;
    QPoint m_resizeOrigin;
    QPoint m_moveOffset;
    QRect m_originalGeometry;

    int hitTest(const QPoint &pos)
    {
        int edge = Edge::None;
        if (pos.x() < m_gripSize)                edge |= Edge::Left;
        if (pos.x() > width() - m_gripSize)      edge |= Edge::Right;
        if (pos.y() < m_gripSize)                edge |= Edge::Top;
        if (pos.y() > height() - m_gripSize)      edge |= Edge::Bottom;
        return edge;
    }

    void updateCursorShape(const QPoint &pos)
    {
        int edge = hitTest(pos);
        if ((edge == (Edge::Left | Edge::Top)) || (edge == (Edge::Right | Edge::Bottom)))
            setCursor(Qt::SizeFDiagCursor);
        else if ((edge == (Edge::Right | Edge::Top)) || (edge == (Edge::Left | Edge::Bottom)))
            setCursor(Qt::SizeBDiagCursor);
        else if (edge & (Edge::Left | Edge::Right))
            setCursor(Qt::SizeHorCursor);
        else if (edge & (Edge::Top | Edge::Bottom))
            setCursor(Qt::SizeVerCursor);
        else
            setCursor(Qt::ArrowCursor);
    }
};
```

### 8.2 画板：鼠标绘图 + 键盘快捷键

一个完整的小画板程序，综合运用键盘和鼠标事件：

```cpp
class DrawingBoard : public QWidget
{
    Q_OBJECT
public:
    DrawingBoard(QWidget *parent = nullptr)
        : QWidget(parent), m_image(800, 600, QImage::Format_RGB32)
    {
        setFocusPolicy(Qt::StrongFocus);
        setMouseTracking(true);
        m_image.fill(Qt::white);
        resize(800, 600);
        setWindowTitle("画板 - 按 C 清空 | 按 +/- 调整笔刷 | Ctrl+Z 撤销");
    }

protected:
    void mousePressEvent(QMouseEvent *event) override
    {
        if (event->button() == Qt::LeftButton) {
            m_drawing = true;
            m_lastPoint = event->pos();

            // 保存当前状态用于撤销
            m_undoStack.push(m_image);
            if (m_undoStack.size() > 20) // 限制撤销栈大小
                m_undoStack.removeFirst();
        }
    }

    void mouseMoveEvent(QMouseEvent *event) override
    {
        m_cursorPos = event->pos();

        if (m_drawing && (event->buttons() & Qt::LeftButton)) {
            QPainter painter(&m_image);
            painter.setRenderHint(QPainter::Antialiasing);
            QPen pen(m_penColor, m_penWidth, Qt::SolidLine, Qt::RoundCap, Qt::RoundJoin);
            painter.setPen(pen);
            painter.drawLine(m_lastPoint, event->pos());
            m_lastPoint = event->pos();
        }

        update(); // 刷新画布和光标指示器
    }

    void mouseReleaseEvent(QMouseEvent *event) override
    {
        if (event->button() == Qt::LeftButton) {
            m_drawing = false;
        }
    }

    void wheelEvent(QWheelEvent *event) override
    {
        // 滚轮调整笔刷大小
        int delta = event->angleDelta().y();
        if (delta > 0)
            m_penWidth = qMin(m_penWidth + 1, 50);
        else
            m_penWidth = qMax(m_penWidth - 1, 1);
        update();
        event->accept();
    }

    void keyPressEvent(QKeyEvent *event) override
    {
        if (event->matches(QKeySequence::Undo)) {
            // Ctrl+Z 撤销
            if (!m_undoStack.isEmpty()) {
                m_image = m_undoStack.pop();
                update();
            }
        } else if (event->key() == Qt::Key_C && !(event->modifiers() & Qt::ControlModifier)) {
            // 按 C 清空画布
            m_image.fill(Qt::white);
            m_undoStack.clear();
            update();
        } else if (event->key() == Qt::Key_Plus || event->key() == Qt::Key_Equal) {
            m_penWidth = qMin(m_penWidth + 2, 50);
            update();
        } else if (event->key() == Qt::Key_Minus) {
            m_penWidth = qMax(m_penWidth - 2, 1);
            update();
        } else if (event->key() == Qt::Key_1) {
            m_penColor = Qt::black;
        } else if (event->key() == Qt::Key_2) {
            m_penColor = Qt::red;
        } else if (event->key() == Qt::Key_3) {
            m_penColor = Qt::blue;
        } else if (event->key() == Qt::Key_4) {
            m_penColor = Qt::green;
        } else {
            QWidget::keyPressEvent(event);
        }
    }

    void paintEvent(QPaintEvent *) override
    {
        QPainter painter(this);
        painter.drawImage(0, 0, m_image);

        // 绘制光标指示器（笔刷大小预览）
        painter.setPen(QPen(Qt::gray, 1, Qt::DashLine));
        painter.setBrush(Qt::NoBrush);
        painter.drawEllipse(m_cursorPos, m_penWidth / 2, m_penWidth / 2);

        // 状态栏
        painter.setPen(Qt::black);
        painter.drawText(10, height() - 10,
            QString("笔刷: %1px | 颜色: %2 | 按 1-4 切换颜色")
            .arg(m_penWidth)
            .arg(m_penColor == Qt::black ? "黑" :
                 m_penColor == Qt::red   ? "红" :
                 m_penColor == Qt::blue  ? "蓝" : "绿"));
    }

private:
    QImage m_image;
    bool m_drawing = false;
    QPoint m_lastPoint;
    QPoint m_cursorPos;
    int m_penWidth = 3;
    QColor m_penColor = Qt::black;
    QStack<QImage> m_undoStack;
};
```

## 9. 常见问题与最佳实践

### 9.1 控件收不到键盘事件

**原因：** 控件没有获得键盘焦点。

**解决方案：**
```cpp
// 1. 设置焦点策略
widget->setFocusPolicy(Qt::StrongFocus);

// 2. 必要时手动获取焦点
widget->setFocus();

// 3. 确保没有其他控件抢走了焦点（如 QLineEdit）
```

### 9.2 mouseMoveEvent 只在按下时触发

**原因：** 默认行为就是如此，出于性能考虑。

**解决方案：**
```cpp
widget->setMouseTracking(true); // 开启鼠标追踪
```

### 9.3 双击后的 Press 事件干扰

**解决方案：** 使用定时器延迟处理单击，以区分单击和双击：

```cpp
class SmartClickWidget : public QWidget
{
    Q_OBJECT
public:
    SmartClickWidget(QWidget *parent = nullptr) : QWidget(parent)
    {
        m_clickTimer.setSingleShot(true);
        m_clickTimer.setInterval(QApplication::doubleClickInterval());
        connect(&m_clickTimer, &QTimer::timeout, this, [this]{
            qDebug() << "确认为单击";
            handleSingleClick(m_lastClickPos);
        });
    }

protected:
    void mousePressEvent(QMouseEvent *event) override
    {
        if (event->button() == Qt::LeftButton) {
            m_lastClickPos = event->pos();
            m_clickTimer.start(); // 启动/重启定时器
        }
    }

    void mouseDoubleClickEvent(QMouseEvent *event) override
    {
        if (event->button() == Qt::LeftButton) {
            m_clickTimer.stop(); // 取消单击处理
            qDebug() << "确认为双击";
            handleDoubleClick(event->pos());
        }
    }

private:
    QTimer m_clickTimer;
    QPoint m_lastClickPos;

    void handleSingleClick(const QPoint &pos) { /* ... */ }
    void handleDoubleClick(const QPoint &pos) { /* ... */ }
};
```

### 9.4 不要忘记调用基类实现

重写事件处理函数时，对于不处理的事件务必调用基类实现，否则可能破坏控件的默认行为：

```cpp
void MyWidget::keyPressEvent(QKeyEvent *event)
{
    if (event->key() == Qt::Key_Space) {
        // 自定义处理
        doSomething();
    } else {
        // ⚠️ 重要：将未处理的事件交给基类
        QWidget::keyPressEvent(event);
    }
}
```

## 10. 总结对比

### 键盘事件 vs 快捷键

| 方式 | 适用场景 | 复杂度 |
|------|---------|--------|
| `keyPressEvent()` | 游戏控制、文本编辑器内部逻辑 | 高，需自行管理状态 |
| `QShortcut` | 简单的全局/窗口快捷键 | 低，声明式绑定 |
| `QAction::setShortcut()` | 菜单/工具栏/快捷键统一管理 | 中，适合应用框架 |

### 鼠标事件处理层次

```
应用层 ← QApplication::notify()
  │
  ├─ 事件过滤器 ← eventFilter()（可拦截任意对象的事件）
  │
  ├─ event() ← QWidget::event()（在具体处理函数前被调用）
  │
  ├─ 具体处理函数 ← mousePressEvent() 等
  │      │
  │      ├─ accept() → 事件停止传播
  │      └─ ignore() → 向父控件传播
  │
  └─ 父控件 ← 如果子控件 ignore()，父控件的对应函数被调用
```

掌握键盘与鼠标事件是 Qt GUI 开发的核心基础，灵活运用 `accept()`/`ignore()` 控制事件传播、善用 `QShortcut`/`QAction` 简化快捷键管理、理解坐标系转换，能帮助构建出响应灵敏、交互丰富的桌面应用。
