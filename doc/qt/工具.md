# Inspect.exe（微软 UIAutomation 工具）

微软 SDK 自带。

比 Spy++ 更现代。

可以查看：

- HWND
- UI Automation Tree
- Accessibility
- Name/Class
- AutomationId

# GammaRay（最强推荐）

这是 Qt 官方前 KDAB 团队做的 Qt 运行时分析工具。

可以直接查看：

- QObject 对象树
- QWidget 树
- 信号槽连接
- 属性（Q_PROPERTY）
- Model/View 数据
- QML Tree
- GraphicsView Scene
- 事件
- Paint 分析

非常适合：

- 调试大型 Qt 客户端
- 查对象泄漏
- 查 parent-child 关系
- 查隐藏控件
- 查信号槽问题
